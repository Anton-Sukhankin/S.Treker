# Пресеты колонок: структура интерфейса

## 1. Расположение

Пресеты колонок находятся внутри правой панели `Настройки`, в области `Общие атрибуты`, над списком настраиваемых колонок.

| Область | Положение | Состав |
|---|---|---|
| Правая панель `Настройки` | Справа поверх рабочей области | Хедер, область `Общие атрибуты`, список колонок, футер действий. |
| Компонент пресетов колонок | В области `Общие атрибуты`, над списком колонок | Поле текущего пресета; выпадающий список пресетов; кнопка `Сохранить`; модальное окно сохранения. |
| Таблица задач | За панелью настроек | Получает новый набор колонок только после `Применить` в панели `Настройки`. |

Компонент пресетов не находится в карточном виде списка задач. Пресеты относятся только к табличному отображению.

## 2. Граница компонента

| Входит в компонент пресетов | Не входит в компонент пресетов |
|---|---|
| Поле текущего пресета | Кнопка открытия панели `Настройки` |
| Выпадающий список пресетов | Список строк колонок |
| Пункт `Базовое отображение` | Чекбоксы видимости колонок |
| Пункты сохраненных пресетов | Перетаскивание колонок |
| Кнопка удаления сохраненного пресета | Библиотека атрибутов |
| Кнопка `Сохранить` | Кнопки `По умолчанию` и `Применить` в футере панели |
| Модальное окно `Сохранение пресета` | Таблица задач |

Кнопка `Применить` не входит в компонент пресетов, но именно она переносит выбранный или сохраненный пресет из черновика панели в таблицу задач.
Библиотека атрибутов не входит в компонент пресетов. После выбора пресета у добавленных в `Общие атрибуты` библиотечных атрибутов кнопка действия в библиотеке показывает стрелку назад; у отсутствующих в `Общие атрибуты` атрибутов кнопка действия показывает плюс.

## 3. Блок пресетов в области `Общие атрибуты`

| Элемент | Позиция | Видимое состояние |
|---|---|---|
| Поле текущего пресета | Слева в строке пресетов | Показывает `Базовое отображение` или название выбранного сохраненного пресета. |
| Стрелка раскрытия | Справа внутри поля текущего пресета | Показывает, что поле открывает выпадающий список. |
| Кнопка `Сохранить` | Справа от поля текущего пресета | Состояние `disabled` или активное состояние. |

## 4. Выпадающий список пресетов

| Элемент списка | Условие показа | Видимое состояние |
|---|---|---|
| `Базовое отображение` | Всегда | Первый пункт списка; не содержит кнопку удаления. |
| Сохраненный пресет | Для текущей области есть минимум один сохраненный пресет | Пункт с названием пресета. |
| Кнопка удаления пресета | Только у сохраненного пресета | Иконка удаления внутри строки пресета. |
| Активный пункт | Пункт соответствует выбранному пресету в открытой панели | Пункт получает активное визуальное состояние. |

Если сохраненных пресетов нет, в выпадающем списке виден только пункт `Базовое отображение`.

## 5. Кнопка `Сохранить`

| Состояние кнопки | Условие |
|---|---|
| `disabled` | Черновик колонок совпадает с выбранным пресетом или с `Базовое отображение`. |
| Активна | Черновик колонок отличается от выбранного пресета или от `Базовое отображение`. |

Отличие черновика определяется по составу колонок, порядку колонок и признаку видимости колонок.

## 6. Модальное окно `Сохранение пресета`

| Часть модального окна | Состав |
|---|---|
| Хедер | Заголовок `Сохранение пресета`; кнопка закрытия. |
| Область ввода | Подпись `Название пресета`; поле с плейсхолдером `Введите название`. |
| Футер | Кнопка `Отмена`; кнопка `Подтвердить`. |

Модальное окно открывается поверх интерфейса после клика по активной кнопке `Сохранить`.

## 7. Состав пресета

| Что входит в пресет | Правило |
|---|---|
| Динамические колонки | Пресет хранит только настраиваемые колонки таблицы. |
| Видимость колонок | Для каждой колонки хранится состояние `видима` или `скрыта`. |
| Порядок колонок | Пресет хранит порядок динамических колонок в черновике. |
| Атрибуты из библиотеки | Добавленные из библиотеки атрибуты входят в пресет как динамические колонки; после выбора пресета они показаны в `Общие атрибуты` со стрелкой возврата. |
| Индикация в библиотеке | Если библиотека раскрыта, строка атрибута из выбранного пресета показывает стрелку назад вместо плюса. |

| Что не входит в пресет | Правило |
|---|---|
| Чекбокс выбора строки | Закрепленная колонка всегда остается в таблице и не сохраняется в пресет. |
| Номер задачи | Закрепленная колонка всегда остается в таблице и не сохраняется в пресет. |
| Наименование задачи | Закрепленная колонка всегда остается в таблице и не сохраняется в пресет. |
| Данные задач | Пресет не меняет значения задач. |
| Фильтры и поиск | Пресет не меняет поиск и параметры фильтрации. |
