# Пресеты колонок: поведение

## 1. Основной пользовательский путь

1. Пользователь находится в табличном виде списка задач.
2. Пользователь открывает панель `Настройки`.
3. В области `Общие атрибуты` поле текущего пресета показывает `Базовое отображение` или последний примененный пресет текущей области.
4. Пользователь меняет черновик колонок: скрывает колонку, показывает колонку, меняет порядок или добавляет атрибут из библиотеки.
5. Кнопка `Сохранить` получает активное состояние.
6. Пользователь нажимает `Сохранить`.
7. Поверх интерфейса видно модальное окно `Сохранение пресета`.
8. Пользователь вводит название пресета и нажимает `Подтвердить`.
9. Модальное окно закрыто; новый пресет виден в выпадающем списке пресетов текущей области; поле текущего пресета показывает название нового пресета.
10. Таблица задач сохраняет прежний набор колонок.
11. Пользователь нажимает `Применить` в панели `Настройки`.
12. Панель `Настройки` закрыта; таблица задач показывает набор колонок из сохраненного пресета.

## 2. Открытие списка пресетов

| Действие пользователя | Условие | Итог интерфейса |
|---|---|---|
| Клик по полю текущего пресета | Выпадающий список закрыт | Под полем виден выпадающий список пресетов. |
| Клик по полю текущего пресета | Выпадающий список открыт | Выпадающий список закрыт. |
| Клик вне поля и выпадающего списка | Выпадающий список открыт | Выпадающий список закрыт. |

## 3. Выбор `Базовое отображение`

| Действие пользователя | Условие | Итог интерфейса |
|---|---|---|
| Клик по `Базовое отображение` | Выпадающий список открыт | Поле текущего пресета получает текст `Базовое отображение`; список колонок в панели получает базовый набор текущей области; выпадающий список закрыт. |
| Клик по `Базовое отображение` | До клика был выбран сохраненный пресет | Черновик колонок получает базовый состав, порядок и видимость колонок. |
| Клик по `Базовое отображение` | Библиотека атрибутов раскрыта | В `Общие атрибуты` отсутствуют добавленные из библиотеки атрибуты; в библиотеке строки атрибутов показывают кнопку с плюсом. |
| Клик по `Базовое отображение` | Таблица задач видна за панелью | Таблица сохраняет прежний примененный набор колонок до `Применить`. |

## 4. Выбор сохраненного пресета

| Действие пользователя | Условие | Итог интерфейса |
|---|---|---|
| Клик по сохраненному пресету | Выпадающий список открыт | Поле текущего пресета получает название выбранного пресета; список колонок в панели получает состав, порядок и видимость из выбранного пресета; выпадающий список закрыт. |
| Клик по сохраненному пресету | Пресет содержит атрибуты из библиотеки | В `Общие атрибуты` добавленные из библиотеки атрибуты показаны со стрелкой возврата. |
| Клик по сохраненному пресету | Пресет содержит атрибуты из библиотеки и библиотека раскрыта | В библиотеке строки этих атрибутов показывают стрелку назад вместо плюса. |
| Клик по сохраненному пресету | Пресет содержит колонку, недоступную текущей области | Недоступная колонка отсутствует в списке колонок панели. |
| Клик по сохраненному пресету | Таблица задач видна за панелью | Таблица сохраняет прежний примененный набор колонок до `Применить`. |

## 5. Активация кнопки `Сохранить`

| Действие пользователя | Условие | Итог интерфейса |
|---|---|---|
| Скрывает или показывает колонку | Черновик отличается от выбранного пресета или базы | Кнопка `Сохранить` активна. |
| Меняет порядок колонок | Черновик отличается от выбранного пресета или базы | Кнопка `Сохранить` активна. |
| Добавляет атрибут из библиотеки | Черновик отличается от выбранного пресета или базы | Кнопка `Сохранить` активна. |
| Возвращает черновик к выбранному пресету или базе | Отличий в составе, порядке и видимости нет | Кнопка `Сохранить` получает состояние `disabled`. |

## 6. Сохранение пресета

| Действие пользователя | Условие | Итог интерфейса |
|---|---|---|
| Клик по активной кнопке `Сохранить` | Панель `Настройки` открыта | Видно модальное окно `Сохранение пресета`; поле `Название пресета` пустое. |
| Клик по `Подтвердить` | Поле `Название пресета` пустое | Модальное окно остается открытым; новый пресет не создается. |
| Клик по `Подтвердить` | Поле `Название пресета` содержит текст | Модальное окно закрыто; в выпадающем списке текущей области есть пресет с введенным названием; поле текущего пресета показывает введенное название. |
| Клик по `Подтвердить` | Черновик содержит атрибуты из библиотеки | Сохраненный пресет получает эти атрибуты; в `Общие атрибуты` они сохраняют стрелку возврата. |
| Клик по `Подтвердить` | Пресет с таким названием уже есть в текущей области | Существующий пресет получает состояние текущего черновика; второго пункта с тем же названием в списке нет. |
| Клик по `Подтвердить` | Таблица задач видна за панелью | Таблица сохраняет прежний примененный набор колонок до `Применить`. |

## 7. Закрытие модального окна без сохранения

| Действие пользователя | Условие | Итог интерфейса |
|---|---|---|
| Клик по кнопке закрытия в хедере модального окна | Модальное окно открыто | Модальное окно закрыто; список пресетов не меняется; черновик колонок в панели сохраняет текущее состояние. |
| Клик по `Отмена` | Модальное окно открыто | Модальное окно закрыто; список пресетов не меняется; черновик колонок в панели сохраняет текущее состояние. |

## 8. Удаление пресета

| Действие пользователя | Условие | Итог интерфейса |
|---|---|---|
| Клик по кнопке удаления у сохраненного пресета | Пресет не выбран в открытой панели | Пункт пресета исчезает из выпадающего списка; текущий черновик колонок сохраняет состояние. |
| Клик по кнопке удаления у сохраненного пресета | Пресет выбран в открытой панели | Пункт пресета исчезает из выпадающего списка; поле текущего пресета получает текст `Базовое отображение`; список колонок в панели получает базовый набор текущей области. |
| Клик по кнопке удаления у сохраненного пресета | Таблица задач видна за панелью | Таблица сохраняет прежний примененный набор колонок до `Применить`. |

У пункта `Базовое отображение` нет кнопки удаления.

## 9. Применение выбранного пресета к таблице

| Действие пользователя | Условие | Итог интерфейса |
|---|---|---|
| Клик по `Применить` в панели `Настройки` | В черновике выбран `Базовое отображение` | Панель закрыта; таблица показывает базовый набор колонок текущей области. |
| Клик по `Применить` в панели `Настройки` | В черновике выбран сохраненный пресет | Панель закрыта; таблица показывает состав, порядок и видимость колонок из выбранного пресета. |
| Клик по закрытию панели без `Применить` | Был выбран или сохранен пресет | Панель закрыта; таблица сохраняет прежний примененный набор колонок. |
