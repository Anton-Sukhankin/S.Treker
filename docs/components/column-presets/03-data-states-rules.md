# Пресеты колонок: данные, состояния и правила

## 1. Состояния компонента

| Состояние | Значение | Видимый результат |
|---|---|---|
| Текущий пресет панели | `Базовое отображение` или название сохраненного пресета | Поле пресета показывает это значение. |
| Выпадающий список | Закрыт или открыт | При открытом состоянии под полем видны доступные пресеты текущей области. |
| Сохраненный пресет | Название и состояние колонок | В выпадающем списке виден пункт с названием пресета. |
| Черновик колонок | Временный набор колонок открытой панели | Список колонок в панели показывает черновик; таблица задач сохраняет примененное состояние до `Применить`. |
| Библиотечные атрибуты в пресете | Есть или отсутствуют | При наличии таких атрибутов `Общие атрибуты` показывает их со стрелкой возврата. |
| Модальное окно сохранения | Закрыто или открыто | При открытом состоянии видны заголовок `Сохранение пресета`, поле названия и кнопки `Отмена`, `Подтвердить`. |

## 2. Контекст пресета

| Текущая область левой навигации | Область действия пресета |
|---|---|
| `Все задачи` | Пресет относится к режиму `Все задачи`. |
| Конкретный бизнес-домен | Пресет относится к выбранному бизнес-домену. |
| Пользовательская группа | Пресет относится к выбранной пользовательской группе. |
| Значение группировки внутри `Все задачи` | Пресет относится к режиму `Все задачи`, а не к отдельному значению группировки. |
| Значение группировки внутри бизнес-домена | Пресет относится к родительскому бизнес-домену, а не к отдельному значению группировки. |

Пресет одной области не виден в выпадающем списке другой области.

## 3. Состав состояния пресета

| Составляющая | Правило |
|---|---|
| Ключ колонки | В пресете хранится ссылка на динамическую колонку текущей области. |
| Видимость | Значение `видима` добавляет колонку в таблицу после `Применить`; значение `скрыта` исключает колонку из таблицы после `Применить`. |
| Порядок | Порядок записей в пресете задает порядок динамических колонок после `Применить`. |
| Библиотечный атрибут | Если атрибут добавлен из библиотеки и сохранен в пресете, после выбора пресета он находится в списке `Общие атрибуты`. |
| Индикация библиотечного атрибута | После выбора пресета библиотечный атрибут показывает стрелку возврата в `Общие атрибуты`; при раскрытой библиотеке его строка в библиотеке показывает стрелку назад вместо плюса. |

Закрепленные колонки выбора строки, номера задачи и наименования задачи не входят в состояние пресета.

## 4. Правила доступности кнопки `Сохранить`

| Условие черновика | Состояние кнопки `Сохранить` |
|---|---|
| Состав, порядок и видимость колонок совпадают с текущим пресетом | `disabled`. |
| Состав, порядок и видимость колонок совпадают с `Базовое отображение` при выбранной базе | `disabled`. |
| Отличается состав колонок | Активна. |
| Отличается порядок колонок | Активна. |
| Отличается видимость минимум одной колонки | Активна. |

## 5. Правила сохранения

| Условие | Видимый результат |
|---|---|
| Пользователь подтвердил пустое название | Модальное окно остается открытым; список пресетов сохраняет прежний состав. |
| Пользователь подтвердил новое название | В списке пресетов текущей области виден новый пункт с этим названием. |
| Пользователь подтвердил название существующего пресета | Существующий пункт с этим названием сохраняет место в списке; его состояние колонок получает состояние текущего черновика. |
| Пресет сохранен | Поле текущего пресета показывает название сохраненного пресета. |
| Пресет сохранен без `Применить` | Таблица задач сохраняет прежний примененный набор колонок. |

## 6. Правила выбора

| Выбранный пункт | Состояние черновика | Состояние таблицы задач |
|---|---|---|
| `Базовое отображение` | Черновик получает базовый набор колонок текущей области. | Таблица сохраняет прежний набор до `Применить`. |
| Сохраненный пресет | Черновик получает состав, порядок и видимость из пресета. | Таблица сохраняет прежний набор до `Применить`. |
| Сохраненный пресет с недоступной колонкой | Черновик получает только доступные динамические колонки. | Таблица сохраняет прежний набор до `Применить`. |

При выборе `Базовое отображение` черновик не содержит библиотечных добавлений. Если библиотека раскрыта, все доступные атрибуты библиотеки показывают кнопку действия с плюсом.
При выборе сохраненного пресета с библиотечными атрибутами черновик содержит эти атрибуты. Если библиотека раскрыта, строки этих атрибутов показывают кнопку действия со стрелкой назад.

## 7. Правила удаления

| Условие удаления | Видимый результат |
|---|---|
| Удаляется невыбранный сохраненный пресет | Пункт пресета исчезает из списка; поле текущего пресета сохраняет текущее значение. |
| Удаляется выбранный сохраненный пресет | Пункт пресета исчезает из списка; поле текущего пресета получает текст `Базовое отображение`; список колонок получает базовый набор текущей области. |
| Удаляется пресет, который был применен к таблице ранее | Таблица сохраняет текущий набор колонок до следующего `Применить`. |
| В списке не осталось сохраненных пресетов | В выпадающем списке виден только пункт `Базовое отображение`. |

## 8. Правила применения

| Действие | Видимый результат |
|---|---|
| Пользователь выбирает пресет без `Применить` | Меняется только список колонок внутри открытой панели. |
| Пользователь сохраняет пресет без `Применить` | Меняется только список пресетов внутри открытой панели. |
| Пользователь удаляет пресет без `Применить` | Меняется только выпадающий список пресетов и, при необходимости, черновик панели. |
| Пользователь нажимает `Применить` | Таблица задач получает состав, порядок и видимость колонок из текущего черновика панели. |
| Пользователь закрывает панель без `Применить` | Таблица задач сохраняет прежнее примененное состояние. |

## 9. Границы влияния

| Область | Правило |
|---|---|
| Табличный вид | После `Применить` получает состав, порядок и видимость колонок из пресета. |
| Карточный вид | Не зависит от пресетов колонок. |
| Данные задач | Не меняются. |
| Значения атрибутов | Не меняются. |
| Фильтры и поиск | Не меняются. |
| Библиотека атрибутов | Не удаляет атрибуты при сохранении, выборе или удалении пресета. |
| Текущая работа страницы | Сохраненные пресеты доступны в рамках текущей работы страницы. |
