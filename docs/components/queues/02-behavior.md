# Очереди: поведение

## 1. Основной пользовательский путь

1. Пользователь находится в обычном списке задач.
2. Над списком виден сегмент-контрол очередей.
3. По умолчанию активен сегмент `К исполнению`.
4. Список задач показывает задачи текущей области, которые относятся к очереди `К исполнению`.
5. Пользователь нажимает другой сегмент очереди.
6. Нажатый сегмент становится активным.
7. Список задач показывает задачи выбранной очереди.
8. Счетчики очередей сохраняют значения для текущей области, поиска и примененных фильтров.

## 2. Переключение очереди

| Действие пользователя | Состояние сегментов | Состояние списка задач |
|---|---|---|
| Нажимает неактивный сегмент | Все остальные сегменты теряют активное состояние; выбранный сегмент становится активным | Список показывает задачи выбранной очереди |
| Нажимает уже активный сегмент | Активный сегмент не меняется | Список задач не меняется |
| Переключает очередь при активных фильтрах | Выбранный сегмент становится активным | Список показывает задачи, которые одновременно относятся к выбранной очереди и проходят примененные фильтры |

Переключение очереди не меняет выбранную область слева и не сбрасывает поиск или примененные фильтры.

## 3. Связь с областью задач

| Выбранная область задач | Как применяется очередь |
|---|---|
| `Все задачи` | Список показывает задачи выбранной очереди по всем доступным бизнес-доменам |
| Конкретный бизнес-домен | Список показывает задачи выбранной очереди только внутри выбранного бизнес-домена |
| Сгруппированное значение | Список показывает задачи выбранной очереди внутри выбранного значения группы |

## 4. Связь с поиском и фильтрами

| Условие | Итоговый набор задач |
|---|---|
| Активна очередь, поиска нет, фильтров нет | Задачи выбранной очереди в текущей области |
| Активна очередь и введен поиск | Задачи выбранной очереди, совпадающие с текстом поиска |
| Активна очередь и применены фильтры | Задачи выбранной очереди, прошедшие все параметры фильтрации |
| Активна очередь, есть поиск и фильтры | Задачи выбранной очереди, которые одновременно совпадают с поиском и проходят фильтры |

## 5. Счетчики после действий

| Действие | Состояние счетчиков |
|---|---|
| Переключение очереди | Значения счетчиков не меняются только из-за самого переключения |
| Изменение области задач | Счетчики пересчитываются под новую область |
| Применение фильтров | Счетчики показывают количество задач, прошедших фильтры в каждой очереди |
| Сброс фильтров | Счетчики возвращаются к значениям текущей области без фильтрационного сужения |

## 6. Связь с режимом отображения

| Режим списка | Поведение очередей |
|---|---|
| Карточный вид | Активная очередь определяет набор карточек |
| Табличный вид | Активная очередь определяет набор строк таблицы |

Переключение между карточным и табличным видом не меняет активную очередь.

## 7. Массовые действия

| Условие | Доступность массовых действий |
|---|---|
| Выбраны задачи из очереди `К исполнению` | Доступно действие перехода в работу |
| Выбраны задачи из очереди `В работе` | Доступно действие завершения |
| Выбраны задачи из очереди `К назначению` | Доступно действие назначения |
| Выбраны задачи из разных очередей | Набор доступных действий зависит от очередей выбранных задач |
