# Очереди: структура интерфейса

## 1. Граница компонента

Очереди находятся в правой рабочей области над списком задач. В интерфейсе они выглядят как горизонтальный сегмент-контрол.

| Зона | Относится к очередям | Что видит пользователь |
|---|---|---|
| Горизонтальная панель сегментов | Да | Набор этапов согласования над списком задач |
| Сегмент очереди | Да | Название очереди и, если есть, счетчик |
| Активное состояние сегмента | Да | Выбранный этап согласования |
| Счетчик сегмента | Да | Количество задач в этой очереди для текущего контекста |
| Красная индикация счетчика | Да | Признак наличия просроченных задач в очереди `К исполнению` |
| Список задач ниже | Нет | Показывает задачи выбранной очереди |
| Панель инструментов выше | Нет | Содержит поиск, переключатель вида и действия |

Очередь не является статусом задачи. Очередь определяет, в каком сегменте над списком задач задача видна.

## 2. Расположение

| Положение | Значение |
|---|---|
| Рабочая область | Правая часть интерфейса |
| Вертикальное место | Под панелью инструментов, над списком задач |
| Компоновка | Горизонтальный ряд сегментов |
| Расстояние между сегментами | Сегменты отделены друг от друга одинаковым промежутком |

## 3. Состав сегментов

Компонент содержит шесть сегментов:

| Сегмент | Счетчик в текущем прототипе |
|---|---|
| `К исполнению` | Да |
| `В работе` | Да |
| `Обработаны мной` | Нет |
| `К назначению` | Да |
| `Наблюдаю` | Нет |
| `Инициированы мной` | Нет |

## 4. Активный сегмент

| Состояние | Визуальный признак |
|---|---|
| Сегмент активен | Светлая поверхность, акцентный текст, более заметная плотность текста |
| Сегмент неактивен | Нейтральный текст, без активного выделения |
| Наведение на сегмент | Фон сегмента становится немного темнее текущего обычного состояния |

По умолчанию активен сегмент `К исполнению`.

## 5. Счетчики

| Сегмент | Что показывает счетчик |
|---|---|
| `К исполнению` | Количество задач в очереди `К исполнению` с учетом текущего контекста |
| `В работе` | Количество задач в очереди `В работе` с учетом текущего контекста |
| `К назначению` | Количество задач в очереди `К назначению` с учетом текущего контекста |

Счетчики не показаны у сегментов `Обработаны мной`, `Наблюдаю` и `Инициированы мной`.

## 6. Красная индикация

| Условие | Визуальный результат |
|---|---|
| В очереди `К исполнению` есть просроченная задача | Счетчик сегмента получает красную индикацию |
| В очереди `К исполнению` нет просроченных задач | Счетчик остается в обычном состоянии |
| Просроченная задача находится в другой очереди | Красная индикация счетчика очереди не применяется |

## 7. Когда компонент скрыт

| Условие | Состояние очередей |
|---|---|
| Обычный список задач | Очереди видны |
| Пользовательская группа | Очереди скрыты |
| Кастомный список без переключения этапов | Очереди скрыты |
