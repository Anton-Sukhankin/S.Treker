# Очереди: данные, состояния и правила

## 1. Значения очередей

| Очередь | Системное значение | Участвует в отборе задач | Счетчик видим |
|---|---|---|---|
| `К исполнению` | `todo` | Да | Да |
| `В работе` | `in_progress` | Да | Да |
| `Обработаны мной` | `done` | Да | Нет |
| `К назначению` | `to_assign` | Да | Да |
| `Наблюдаю` | `review` | Да | Нет |
| `Инициированы мной` | `initiated` | Да | Нет |

Очередь не является статусом задачи. Статус задачи описывает состояние самой задачи; очередь определяет, в каком сегменте над списком задач она видна.

## 2. Активное состояние

| Состояние | Правило | Видимый результат |
|---|---|---|
| Очередь активна | Сегмент выбран пользователем или установлен по умолчанию | Сегмент визуально выделен; список показывает задачи этой очереди. |
| Очередь неактивна | Выбран другой сегмент | Сегмент не выделен; задачи этой очереди не видны в списке. |
| По умолчанию | Открыт обычный список задач | Активна очередь `К исполнению`. |

В каждый момент активна одна очередь.

## 3. Правило попадания задачи в очередь

| Условие | Итог |
|---|---|
| Значение очереди задачи совпадает с активной очередью | Задача может попасть в список после проверки области задач, поиска и фильтров. |
| Значение очереди задачи не совпадает с активной очередью | Задача не видна в списке. |

## 4. Правило счетчиков

Счетчик очереди показывает количество задач, которые одновременно проходят все условия.

| Условие | Требование |
|---|---|
| Очередь | Задача относится к этой очереди. |
| Область задач | Задача входит в выбранный бизнес-домен, `Все задачи`, группу или сгруппированное значение. |
| Поиск | Задача проходит текущую поисковую строку. |
| Фильтры | Задача проходит примененные параметры фильтрации. |

| Значение счетчика | Видимый результат |
|---|---|
| `0` | Счетчик показывает `0` и имеет пониженную визуальную насыщенность. |
| Больше `0` | Счетчик показывает количество подходящих задач. |

## 5. Правило красной индикации

| Условие | Видимый результат |
|---|---|
| Очередь `К исполнению`; счетчик больше `0`; среди подходящих задач есть просроченная | Счетчик получает красную индикацию. |
| Очередь `К исполнению`; просроченных задач нет | Счетчик остается в обычном состоянии. |
| Любая очередь кроме `К исполнению` | Красная индикация счетчика не рассчитывается. |

## 6. Контракт с соседними компонентами

| Соседний компонент | Роль соседнего компонента | Условие | Результат в очередях | Результат в соседнем компоненте |
|---|---|---|---|---|
| Панель инструментов | Соседняя область над очередями | Обычный список задач открыт | Очереди видны под панелью инструментов | Панель инструментов сохраняет поиск, переключатель вида и кнопки действий. |
| Левая навигация | Источник выбранной области задач | Пользователь выбирает `Все задачи`, бизнес-домен или значение группировки | Счетчики очередей показывают количество задач в выбранной области | Список задач показывает задачи активной очереди внутри выбранной области. |
| Левая навигация | Источник режима пользовательской группы | Пользователь выбирает пользовательскую группу | Очереди скрыты | Список задач показывает задачи выбранной пользовательской группы без сегментов очередей. |
| Панель инструментов | Источник поисковой строки | Пользователь вводит или очищает поиск | Счетчики очередей показывают количество задач с учетом поисковой строки | Список задач показывает задачи активной очереди с учетом поиска. |
| Панель фильтрации | Источник примененных параметров | Пользователь нажимает `Применить` или быстро сбрасывает фильтры | Счетчики очередей показывают количество задач с учетом примененных параметров | Список задач показывает задачи активной очереди с учетом фильтров. |
| Список задач | Получатель активной очереди | Пользователь выбирает сегмент очереди | Выбранный сегмент получает активное состояние | Список задач показывает задачи выбранной очереди. |
