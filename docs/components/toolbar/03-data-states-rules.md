# Панель инструментов списка задач: данные, состояния и правила

## 1. Владельцы состояния

| Область | Состояние или значение | Правило видимого результата |
|---|---|---|
| Панель инструментов | Поисковая строка, выбранный режим отображения, состояние кнопки `Фильтры` | Панель показывает текущее значение поиска, активный вариант переключателя вида и состояние кнопки `Фильтры`. |
| Панель фильтрации | Примененные параметры фильтрации и количество активных параметров | Кнопка `Фильтры` получает иконку фильтра или круглый счетчик на основе примененных параметров. |
| Список задач | Итоговый набор задач и активный режим отображения | Список показывает задачи с учетом поисковой строки и выбранного режима отображения. |
| Очереди | Активный этап согласования и счетчики этапов | Панель инструментов не владеет активным этапом и не рассчитывает счетчики этапов. |

## 2. Интерфейсные состояния панели

| Состояние | Пустое или базовое значение | Активное значение | Видимый результат |
|---|---|---|---|
| Поисковая строка | Поле пустое | Введен текст | Заполненная строка ограничивает список задач по значениям задачи. |
| Режим отображения | Карточный вид | Табличный вид | Активный вариант переключателя определяет, карточки или таблица видны в списке. |
| Параметры панели фильтрации | Все примененные параметры пустые | Активен минимум один примененный параметр | Кнопка `Фильтры` показывает иконку фильтра или круглый счетчик. |

## 3. Правила поиска

| Состояние поля `Поиск по задачам` | Правило для списка задач |
|---|---|
| Поле пустое | Поисковая строка не ограничивает список задач. |
| Поле заполнено | Задача остается в списке, если введенный текст найден минимум в одном простом значении задачи. |
| Поле заполнено и применены фильтры | Задача остается в списке только при выполнении поисковой строки и всех примененных параметров фильтрации. |
| Поле заполнено и активна очередь | Задача остается в списке только внутри активной очереди. |

Поиск не входит в счетчик активных параметров на кнопке `Фильтры`.

## 4. Правила кнопки `Фильтры`

| Состояние примененных параметров панели фильтрации | Значение счетчика | Вид кнопки |
|---|---:|---|
| Все параметры панели пустые | `0` | Иконка фильтра видима; круглый счетчик скрыт. |
| Активен один параметр панели | `1` | Иконка фильтра скрыта; круглый счетчик показывает `1`. |
| Активны несколько параметров панели | `N` | Иконка фильтра скрыта; круглый счетчик показывает количество активных параметров. |
| Активна только поисковая строка | `0` | Иконка фильтра видима; круглый счетчик скрыт. |

Быстрый сброс через круглый счетчик очищает примененные параметры панели фильтрации. Поисковая строка после быстрого сброса остается заполненной текущим значением.

## 5. Контракты взаимодействия

| Связанный компонент | Роль связанного компонента | Условие | Результат в панели инструментов | Результат в связанном компоненте |
|---|---|---|---|---|
| Список задач | Получатель поисковой строки | Пользователь вводит или очищает текст в поле `Поиск по задачам` | Поле содержит введенный текст или пустое значение | Список показывает задачи, которые проходят поисковую строку. |
| Список задач | Получатель выбранного режима отображения | Пользователь выбирает вариант в переключателе вида | Переключатель показывает выбранный вариант | Итоговый набор задач показан карточками или таблицей. |
| Панель фильтрации | Получатель действия открытия | Пользователь нажал кнопку `Фильтры` | Кнопка остается точкой входа в фильтрацию | Правая панель фильтрации открыта. |
| Панель фильтрации | Источник количества активных параметров | Активен минимум один примененный параметр фильтрации | Кнопка `Фильтры` показывает круглый счетчик `N` вместо иконки фильтра | Панель фильтрации хранит примененные параметры. |
| Панель фильтрации | Получатель быстрого сброса | Пользователь нажал круглый счетчик на кнопке `Фильтры` | Кнопка `Фильтры` показывает иконку фильтра; круглый счетчик скрыт | Примененные параметры фильтрации получают пустые значения. |
| Очереди | Соседняя область под панелью инструментов | Обычный список задач открыт | Панель инструментов показывает поиск, переключатель вида и кнопки действий | Очереди показывают сегменты этапов под панелью инструментов. |

## 6. Ограничения ответственности

| Область | Правило |
|---|---|
| Исходные данные задач | Панель инструментов не изменяет значения задач. |
| Бизнес-домены | Панель инструментов не изменяет состав бизнес-доменов. |
| Библиотека атрибутов | Панель инструментов не изменяет состав библиотеки атрибутов. |
| Пресеты колонок | Панель инструментов не создает, не сохраняет и не удаляет пресеты колонок. |
| Очереди | Панель инструментов не хранит активный этап и не рассчитывает счетчики этапов. |
