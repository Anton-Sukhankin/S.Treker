# Панель инструментов списка задач: данные, состояния и правила

## 1. Интерфейсные состояния панели

| Состояние | Пустое или базовое значение | Активное значение | Видимый результат |
|---|---|---|---|
| Поисковая строка | Поле пустое | Введен текст | Заполненная строка ограничивает список задач по значениям задачи. |
| Режим отображения | Карточный вид | Табличный вид | Активный режим определяет, карточки или таблица видны в списке. |
| Параметры панели фильтрации | Все параметры пустые | Активен минимум один параметр | Кнопка `Фильтры` показывает иконку или круглый счетчик. |

## 2. Правила поиска

| Состояние поля `Поиск по задачам` | Правило для списка задач |
|---|---|
| Поле пустое | Поисковая строка не ограничивает список задач. |
| Поле заполнено | Задача остается в списке, если введенный текст найден минимум в одном простом значении задачи. |
| Поле заполнено и применены фильтры | Задача остается в списке только при выполнении поисковой строки и всех примененных параметров фильтрации. |
| Поле заполнено и активна очередь | Задача остается в списке только внутри активной очереди. |

Поиск не входит в счетчик активных параметров на кнопке `Фильтры`.

## 3. Правила кнопки `Фильтры`

| Состояние параметров панели фильтрации | Значение счетчика | Вид кнопки |
|---|---:|---|
| Все параметры панели пустые | `0` | Иконка фильтра видима; круглый счетчик скрыт. |
| Активен один параметр панели | `1` | Иконка фильтра скрыта; круглый счетчик показывает `1`. |
| Активны несколько параметров панели | `N` | Иконка фильтра скрыта; круглый счетчик показывает количество активных параметров. |
| Активна только поисковая строка | `0` | Иконка фильтра видима; круглый счетчик скрыт. |

Быстрый сброс через круглый счетчик очищает параметры панели фильтрации. Поисковая строка после быстрого сброса сохраняет текущее значение.

## 4. Результат правил панели

| Изменение состояния панели | Результат в списке задач |
|---|---|
| Поисковая строка заполнена | Список показывает задачи, проходящие поисковую строку. |
| Выбран карточный вид | Итоговый набор задач показан карточками. |
| Выбран табличный вид | Итоговый набор задач показан таблицей. |
| Параметры фильтрации быстро сброшены через счетчик `Фильтры` | Список показывает набор без параметров панели фильтрации, но с учетом поисковой строки, выбранной области задач и активной очереди. |

Панель инструментов не меняет исходные данные задач, состав бизнес-доменов, состав библиотеки атрибутов и сохраненные пресеты колонок.

## 5. Граница с компонентом `Очереди`

| Область | Правило |
|---|---|
| Состав сегментов этапов | Описан в `queues/01-structure.md`. |
| Активный этап согласования | Описан в `queues/02-behavior.md` и `queues/03-data-states-rules.md`. |
| Счетчики этапов и красная индикация | Описаны в `queues/03-data-states-rules.md`. |

Панель инструментов не хранит активный этап и не рассчитывает счетчики этапов.
