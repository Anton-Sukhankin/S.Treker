# Панель инструментов списка задач: структура интерфейса

## 1. Расположение

| Область | Положение | Состав |
|---|---|---|
| Заголовок правого блока | Над панелью инструментов | Название текущей области задач. Не входит в панель инструментов. |
| Панель инструментов списка задач | Под заголовком правого блока, над очередями | Верхняя строка поиска и действий. |
| Очереди | Под панелью инструментов, над списком задач | Сегменты этапов согласования. Не входят в панель инструментов. |
| Список задач | Под очередями | Карточный список или табличный список. Не входит в панель инструментов. |

Панель инструментов находится в правой рабочей области. Левая навигация бизнес-доменов не входит в компонент.

## 2. Каркас компонента

| Зона панели | Позиция | Состав |
|---|---|---|
| Верхняя строка | Первая строка панели | Поле `Поиск по задачам`; переключатель вида; кнопки действий. |

## 3. Верхняя строка

| Элемент | Позиция | Условие показа | Видимое состояние |
|---|---|---|---|
| Поле `Поиск по задачам` | Слева | Всегда в обычном списке задач | Поле ввода с иконкой поиска и плейсхолдером. |
| Переключатель вида | Справа | Всегда в обычном списке задач | Единый сегмент-контрол с двумя сегментами: карточный вид и табличный вид. Активен один сегмент. |
| `Настройка колонок` | Справа после переключателя вида | Только в табличном виде | Иконка-кнопка открытия настроек колонок. |
| `Скачать` | Справа после `Настройка колонок` | Только в табличном виде | Иконка-кнопка скачивания. |
| `Печать` | Справа после `Скачать` в табличном виде; после переключателя вида в карточном виде | В карточном и табличном виде | Иконка-кнопка печати. |
| `Фильтры` | Справа после `Печать` | Всегда в обычном списке задач | Кнопка с текстом `Фильтры`; иконка фильтра или счетчик активных параметров. |
| `Создать задачу` | Справа последней кнопкой | Всегда в обычном списке задач | Основная кнопка с текстом `Создать задачу`. |

## 4. Состояния панели

| Состояние | Видимые части | Скрытые части |
|---|---|---|
| Карточный вид | Поиск; переключатель вида; `Печать`; `Фильтры`; `Создать задачу` | `Настройка колонок`; `Скачать`; табличный список. |
| Табличный вид | Поиск; переключатель вида; `Настройка колонок`; `Скачать`; `Печать`; `Фильтры`; `Создать задачу` | Карточный список. |
| Пользовательская группа | Переключатель вида, `Настройка колонок`, `Скачать`, `Печать` вынесены в область заголовка | `Создать задачу`. |

## 5. Внешние связи

| Внешняя область | Условие или действие | Конкретное правило |
|---|---|---|
| Левая навигация | Выбраны `Все задачи` или бизнес-домен | Панель показывает поиск, переключатель вида и кнопки действий. Очереди под панелью видны и описаны в `queues/01-structure.md`. |
| Левая навигация | Выбрана пользовательская группа | Кнопка `Создать задачу` скрыта; переключатель вида, `Настройка колонок`, `Скачать` и `Печать` находятся в области заголовка правого блока. Очереди скрыты по правилам `queues/01-structure.md`. |
| Заголовок правого блока | Выбрана пользовательская группа | В заголовке видны переключатель вида, `Настройка колонок`, `Скачать` и `Печать`. |
| Заголовок правого блока | Выбраны `Все задачи` или бизнес-домен | Переключатель вида, `Настройка колонок`, `Скачать` и `Печать` находятся в верхней строке панели. |
| Список задач | Выбран карточный вид | Список задач показывает карточки. |
| Список задач | Выбран табличный вид | Список задач показывает таблицу. |
| Список задач | В поле `Поиск по задачам` введен текст | Список задач показывает только задачи, где введенный текст найден в значениях задачи. |
| Панель фильтрации | Клик по `Фильтры` | Видны затемнение фона и правая панель фильтрации. |
| Кнопка `Фильтры` | В панели фильтрации есть активные параметры | Иконка фильтра скрыта; круглый счетчик активных параметров видим. |
| Список задач | Клик по круглому счетчику на кнопке `Фильтры` | Параметры панели фильтрации пустые; список задач сразу показывает результат без параметров панели фильтрации. |
| Настройки колонок | Клик по `Настройка колонок` в табличном виде | Видны затемнение фона и правая панель настройки колонок. |
| Системное уведомление | Клик по `Печать` | Видимо уведомление `Подготовка к печати...`. |
