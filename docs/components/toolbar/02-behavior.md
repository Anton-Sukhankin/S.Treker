# Панель инструментов списка задач: поведение

## 1. Основной пользовательский путь

1. Пользователь открывает обычный список задач.
2. Панель инструментов показывает поле поиска, переключатель вида, кнопки действий и сегменты этапов.
3. Пользователь вводит текст в `Поиск по задачам`.
4. Список задач показывает только задачи, где введенный текст найден в значениях задачи.
5. Пользователь выбирает `Табличный вид`.
6. Карточный список скрыт; табличный список видим; кнопки `Настройка колонок` и `Скачать` видимы.
7. Пользователь нажимает `Фильтры`.
8. Поверх рабочей области видны затемнение фона и правая панель фильтрации.
9. Пользователь выбирает сегмент этапа.
10. Список задач показывает задачи выбранного этапа с учетом текущего поиска и примененных фильтров.

## 2. Поиск

| Действие пользователя | Условие | Итог интерфейса |
|---|---|---|
| Ввод текста в `Поиск по задачам` | Введена минимум одна буква или цифра | Таблица или карточки показывают задачи, где введенный текст найден в простых текстовых или числовых значениях задачи. |
| Очистка поля `Поиск по задачам` | Поле было заполнено | Таблица или карточки показывают набор задач без ограничения поисковой строкой. |
| Ввод текста в `Поиск по задачам` | Открыта панель фильтрации | Панель фильтрации остается открытой; список задач за панелью получает результат поиска. |

Поиск не входит в счетчик активных параметров на кнопке `Фильтры`.

## 3. Переключение вида

| Действие пользователя | Условие | Итог интерфейса |
|---|---|---|
| Клик по иконке карточного вида | Активен табличный вид | Табличный список скрыт; карточный список видим; `Настройка колонок` скрыта; `Скачать` скрыта; `Печать` остается видимой. |
| Клик по иконке табличного вида | Активен карточный вид | Карточный список скрыт; табличный список видим; `Настройка колонок` видима; `Скачать` видима; `Печать` остается видимой. |
| Клик по активной иконке вида | Вид уже активен | Видимый список и состав кнопок не меняются. |

При переключении вида выбранная область задач, выбранный этап, поисковая строка, примененные параметры фильтрации и текущий набор задач сохраняются.

## 4. Кнопка `Фильтры`

| Действие пользователя | Условие | Итог интерфейса |
|---|---|---|
| Клик по кнопке `Фильтры` | Панель фильтрации закрыта | Затемнение фона видимо; правая панель фильтрации видима; область кнопки перекрыта панелью. |
| Клик по кнопке `Фильтры` | Есть активные параметры панели фильтрации | Панель открывается с текущими выбранными значениями параметров. |
| Клик по кнопке `Фильтры` | Активных параметров панели фильтрации нет | Панель открывается с пустыми значениями параметров. |

## 5. Индикатор активных фильтров на кнопке `Фильтры`

| Условие | Вид кнопки `Фильтры` |
|---|---|
| Активных параметров панели фильтрации нет | Видна иконка фильтра и текст `Фильтры`; круглый счетчик не виден. |
| Есть минимум один активный параметр панели фильтрации | Иконка фильтра скрыта; перед текстом `Фильтры` виден круглый счетчик с числом активных параметров. |
| Курсор наведен на круглый счетчик | Число внутри счетчика скрыто; внутри счетчика виден крестик. |

| Действие пользователя | Условие | Итог интерфейса |
|---|---|---|
| Клик по круглому счетчику на кнопке `Фильтры` | Есть минимум один активный параметр панели фильтрации | Все параметры панели фильтрации получают пустое значение; список задач сразу показывает набор без параметров панели фильтрации; поисковая строка сохраняет текущее значение. |
| Клик по текстовой части кнопки `Фильтры` | Есть минимум один активный параметр панели фильтрации | Правая панель фильтрации открыта; значения параметров не сброшены. |

## 6. Этапы над списком

| Действие пользователя | Условие | Итог интерфейса |
|---|---|---|
| Клик по неактивному сегменту этапа | Сегмент видим | Сегмент получает активное состояние; предыдущий активный сегмент теряет активное состояние; список задач показывает задачи выбранного этапа. |
| Клик по активному сегменту этапа | Сегмент уже активен | Активный этап и набор задач не меняются. |
| Переключение этапа | Есть поисковая строка или примененные фильтры | Список задач показывает задачи выбранного этапа, которые проходят текущий поиск и примененные параметры фильтрации. |

## 7. Кнопки действий

| Действие пользователя | Условие | Итог интерфейса |
|---|---|---|
| Клик по `Настройка колонок` | Активен табличный вид | Затемнение фона видимо; правая панель настройки колонок видима. |
| Клик по `Скачать` | Активен табличный вид | В текущем прототипе видимое состояние интерфейса не меняется. |
| Клик по `Печать` | Кнопка видима | Появляется системное уведомление `Подготовка к печати...`. |
| Клик по `Создать задачу` | Кнопка видима | В текущем прототипе видимое состояние интерфейса не меняется. |
