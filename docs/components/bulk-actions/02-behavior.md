# Массовые действия: поведение

## 1. Основной пользовательский путь

1. Пользователь находится в списке задач.
2. Пользователь выбирает чекбокс одной задачи в карточке или строке таблицы.
3. Задача получает выбранное состояние.
4. Функциональный блок `Массовые действия` показывает плавающую панель в нижней центральной части блока списка задач.
5. Бейдж в панели показывает `1`.
6. Пользователь нажимает `Переместить в группу`.
7. Поверх интерфейса видно модальное окно `Перемещение в группу`.
8. Пользователь вводит название новой группы.
9. Пользователь нажимает `Переместить`.
10. Модальное окно закрыто.
11. Левая навигация показывает новую пользовательскую группу с выбранной задачей.
12. Все выбранные задачи теряют выбранное состояние.
13. Плавающая панель массовых действий скрыта.
14. Toast-уведомление показывает текст `Задачи перемещены в "..."`.

## 2. Выбор одной задачи

| Действие пользователя | Условие | Итог интерфейса |
|---|---|---|
| Клик по пустому чекбоксу карточки | Карточный вид активен | Карточка получает выбранное состояние; плавающая панель видна; счетчик показывает количество выбранных задач. |
| Клик по пустому чекбоксу строки | Табличный вид активен | Строка получает выбранное состояние; плавающая панель видна; счетчик показывает количество выбранных задач. |
| Клик по выбранному чекбоксу карточки | Карточка выбрана | Карточка теряет выбранное состояние; счетчик панели получает новое количество выбранных задач. |
| Клик по выбранному чекбоксу строки | Строка выбрана | Строка теряет выбранное состояние; счетчик панели получает новое количество выбранных задач. |
| Снятие последнего выбранного чекбокса | После действия не остается выбранных задач | Плавающая панель скрыта; счетчик выбора получает значение `0`. |

Чекбоксы и выбранное состояние карточек и строк расположены в компоненте `Список задач`. Массовые действия используют полученный набор выбранных задач.

## 3. Выбор всех видимых строк таблицы

| Действие пользователя | Условие | Итог интерфейса |
|---|---|---|
| Клик по пустому чекбоксу в заголовке таблицы | Табличный вид активен; на текущей странице есть минимум одна видимая строка | Все видимые строки текущей страницы получают выбранное состояние; плавающая панель видна; счетчик показывает количество выбранных задач. |
| Клик по чекбоксу в заголовке таблицы | Все видимые строки текущей страницы выбраны | Все видимые строки текущей страницы теряют выбранное состояние; счетчик панели получает новое количество выбранных задач. |
| Клик по чекбоксу одной строки после выбора всех видимых строк | Часть видимых строк остается выбранной | Чекбокс в заголовке получает состояние частичного выбора; счетчик панели получает новое количество выбранных задач. |

Выбор всех через заголовок действует только на видимые строки текущей страницы таблицы. В карточном виде нет чекбокса выбора всех задач.

## 4. Реакция плавающей панели на выбранные задачи

| Условие | Итог интерфейса |
|---|---|
| Выбрана одна задача | Бейдж в панели показывает `1`. |
| Выбрано несколько задач | Бейдж в панели показывает количество выбранных задач. |
| Выбор снят со всех задач | Бейдж получает значение `0`; плавающая панель скрыта. |

При смене страницы, поиска, фильтров или области левой навигации набор выбранных задач продолжает содержать ранее выбранные задачи до явного снятия выбора или выполнения группового действия.

## 5. Снятие выделения

| Действие пользователя | Условие | Итог интерфейса |
|---|---|---|
| Клик по круглой кнопке с иконкой крестика в плавающей панели | Выбрана минимум одна задача | Все чекбоксы карточек, строк и заголовка таблицы становятся пустыми; карточки и строки теряют выбранное состояние; плавающая панель скрыта. |

## 6. Перемещение в группу

| Действие пользователя | Условие | Итог интерфейса |
|---|---|---|
| Клик по `Переместить в группу` | Выбрана минимум одна задача | Видно модальное окно `Перемещение в группу`; поле `Название новой группы` пустое. |
| Клик по `Переместить` | Поле `Название новой группы` содержит текст | Создана пользовательская группа с введенным названием; модальное окно закрыто; выбранные задачи теряют выбранное состояние; плавающая панель скрыта; левая навигация показывает новую группу. |
| Клик по `Переместить` | Поле `Название новой группы` пустое | Создана пользовательская группа с названием `Группа`; модальное окно закрыто; выбранные задачи теряют выбранное состояние; плавающая панель скрыта; левая навигация показывает новую группу. |
| Клик по кнопке закрытия модального окна | Модальное окно открыто | Модальное окно закрыто; выбранные чекбоксы остаются выбранными; плавающая панель остается видимой. |
| Клик по `Отмена` | Модальное окно открыто | Модальное окно закрыто; выбранные чекбоксы остаются выбранными; плавающая панель остается видимой. |

Создание группы не удаляет задачи из исходных бизнес-доменов.

## 7. Удаление из пользовательской группы

| Действие пользователя | Условие | Итог интерфейса |
|---|---|---|
| Клик по `Удалить из группы` | Активна пользовательская группа; выбрана минимум одна задача из этой группы | Выбранные задачи удалены из пользовательской группы; выбранные задачи теряют выбранное состояние; плавающая панель скрыта. |
| Клик по `Удалить из группы` | После удаления в пользовательской группе остается минимум одна задача | Левая навигация содержит пункт этой группы; список задач показывает оставшиеся задачи группы. |
| Клик по `Удалить из группы` | После удаления в пользовательской группе не остается задач | Пункт группы исчезает из левой навигации; активная область получает значение `Все задачи`; список задач показывает задачи всех доступных бизнес-доменов. |
| Клик по `Удалить из группы` | Активно сгруппированное значение, а не пользовательская группа | Плавающая панель остается видимой; выбранные чекбоксы остаются выбранными; список задач остается на текущем сгруппированном значении; активный пункт левой навигации не меняется. |
| Клик по `Удалить из группы` | Активна группа внутри закладки, а не пользовательская группа | Плавающая панель остается видимой; выбранные чекбоксы остаются выбранными; список задач остается на текущей группе внутри закладки; активный пункт левой навигации не меняется. |

После удаления каждой выбранной задачи из группы toast-уведомление показывает сообщение об удалении этой задачи. Если группа стала пустой, следующее toast-уведомление сообщает, что группа удалена.
