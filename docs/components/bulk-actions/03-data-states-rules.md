# Массовые действия: данные, состояния и правила

## 1. Владельцы состояния

| Область | Состояние или значение | Правило видимого результата |
|---|---|---|
| Список задач | Чекбоксы карточек, чекбоксы строк, чекбокс заголовка таблицы, выбранное состояние карточек и строк | Чекбокс задачи задает выбранность этой задачи; карточка или строка показывает выбранное состояние этой задачи. |
| Функциональный блок `Массовые действия` | Видимость плавающей панели, число в счетчике, состав кнопок, состояние модального окна `Перемещение в группу` | При количестве выбранных задач больше `0` плавающая панель видна; счетчик панели равен количеству выбранных задач. |
| Левая навигация | Пользовательские группы и активная область задач | После создания пользовательской группы левая навигация содержит новый пункт группы; после удаления последней задачи из активной пользовательской группы этот пункт отсутствует, активная область получает значение `Все задачи`. |
| Список задач как получатель результата | Набор задач в правой рабочей области | При выборе пользовательской группы список задач содержит задачи этой группы; после удаления пустой активной пользовательской группы список задач содержит задачи из `Все задачи`. |
| Toast-уведомление | Текст результата действия | После создания группы toast показывает текст `Задачи перемещены в "Название группы"`; после удаления задач из группы toast показывает сообщение об удалении. |

## 2. Состояния выбора

| Состояние | Условие | Видимый результат |
|---|---|---|
| Пустой выбор | Не выбрана ни одна задача | Все чекбоксы пустые; карточки и строки не имеют выбранного состояния; плавающая панель скрыта. |
| Одна выбранная задача | В списке задач выбран один чекбокс задачи | Чекбокс задачи выбран; карточка или строка имеет выбранное состояние; плавающая панель видна; счетчик показывает `1`. |
| Несколько выбранных задач | В списке задач выбрано два или больше чекбоксов задач | Каждый выбранный чекбокс активен; каждая выбранная карточка или строка имеет выбранное состояние; плавающая панель видна; счетчик показывает количество выбранных задач. |
| Все видимые строки таблицы выбраны | Количество выбранных видимых строк равно количеству видимых строк текущей страницы | Чекбокс в заголовке таблицы выбран. |
| Часть видимых строк таблицы выбрана | Выбрана минимум одна, но не все видимые строки текущей страницы | Чекбокс в заголовке таблицы имеет состояние частичного выбора; внутри чекбокса виден синий квадрат `12 x 12 px` по центру. |

## 3. Область действия выбора

| Область | Правило |
|---|---|
| Карточный вид | Чекбокс карточки добавляет одну задачу в набор выбранных задач или убирает ее из набора. |
| Табличный вид | Чекбокс строки добавляет одну задачу в набор выбранных задач или убирает ее из набора. |
| Текущая страница таблицы | Чекбокс заголовка таблицы добавляет в набор выбранных задач только видимые строки текущей страницы или убирает их из набора. |
| Невидимые задачи | Если выбранная задача стала невидимой после смены страницы, поиска, фильтров или области левой навигации, значение счетчика продолжает включать эту задачу в общее количество выбранных задач. |
| Очистка выбора | Крестик в плавающей панели устанавливает количество выбранных задач в `0`; все чекбоксы списка задач становятся пустыми; плавающая панель скрыта. |

## 4. Видимость плавающей панели и счетчик

| Условие | Видимость панели | Значение счетчика |
|---|---|---|
| Количество выбранных задач равно `0` | Скрыта | `0`. |
| Количество выбранных задач больше `0` | Видна | Количество выбранных задач. |

Плавающая панель не задает выбранное состояние карточек и строк. Выбранное состояние карточки или строки задается чекбоксом этой задачи, чекбоксом заголовка таблицы или крестиком снятия выбора в плавающей панели.

## 5. Видимость кнопок плавающей панели

| Кнопка | Условие видимости |
|---|---|
| `Изменить статус` | Выбрана минимум одна задача. |
| `Переместить в группу` | Выбрана минимум одна задача. |
| `Назначить` | Среди выбранных задач есть минимум одна задача из этапа `К назначению`. |
| `Взять в работу` | Среди выбранных задач есть минимум одна задача из этапа `К исполнению`. |
| `Завершить` | Среди выбранных задач есть минимум одна задача из этапа `В работе`. |
| `Удалить из группы` | Активная область относится к пользовательской группе, закладочной группе или сгруппированному значению. |
| Крестик снятия выбора | Выбрана минимум одна задача. |

Если выбранные задачи относятся к нескольким этапам, плавающая панель содержит несколько контекстных кнопок одновременно.

## 6. Состояния модального окна `Перемещение в группу`

| Состояние | Условие | Видимый результат |
|---|---|---|
| Закрыто | Модальное окно не открыто или пользователь закрыл его | Модальное окно не видно; выбранные чекбоксы остаются в состоянии, которое было до закрытия окна. |
| Открыто | Пользователь нажал `Переместить в группу` | Видны заголовок `Перемещение в группу`, описание, поле названия группы, кнопки `Отмена` и `Переместить`. |
| Поле названия пустое | Пользователь нажимает `Переместить` | Новая группа получает название `Группа`. |
| Поле названия заполнено | Пользователь нажимает `Переместить` | Новая группа получает введенное название. |

## 7. Контракты взаимодействия

| Действие или условие | Источник | Передаваемый результат | Получатель | Видимый итог |
|---|---|---|---|---|
| Выбор одной задачи | Чекбокс карточки или строки в списке задач | Набор выбранных задач больше `0` | Плавающая панель массовых действий | Панель видна; счетчик равен количеству выбранных задач. |
| Выбор всех видимых строк текущей страницы | Чекбокс заголовка таблицы | Видимые строки текущей страницы входят в набор выбранных задач | Плавающая панель массовых действий | Панель видна; счетчик равен количеству выбранных задач. |
| Снятие выбора через крестик панели | Крестик в плавающей панели | Набор выбранных задач равен `0` | Список задач и плавающая панель | Чекбоксы пустые; карточки и строки не имеют выбранного состояния; панель скрыта. |
| Подтверждение перемещения в группу | Модальное окно `Перемещение в группу` | Название группы и набор выбранных задач | Левая навигация | Левая навигация содержит новую пользовательскую группу. |
| Подтверждение перемещения в группу | Модальное окно `Перемещение в группу` | Набор выбранных задач равен `0` после создания группы | Список задач и плавающая панель | Выбранные задачи не имеют выбранного состояния; панель скрыта. |
| Создание пользовательской группы | Модальное окно `Перемещение в группу` | Название созданной группы | Toast-уведомление | Toast показывает текст `Задачи перемещены в "Название группы"`. |
| Удаление из активной пользовательской группы | Кнопка `Удалить из группы` | Выбранные задачи исключены из активной пользовательской группы | Левая навигация и список задач | Левая навигация и список задач показывают состояние группы после удаления выбранных задач. |

## 8. Правила создания пользовательской группы

| Условие | Видимый результат |
|---|---|
| Пользователь нажал `Переместить` в открытом модальном окне; поле `Название новой группы` содержит текст | Левая навигация содержит новую пользовательскую группу с введенным названием. |
| Пользователь нажал `Переместить` в открытом модальном окне; поле `Название новой группы` пустое | Левая навигация содержит новую пользовательскую группу с названием `Группа`. |
| Пользовательская группа создана из выбранных задач | Задачи остаются в своих исходных бизнес-доменах. |
| Пользовательская группа создана из выбранных задач | Все выбранные чекбоксы становятся пустыми; карточки и строки не имеют выбранного состояния; плавающая панель скрыта. |
| Пользовательская группа создана из выбранных задач | Toast-уведомление показывает текст `Задачи перемещены в "Название группы"`. |

## 9. Правила удаления из группы

| Условие | Видимый результат |
|---|---|
| Активна пользовательская группа; выбраны задачи этой группы; пользователь нажал `Удалить из группы` | Выбранные задачи отсутствуют в пользовательской группе; все выбранные чекбоксы становятся пустыми; плавающая панель скрыта. |
| После удаления в пользовательской группе есть минимум одна задача | Левая навигация содержит пункт группы; список задач содержит оставшиеся задачи группы. |
| После удаления в пользовательской группе нет задач | Левая навигация не содержит пункт этой группы; активная область получает значение `Все задачи`; список задач содержит задачи из `Все задачи`. |
| Активная область является сгруппированным значением; пользователь нажал `Удалить из группы` | Плавающая панель остается видимой; выбранные чекбоксы остаются выбранными; список задач остается на текущем сгруппированном значении; активный пункт левой навигации не меняется. |
| Активная область является группой внутри закладки; пользователь нажал `Удалить из группы` | Плавающая панель остается видимой; выбранные чекбоксы остаются выбранными; список задач остается на текущей группе внутри закладки; активный пункт левой навигации не меняется. |

## 10. Ограничения ответственности

| Область | Правило |
|---|---|
| Данные задач | Массовые действия не изменяют атрибуты задач. |
| Бизнес-домены | Массовые действия не удаляют задачи из бизнес-доменов. |
| Очереди | Клики по `Изменить статус`, `Назначить`, `Взять в работу`, `Завершить` в текущем прототипе не изменяют этап задачи. |
| Левая навигация | Создание группы добавляет пользовательскую группу; удаление последней задачи из активной пользовательской группы убирает пункт этой группы. |
| Фильтры и поиск | Массовые действия не изменяют параметры фильтрации и значение поисковой строки. |
| Пагинация | Массовые действия не устанавливают новое значение текущей страницы. |
| Настройки колонок | Массовые действия не изменяют состав и порядок динамических колонок. |
