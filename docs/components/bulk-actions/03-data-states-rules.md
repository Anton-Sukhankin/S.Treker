# Массовые действия: данные, состояния и правила

## 1. Владельцы состояния

| Область | Чем владеет | Видимый результат |
|---|---|---|
| Список задач | Чекбоксы карточек, чекбоксы строк, чекбокс заголовка таблицы, выбранное состояние карточек и строк | Пользователь видит, какие задачи выбраны. |
| Функциональный блок `Массовые действия` | Видимость плавающей панели, значение счетчика, состав кнопок, состояние модального окна `Перемещение в группу` | Пользователь видит действия, доступные для текущего набора выбранных задач. |
| Левая навигация | Пользовательские группы и активная область задач | После создания группы показывает новую группу; после удаления пустой активной группы скрывает ее. |
| Список задач как получатель результата | Итоговый набор задач после выбора области или удаления из группы | После выбора пользовательской группы показывает задачи группы; после удаления пустой группы показывает `Все задачи`. |
| Toast-уведомление | Текст результата действия | Показывает сообщение о создании группы или удалении задач из группы. |

## 2. Входное состояние выбора

| Состояние | Условие | Видимый результат |
|---|---|---|
| Пустой выбор | Не выбрана ни одна задача | Все чекбоксы пустые; плавающая панель скрыта. |
| Одна выбранная задача | В списке задач выбран один чекбокс задачи | Чекбокс активен; карточка или строка получает выбранное состояние; плавающая панель видна; счетчик показывает `1`. |
| Несколько выбранных задач | В списке задач выбрано два или больше чекбоксов задач | Все выбранные карточки или строки получают выбранное состояние; плавающая панель видна; счетчик показывает количество выбранных задач. |
| Все видимые строки таблицы выбраны | Число выбранных видимых строк равно числу видимых строк текущей страницы | Чекбокс в заголовке таблицы получает выбранное состояние. |
| Часть видимых строк таблицы выбрана | Выбрана минимум одна, но не все видимые строки текущей страницы | Чекбокс в заголовке таблицы получает состояние частичного выбора. |

## 3. Область действия выбора

| Область | Правило |
|---|---|
| Карточный вид | Выбор доступен через чекбокс каждой карточки. |
| Табличный вид | Выбор доступен через чекбокс каждой строки и чекбокс заголовка таблицы. |
| Текущая страница таблицы | Чекбокс заголовка выбирает только видимые строки текущей страницы. |
| Невидимые задачи | Если задача выбрана, но не видна после смены страницы, поиска, фильтров или области, счетчик панели сохраняет эту задачу в общем количестве выбранных задач. |

## 4. Видимость плавающей панели и счетчик

| Условие | Видимость панели | Значение счетчика |
|---|---|---|
| Количество выбранных задач равно `0` | Скрыта | `0`. |
| Количество выбранных задач больше `0` | Видна | Количество выбранных задач. |

Плавающая панель не меняет выбранное состояние карточек и строк сама по себе. Выбранное состояние меняется после выбора чекбокса, снятия выбора через крестик панели или выполнения группового действия.

## 5. Видимость кнопок плавающей панели

| Кнопка | Условие видимости |
|---|---|
| `Изменить статус` | Выбрана минимум одна задача. |
| `Переместить в группу` | Выбрана минимум одна задача. |
| `Назначить` | Среди выбранных задач есть минимум одна задача из этапа `К назначению`. |
| `Взять в работу` | Среди выбранных задач есть минимум одна задача из этапа `К исполнению`. |
| `Завершить` | Среди выбранных задач есть минимум одна задача из этапа `В работе`. |
| `Удалить из группы` | Активная область относится к пользовательской группе, закладочной группе или сгруппированному значению. |
| Крестик снятия выделения | Выбрана минимум одна задача. |

Если выбранные задачи относятся к нескольким этапам, панель может показывать несколько контекстных кнопок одновременно.

## 6. Состояния модального окна `Перемещение в группу`

| Состояние | Условие | Видимый результат |
|---|---|---|
| Закрыто | Пользователь не нажимал `Переместить в группу` или закрыл окно | Модальное окно не видно; выбранные задачи сохраняют текущее состояние. |
| Открыто | Пользователь нажал `Переместить в группу` | Видны заголовок `Перемещение в группу`, описание, поле названия группы, `Отмена`, `Переместить`. |
| Поле названия пустое | Пользователь нажимает `Переместить` | Новая группа получает название `Группа`. |
| Поле названия заполнено | Пользователь нажимает `Переместить` | Новая группа получает введенное название. |

## 7. Контракты взаимодействия

| Тип связи | Источник | Передаваемый результат | Получатель | Видимый итог |
|---|---|---|---|---|
| Запуск выбора | Чекбокс карточки или строки в списке задач | Набор выбранных задач больше `0` | Плавающая панель массовых действий | Панель видна; счетчик показывает количество выбранных задач. |
| Выбор всех видимых строк | Чекбокс заголовка таблицы | Все видимые строки текущей страницы входят в набор выбранных задач | Плавающая панель массовых действий | Панель видна; счетчик показывает количество выбранных задач. |
| Снятие выбора | Крестик в плавающей панели | Набор выбранных задач равен `0` | Список задач и плавающая панель | Чекбоксы пустые; карточки и строки теряют выбранное состояние; панель скрыта. |
| Создание группы | Модальное окно `Перемещение в группу` | Название группы и набор выбранных задач | Левая навигация | Новая пользовательская группа видна в левой навигации. |
| Завершение группового действия | Плавающая панель массовых действий | Набор выбранных задач равен `0` | Список задач и плавающая панель | Выбранные задачи теряют выбранное состояние; панель скрыта. |
| Уведомление результата | Массовое действие выполнено | Текст результата | Toast-уведомление | Пользователь видит сообщение о выполненном действии. |

## 8. Правила создания пользовательской группы

| Условие | Видимый результат |
|---|---|
| Группа создана из выбранных задач | Левая навигация показывает блок пользовательских групп и новый пункт группы. |
| Группа создана | Выбранные задачи теряют выбранное состояние; плавающая панель скрыта. |
| Группа создана | Toast-уведомление показывает текст `Задачи перемещены в "Название группы"`. |
| Группа создана | Задачи остаются в своих исходных бизнес-доменах. |

## 9. Правила удаления из группы

| Условие | Видимый результат |
|---|---|
| Активна пользовательская группа, выбраны задачи этой группы, нажато `Удалить из группы` | Выбранные задачи отсутствуют в пользовательской группе. |
| После удаления в группе есть задачи | Левая навигация сохраняет пункт группы; список задач показывает оставшиеся задачи группы. |
| После удаления в группе нет задач | Левая навигация не показывает пункт этой группы; активная область получает значение `Все задачи`. |
| Активная область является сгруппированным значением или группой внутри закладки | Панель, выбранные задачи, список задач и левая навигация сохраняют текущее состояние. |

## 10. Границы влияния

| Область | Правило |
|---|---|
| Данные задач | Массовые действия не меняют атрибуты задач. |
| Бизнес-домены | Массовые действия не удаляют задачи из бизнес-доменов. |
| Очереди | Кнопки `Изменить статус`, `Назначить`, `Взять в работу`, `Завершить` в текущем прототипе не меняют этап задачи. |
| Левая навигация | Создание группы добавляет пользовательскую группу; удаление пустой активной группы убирает ее из левой навигации. |
| Фильтры и поиск | Массовые действия не меняют параметры фильтрации и поисковую строку. |
| Пагинация | Массовые действия не меняют текущую страницу напрямую. |
| Настройки колонок | Массовые действия не меняют состав и порядок динамических колонок. |
