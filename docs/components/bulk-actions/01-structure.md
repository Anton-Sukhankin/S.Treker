# Массовые действия: структура интерфейса

## 1. Расположение

Массовые действия связаны с выбором задач в карточном и табличном списке. После выбора минимум одной задачи внизу экрана видна плавающая панель действий.

| Область | Положение | Состав |
|---|---|---|
| Карточный список задач | Правая рабочая область | Чекбокс выбора в каждой карточке задачи. |
| Табличный список задач | Правая рабочая область | Чекбокс выбора в каждой строке; чекбокс выбора всех видимых строк в заголовке таблицы. |
| Плавающая панель массовых действий | Нижняя центральная часть экрана, поверх интерфейса | Счетчик выбранных задач; кнопки действий; кнопка снятия выделения. |
| Модальное окно `Перемещение в группу` | Поверх интерфейса | Хедер, описание, поле названия группы, кнопки `Отмена` и `Переместить`. |

## 2. Граница компонента

| Объект | Статус относительно массовых действий | Где описано | Правило |
|---|---|---|---|
| Чекбокс выбора задачи в карточке | Входит | В этом файле; поведение описано в `02-behavior.md`. | Запускает выбор одной задачи в карточном виде. |
| Чекбокс выбора задачи в строке таблицы | Входит | В этом файле; поведение описано в `02-behavior.md`. | Запускает выбор одной задачи в табличном виде. |
| Чекбокс выбора всех видимых строк таблицы | Входит | В этом файле; поведение описано в `02-behavior.md`. | Запускает выбор всех видимых строк текущей страницы. |
| Выбранное состояние карточки или строки | Входит | В этом файле; правила состояния описаны в `03-data-states-rules.md`. | Показывает, какие задачи входят в текущий набор выбранных задач. |
| Плавающая панель массовых действий | Входит | В этом файле. | Показывает счетчик выбранных задач и доступные массовые действия. |
| Счетчик выбранных задач | Входит | В этом файле; правила подсчета описаны в `03-data-states-rules.md`. | Показывает количество выбранных задач. |
| Кнопки массовых действий | Входят | В этом файле; реакции описаны в `02-behavior.md`. | Запускают действия над выбранными задачами. |
| Модальное окно `Перемещение в группу` | Входит | В этом файле; поведение описано в `02-behavior.md`. | Создает пользовательскую группу через сценарий массового действия. |
| Данные задачи и значения атрибутов | Не отдельный UI-компонент | Источник данных описан в `src/data`; связь с кодом указана в `docs/component-code-map.md`. | Массовые действия не редактируют значения атрибутов. |
| Фильтрация задач | Соседний компонент, задает видимый набор задач | [`filters/01-structure.md`](../filters/01-structure.md) | Массовые действия работают только с задачами, которые видны после применения фильтров. |
| Настройки колонок | Соседний компонент, влияет на табличное отображение | [`column-settings/01-structure.md`](../column-settings/01-structure.md) | Массовые действия не меняют состав и порядок колонок. |
| Пагинация | Соседний компонент, задает текущую страницу | [`pagination/01-structure.md`](../pagination/01-structure.md) | Выбор всех строк относится к видимым строкам текущей страницы. |
| Левая навигация | Соседний компонент, получает результат создания или удаления группы | [`left-navigation/01-structure.md`](../left-navigation/01-structure.md) | После создания группы или удаления пустой активной группы левая навигация показывает результат. |
| Очереди | Соседний компонент, задает контекст списка задач | [`queues/01-structure.md`](../queues/01-structure.md) | Массовые действия не меняют состав, порядок и выбранность этапов. |
| Создание задачи | Не входит в текущий компонент | Нет отдельной спецификации в текущем комплекте компонентов. | Массовые действия работают с уже существующими задачами. |

Массовые действия не меняют состав задач в обычных бизнес-доменах. Создание группы добавляет пользовательскую группу в левую навигацию.

## 3. Чекбокс выбора задачи

| Место | Условие показа | Видимое состояние |
|---|---|---|
| Карточка задачи | Карточный вид активен | Чекбокс виден слева в карточке. |
| Строка таблицы | Табличный вид активен | Чекбокс виден в первой закрепленной колонке строки. |
| Заголовок таблицы | Табличный вид активен | Чекбокс виден в первой закрепленной колонке заголовка. |

| Состояние чекбокса | Видимый результат |
|---|---|
| Не выбран | Пустой чекбокс. |
| Выбран | Чекбокс с синей заливкой и белой галочкой. |
| Частичный выбор в заголовке таблицы | Чекбокс с синим квадратом 12 x 12 px по центру. |

## 4. Выбранное состояние задачи

| Область | Условие | Видимый результат |
|---|---|---|
| Карточка задачи | Чекбокс карточки выбран | Карточка получает выбранное состояние. |
| Строка таблицы | Чекбокс строки выбран | Строка получает выбранное состояние. |
| Карточка или строка | Чекбокс снят | Карточка или строка теряет выбранное состояние. |

## 5. Плавающая панель массовых действий

| Часть панели | Состав |
|---|---|
| Счетчик результата | Текст `Результат:`; бейдж с числом выбранных задач. |
| Основные действия | `Изменить статус`; `Переместить в группу`. |
| Контекстные действия | `Назначить`; `Взять в работу`; `Завершить`; `Удалить из группы`. |
| Снятие выделения | Круглая кнопка с крестиком справа. |

| Состояние выбора | Видимость панели |
|---|---|
| Не выбрана ни одна задача | Плавающая панель скрыта. |
| Выбрана минимум одна задача | Плавающая панель видна внизу экрана. |

## 6. Кнопки панели

| Кнопка | Условие показа | Текущий видимый результат при клике |
|---|---|---|
| `Изменить статус` | Выбрана минимум одна задача | Панель, выбранные задачи, список задач и левая навигация сохраняют текущее состояние. |
| `Переместить в группу` | Выбрана минимум одна задача | Открыто модальное окно `Перемещение в группу`. |
| `Назначить` | Среди выбранных задач есть задача из этапа `К назначению` | Панель, выбранные задачи, список задач и левая навигация сохраняют текущее состояние. |
| `Взять в работу` | Среди выбранных задач есть задача из этапа `К исполнению` | Панель, выбранные задачи, список задач и левая навигация сохраняют текущее состояние. |
| `Завершить` | Среди выбранных задач есть задача из этапа `В работе` | Панель, выбранные задачи, список задач и левая навигация сохраняют текущее состояние. |
| `Удалить из группы` | Выбрана пользовательская группа, закладочная группа или сгруппированное значение | При активной пользовательской группе выбранные задачи удаляются из группы; в остальных видимых состояниях панель, выбранные задачи, список задач и левая навигация сохраняют текущее состояние. |
| Крестик снятия выделения | Выбрана минимум одна задача | Все выбранные задачи теряют выбранное состояние; панель скрыта. |

## 7. Модальное окно `Перемещение в группу`

| Часть модального окна | Состав |
|---|---|
| Хедер | Заголовок `Перемещение в группу`; кнопка закрытия. |
| Описание | Текст `Задачи будут объединены в новую кастомную группу.` |
| Поле | Подпись `Название новой группы`; поле ввода с плейсхолдером `Напр: Срочный ремонт БЦ`. |
| Футер | Кнопка `Отмена`; кнопка `Переместить`. |

Модальное окно открывается после клика по `Переместить в группу`.

## 8. Соседние области

| Соседняя область | Связь с массовыми действиями |
|---|---|
| Список задач | Передает выбранные задачи через чекбоксы карточек и строк. |
| Заголовок таблицы | Передает выбор всех видимых строк текущей страницы. |
| Левая навигация | После создания группы показывает новую пользовательскую группу. |
| Левая навигация | После удаления всех задач из активной пользовательской группы возвращает активную область к `Все задачи`. |
| Toast-уведомление | После создания группы или удаления задач из группы показывает текст результата. |
