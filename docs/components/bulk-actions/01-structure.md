# Массовые действия: структура интерфейса

## 1. Граница функционального блока

`Массовые действия` — функциональный блок для действий над выбранными задачами. Главный UI блока — плавающая панель массовых действий. Чекбоксы выбора расположены в списке задач и запускают состояние выбора, но не являются частью плавающей панели.

| Объект | Роль в функциональности | Где расположен | Правило |
|---|---|---|---|
| Чекбокс карточки задачи | Точка запуска выбора | Карточка задачи | После выбора задача входит в набор выбранных задач; после повторного выбора задача выходит из набора. |
| Чекбокс строки таблицы | Точка запуска выбора | Первая закрепленная колонка строки таблицы | После выбора задача входит в набор выбранных задач; после повторного выбора задача выходит из набора. |
| Чекбокс заголовка таблицы | Точка запуска выбора всех видимых строк | Первая закрепленная колонка заголовка таблицы | После выбора в набор выбранных задач входят все видимые строки текущей страницы. |
| Выбранное состояние карточки или строки | Состояние элемента списка задач | Карточка или строка таблицы | Показывает, что задача входит в набор выбранных задач. |
| Плавающая панель массовых действий | Главный UI функционального блока | Нижняя центральная часть экрана поверх интерфейса | Видна, если выбрана минимум одна задача. |
| Счетчик выбранных задач | Элемент плавающей панели | Плавающая панель | Показывает количество выбранных задач. |
| Кнопки массовых действий | Элементы плавающей панели | Плавающая панель | Запускают действия над набором выбранных задач. |
| Модальное окно `Перемещение в группу` | Дочерний UI функционального блока | Поверх интерфейса | Открывается после действия `Переместить в группу`. |
| Левая навигация | Внешний получатель результата | Левая панель | Показывает созданную пользовательскую группу или убирает пустую активную группу. |
| Toast-уведомление | Внешний получатель результата | Системная область уведомлений | Показывает результат создания группы или удаления задач из группы. |
| Список задач | Внешний получатель результата | Правая рабочая область | Снимает выбранное состояние задач после выполненного группового действия. |

## 2. Точки запуска выбора

| Точка запуска | Условие видимости | Результат для функционального блока |
|---|---|---|
| Чекбокс карточки задачи | Активен карточный вид | Набор выбранных задач получает выбранную задачу или теряет ее. |
| Чекбокс строки таблицы | Активен табличный вид | Набор выбранных задач получает выбранную задачу или теряет ее. |
| Чекбокс заголовка таблицы | Активен табличный вид; на текущей странице есть минимум одна видимая строка | Набор выбранных задач получает все видимые строки текущей страницы или теряет их. |

## 3. Плавающая панель массовых действий

| Часть панели | Состав |
|---|---|
| Счетчик результата | Текст `Результат:`; бейдж с числом выбранных задач. |
| Основные действия | `Изменить статус`; `Переместить в группу`. |
| Контекстные действия | `Назначить`; `Взять в работу`; `Завершить`; `Удалить из группы`. |
| Снятие выделения | Круглая кнопка с крестиком справа. |

| Состояние выбора | Видимость панели |
|---|---|
| Не выбрана ни одна задача | Плавающая панель скрыта. |
| Выбрана минимум одна задача | Плавающая панель видна внизу экрана поверх интерфейса. |

## 4. Кнопки плавающей панели

| Кнопка | Условие показа | Роль |
|---|---|---|
| `Изменить статус` | Выбрана минимум одна задача | Кнопка без реализованного видимого действия в текущем прототипе. |
| `Переместить в группу` | Выбрана минимум одна задача | Открывает модальное окно `Перемещение в группу`. |
| `Назначить` | Среди выбранных задач есть задача из этапа `К назначению` | Кнопка без реализованного видимого действия в текущем прототипе. |
| `Взять в работу` | Среди выбранных задач есть задача из этапа `К исполнению` | Кнопка без реализованного видимого действия в текущем прототипе. |
| `Завершить` | Среди выбранных задач есть задача из этапа `В работе` | Кнопка без реализованного видимого действия в текущем прототипе. |
| `Удалить из группы` | Активная область относится к пользовательской группе, закладочной группе или сгруппированному значению | Удаляет выбранные задачи только из пользовательской группы; в остальных указанных областях видимое состояние не меняется. |
| Крестик снятия выделения | Выбрана минимум одна задача | Снимает выбор со всех выбранных задач. |

## 5. Дочерний UI: модальное окно `Перемещение в группу`

| Часть модального окна | Состав |
|---|---|
| Хедер | Заголовок `Перемещение в группу`; кнопка закрытия. |
| Описание | Текст `Задачи будут объединены в новую кастомную группу.` |
| Поле | Подпись `Название новой группы`; поле ввода с плейсхолдером `Напр: Срочный ремонт БЦ`. |
| Футер | Кнопка `Отмена`; кнопка `Переместить`. |

Модальное окно открывается после выбора действия `Переместить в группу` в плавающей панели.

## 6. Внешние условия и получатели результата

| Внешняя область | Роль для массовых действий | Видимый результат |
|---|---|---|
| Карточный и табличный список задач | Содержит точки запуска выбора и выбранное состояние задач. | После выбора минимум одной задачи плавающая панель видна. |
| Заголовок таблицы | Содержит точку запуска выбора всех видимых строк текущей страницы. | После выбора всех видимых строк счетчик панели показывает количество выбранных задач. |
| Фильтрация и поиск | Задают видимый набор задач. | Массовые действия работают с задачами из текущего видимого набора и ранее выбранными задачами, если они сохранены в выборе. |
| Пагинация | Задает текущую страницу таблицы. | Чекбокс заголовка выбирает только видимые строки текущей страницы. |
| Настройки колонок | Меняют динамические колонки таблицы. | Чекбоксы выбора остаются доступными независимо от состава динамических колонок. |
| Очереди | Задают активный этап согласования для списка задач. | Кнопки `Назначить`, `Взять в работу` и `Завершить` видны по этапам выбранных задач. |
| Левая навигация | Получает результат создания или удаления пользовательской группы. | Новая группа появляется; пустая активная группа исчезает. |
| Toast-уведомление | Получает текст результата действия. | Пользователь видит уведомление о создании группы или удалении задач из группы. |

Массовые действия не создают задачи, не меняют атрибуты задач, не меняют состав бизнес-доменов, не меняют поисковую строку и не меняют параметры фильтрации.
