# Пагинация: данные, состояния и правила

## 1. Владельцы состояния

| Область | Состояние или значение | Правило видимого результата |
|---|---|---|
| Пагинация | Текущая страница, количество страниц, видимые номера страниц | Пагинация показывает номера страниц для итогового набора задач. |
| Список задач | Итоговый набор задач после области, очереди, поиска, фильтров и сортировки | Пагинация получает количество задач итогового набора и возвращает номер текущей страницы. |
| Массовые действия | Набор выбранных задач | Пагинация не владеет выбранными задачами; выбранные задачи могут находиться на разных страницах. |

## 2. Интерфейсные состояния

| Состояние | Значение | Видимый результат |
|---|---|---|
| Итоговый набор задач | Задачи после области, этапа, поиска, фильтров и сортировки | Количество страниц равно количеству групп по 20 задач. |
| Размер страницы | 20 задач | Одна страница содержит не больше 20 задач. |
| Текущая страница | Число от 1 до последней страницы | Список показывает задачи этой страницы. |
| Количество страниц | Итоговый набор, разделенный на страницы по 20 задач | Пагинация показывает доступные номера страниц. |
| Активный вид списка | Карточный или табличный | Текущая страница одинакова для карточного и табличного вида. |

## 3. Правило разбиения задач на страницы

| Текущая страница | Какие задачи видны |
|---|---|
| 1 | Первые 20 задач итогового набора. |
| 2 | Задачи с 21-й по 40-ю в итоговом наборе. |
| 3 и далее | Следующие группы по 20 задач. |
| Последняя страница | Оставшиеся задачи итогового набора; количество может быть меньше 20. |

Порядок задач внутри страниц совпадает с порядком итогового набора после сортировки.

## 4. Видимость пагинации

| Условие | Состояние текущей страницы | Видимый результат |
|---|---|---|
| Итоговый набор пуст | 1 | Пагинация скрыта. |
| Итоговый набор содержит 1-20 задач | 1 | Пагинация скрыта. |
| Итоговый набор содержит 21 и больше задач | От 1 до последней страницы | Пагинация видима. |

## 5. Защита текущей страницы

| Условие | Состояние текущей страницы |
|---|---|
| Итоговый набор пуст | Текущая страница получает значение 1. |
| Текущая страница больше последней доступной страницы | Текущая страница получает значение последней доступной страницы. |
| Текущая страница находится между 1 и последней страницей | Текущая страница остается без изменения. |

## 6. Правило видимых номеров

| Количество страниц и текущая позиция | Видимые элементы пагинации |
|---|---|
| Всего 7 страниц или меньше | Все номера страниц без `...`. |
| Всего больше 7 страниц; текущая страница 1, 2 или 3 | `1`, `2`, `3`, `4`, `...`, последняя страница. |
| Всего больше 7 страниц; текущая страница в середине диапазона | `1`, `...`, предыдущая страница, текущая страница, следующая страница, `...`, последняя страница. |
| Всего больше 7 страниц; текущая страница входит в последние 3 страницы | `1`, `...`, четвертая с конца страница, третья с конца страница, предпоследняя страница, последняя страница. |

## 7. Правила сброса и сохранения страницы

| Событие | Состояние текущей страницы |
|---|---|
| Первичное открытие прототипа | 1 |
| Клик по номеру страницы | Номер нажатой страницы. |
| Клик по активной стрелке назад | Текущая страница минус 1. |
| Клик по активной стрелке вперед | Текущая страница плюс 1. |
| Выбор другой области задач | 1 |
| Выбор другого этапа | 1 |
| Изменение поисковой строки | 1 |
| Применение фильтров | 1 |
| Быстрый сброс фильтров | 1 |
| Применение настроек колонок | 1 |
| Переключение карточного и табличного вида | Текущее значение остается без изменения. |
| Выбор или снятие выбора задач | Текущее значение остается без изменения. |

## 8. Контракты взаимодействия

| Связанный компонент | Роль связанного компонента | Условие | Результат в пагинации | Результат в связанном компоненте |
|---|---|---|---|---|
| Список задач | Источник итогового набора задач | Изменилась область, очередь, поиск, фильтры, сортировка или настройки колонок | Количество страниц пересчитывается по итоговому набору; текущая страница получает допустимое значение | Список показывает задачи текущей страницы. |
| Карточный список | Получатель текущей страницы | Активен карточный вид | Текущая страница задает набор задач страницы | Видны карточки текущей страницы. |
| Табличный список | Получатель текущей страницы | Активен табличный вид | Текущая страница задает набор задач страницы | Видны строки текущей страницы. |
| Чекбокс выбора всех строк таблицы | Получатель текущей страницы | Текущая страница изменилась | Пагинация передает новый набор видимых строк | Чекбокс получает пустое, выбранное или частичное состояние по строкам новой текущей страницы. |
| Массовые действия | Владелец набора выбранных задач | Есть выбранные задачи на других страницах | Текущая страница может измениться без очистки набора выбранных задач | Счетчик выбранных задач включает выбранные задачи с других страниц. |

## 9. Ограничения ответственности

| Область | Правило |
|---|---|
| Итоговый набор задач | Пагинация не формирует итоговый набор задач; она получает его от списка задач. |
| Поиск и фильтры | Пагинация не изменяет поисковую строку и параметры фильтрации. |
| Левая навигация | Пагинация не изменяет выбранную область задач. |
| Очереди | Пагинация не изменяет активный этап согласования. |
| Массовые действия | Пагинация не очищает набор выбранных задач. |
