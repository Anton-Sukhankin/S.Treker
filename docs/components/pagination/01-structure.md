# Пагинация: структура интерфейса

## 1. Расположение

| Область | Положение | Состав |
|---|---|---|
| Список задач | Над пагинацией | Карточки задач или таблица задач текущей страницы. |
| Пагинация | Под списком задач в правой рабочей области | Стрелка назад, номера страниц, возможные разрывы `...`, стрелка вперед. |
| Нижняя граница рабочей области | Под пагинацией | Не содержит элементов пагинации. |

Пагинация видима в карточном и табличном режимах списка задач. Режим отображения списка не меняет состав пагинации.

## 2. Каркас компонента

| Часть компонента | Порядок | Условие показа | Видимое состояние |
|---|---:|---|---|
| Стрелка назад | 1 | Пагинация видима | Кнопка со стрелкой влево. |
| Номера страниц | 2 | Пагинация видима | Последовательность номеров доступных страниц. |
| Разрыв `...` | Между номерами страниц | Количество страниц больше видимого диапазона | Текст `...` без действия. |
| Стрелка вперед | Последняя | Пагинация видима | Кнопка со стрелкой вправо. |

## 3. Условия видимости

| Условие итогового набора задач | Состояние пагинации | Состояние списка задач |
|---|---|---|
| В наборе нет задач | Пагинация скрыта | Показано пустое состояние списка. |
| В наборе от 1 до 20 задач | Пагинация скрыта | Показаны все задачи набора. |
| В наборе больше 20 задач | Пагинация видима | Показаны задачи текущей страницы. |

Размер страницы: 20 задач.

## 4. Состояния элементов

| Элемент | Условие | Видимое состояние |
|---|---|---|
| Номер текущей страницы | Страница выбрана | Номер выделен как активный; клик по нему не меняет список. |
| Номер другой страницы | Страница доступна и не выбрана | Номер доступен для клика. |
| Стрелка назад | Текущая страница равна 1 | Кнопка неактивна. |
| Стрелка назад | Текущая страница больше 1 | Кнопка активна. |
| Стрелка вперед | Текущая страница равна последней странице | Кнопка неактивна. |
| Стрелка вперед | Текущая страница меньше последней страницы | Кнопка активна. |
| Разрыв `...` | Между видимыми диапазонами есть скрытые страницы | Элемент видим и не кликабелен. |

## 5. Внешние области

| Внешняя область | Связь с пагинацией |
|---|---|
| Список задач | Получает только задачи текущей страницы. |
| Левая навигация | При выборе другой области задач текущая страница пагинации получает значение 1. |
| Очереди | При выборе другой очереди текущая страница пагинации получает значение 1. |
| Поиск по задачам | При изменении поисковой строки текущая страница пагинации получает значение 1. |
| Панель фильтрации | После применения или быстрого сброса фильтров текущая страница пагинации получает значение 1. |
| Переключатель вида | При переключении карточного и табличного вида текущая страница сохраняется. |
| Настройки колонок | После применения настроек колонок текущая страница пагинации получает значение 1. |
| Массовые действия | Выбор задач не меняет текущую страницу. Действие, которое пересобирает список, возвращает пагинацию на страницу 1. |
