# Пагинация: поведение

## 1. Основной пользовательский путь

1. Пользователь открывает список задач.
2. Список применяет выбранную область задач, выбранный этап, поиск, фильтры и сортировку.
3. Если итоговый набор содержит больше 20 задач, под списком видна пагинация.
4. Пользователь нажимает номер другой страницы.
5. Нажатый номер становится текущей страницей.
6. Список показывает задачи выбранной страницы.
7. Пользователь нажимает кнопку навигации со стрелкой назад или кнопку навигации со стрелкой вперед.
8. Текущая страница уменьшается или увеличивается на 1.
9. Список показывает задачи новой текущей страницы.

## 2. Переход по номеру страницы

| Действие пользователя | Условие | Итог пагинации | Итог списка задач |
|---|---|---|---|
| Клик по номеру другой страницы | Номер видим и не активен | Нажатый номер становится активным. | Список показывает задачи выбранной страницы. |
| Клик по активному номеру страницы | Номер уже активен | Активная страница не меняется. | Состав видимых задач не меняется. |
| Клик по `...` | Разрыв видим между номерами | Текущая страница не меняется. | Состав видимых задач не меняется. |

## 3. Переход стрелками

| Действие пользователя | Условие | Итог пагинации | Итог списка задач |
|---|---|---|---|
| Клик по стрелке назад | Текущая страница больше 1 | Активной становится предыдущая страница. | Список показывает задачи предыдущей страницы. |
| Клик по стрелке назад | Текущая страница равна 1 | Стрелка неактивна; текущая страница остается 1. | Состав видимых задач не меняется. |
| Клик по стрелке вперед | Текущая страница меньше последней страницы | Активной становится следующая страница. | Список показывает задачи следующей страницы. |
| Клик по стрелке вперед | Текущая страница равна последней странице | Стрелка неактивна; текущая страница остается последней. | Состав видимых задач не меняется. |

## 4. Реакция на внешние изменения

| Внешнее действие | Итог пагинации | Итог списка задач |
|---|---|---|
| Пользователь выбирает другую область в левой навигации | Текущая страница становится 1. | Список показывает первую страницу новой области. |
| Пользователь выбирает другой этап над списком | Текущая страница становится 1. | Список показывает первую страницу выбранного этапа. |
| Пользователь вводит или очищает поисковую строку | Текущая страница становится 1. | Список показывает первую страницу результата поиска. |
| Пользователь нажимает `Применить` в панели фильтрации | Текущая страница становится 1. | Список показывает первую страницу результата фильтрации. |
| Пользователь выполняет быстрый сброс фильтров | Текущая страница становится 1. | Список показывает первую страницу без активных фильтров. |
| Пользователь применяет настройки колонок | Текущая страница становится 1. | Табличный список показывает первую страницу с примененным составом колонок. |
| Пользователь переключает карточный и табличный вид | Текущая страница не меняется. | Новый режим показывает задачи той же страницы. |

## 5. Пустой и короткий результат

| Ситуация | Итог пагинации | Итог списка задач |
|---|---|---|
| Итоговый набор пуст | Пагинация скрыта; текущая страница равна 1. | Показано пустое состояние. |
| Итоговый набор содержит от 1 до 20 задач | Пагинация скрыта; текущая страница равна 1. | Показаны все задачи итогового набора. |
| Итоговый набор содержит больше 20 задач | Пагинация видима. | Показаны только задачи текущей страницы. |

## 6. Связь с выбором задач

| Действие пользователя | Итог выбора задач | Итог пагинации |
|---|---|---|
| Пользователь меняет страницу | Ранее выбранные задачи остаются выбранными. | Активной становится выбранная пользователем страница. |
| Пользователь выбирает задачу чекбоксом | Набор выбранных задач получает эту задачу или теряет ее. | Текущая страница не меняется. |
| Пользователь выбирает все видимые строки таблицы | Выбранными становятся только строки текущей страницы. | Текущая страница не меняется. |
