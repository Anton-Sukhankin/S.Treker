# Левая навигация: данные, состояния и правила

## 1. Состояния выбранной области

| Состояние | Активный пункт в левой навигации | Заголовок правой рабочей области | Набор задач в списке |
|---|---|---|---|
| Все задачи | `Все задачи` | `Все задачи` | Задачи всех бизнес-доменов, доступных текущей роли. |
| Бизнес-домен | Один пункт бизнес-домена | Название выбранного бизнес-домена | Задачи выбранного бизнес-домена. |
| Значение группировки | Одно значение внутри группировки | `Область • Значение` | Задачи выбранной области, у которых атрибут группировки равен выбранному значению. |
| Пользовательская группа | Один пункт пользовательской группы | Название пользовательской группы | Задачи, сохраненные в выбранной группе. |
| Группа внутри закладки | Один пункт группы внутри закладки | Название группы | Задачи, сохраненные в выбранной группе внутри закладки. |

В каждый момент активна только одна область: `Все задачи`, один бизнес-домен, одно значение группировки или одна пользовательская группа.

## 2. Состав бизнес-доменов

| Условие | Видимый состав левой навигации |
|---|---|
| Текущая роль прототипа | Видны `Строительный договор (ECM)`, `Закупки ТМЦ (Трекер)`, `Поручения СД`. |
| Другая роль | Набор бизнес-доменов может отличаться от текущего прототипа. |
| Бизнес-домен недоступен роли | Пункт бизнес-домена не виден в левой навигации; задачи этого домена не входят в `Все задачи`. |

`Все задачи` всегда собирает задачи только из бизнес-доменов, доступных текущей роли.

## 3. Счетчики задач

| Счетчик | Условие расчета | Видимое значение |
|---|---|---|
| `Все задачи` | Считаются задачи всех доступных бизнес-доменов, прошедшие примененные фильтры. | Бейдж показывает иконку задачи и число задач. |
| Бизнес-домен | Считаются задачи этого бизнес-домена, прошедшие примененные фильтры. | Бейдж показывает иконку задачи и число задач. |
| Пользовательская группа | Считаются задачи группы, прошедшие примененные фильтры. | Бейдж показывает иконку задачи и число задач. |
| Группа внутри закладки | Считаются задачи группы внутри закладки, прошедшие примененные фильтры. | Бейдж показывает иконку задачи и число задач. |
| Значение группировки | Считаются задачи значения группировки, прошедшие примененные фильтры. | Бейдж показывает иконку задачи и число задач. |

Очереди `Обработаны мной`, `Наблюдаю` и `Инициированы мной` не участвуют в счетчиках левой навигации. Активный этап над списком задач ограничивает видимый список задач справа, но счетчики левой навигации считаются по правилам левой навигации.

## 4. Правила группировок

| Правило | Видимый результат |
|---|---|
| Группировка создана для пункта `Все задачи` или бизнес-домена | Под этим пунктом виден блок вложенных группировок. |
| У одного пункта есть несколько группировок по разным атрибутам | Под пунктом видны несколько заголовков `По ...`. |
| Группировка создана повторно по тому же атрибуту | В левой навигации остается одна группировка по этому атрибуту с актуальным набором значений. |
| Значение группировки содержит ноль задач после примененных фильтров | Значение не видно в списке значений. |
| Значение группировки содержит пустое значение атрибута | Значение видно как `Без значения` и находится после остальных значений. |
| Все значения группировки стали невидимыми | Группировка не показывает значения. |

## 5. Индикаторы внутри значений группировки

| Условие в значении группировки | Видимый индикатор |
|---|---|
| Есть минимум одна просроченная задача | Красный бейдж с числом просроченных задач. |
| Есть минимум одна новая задача | Зеленый бейдж с числом новых задач. |
| Сумма задач больше нуля | Синий бейдж с суммой. |
| Есть минимум одна задача | Бейдж с иконкой задачи и числом задач. |
| Ни одной задачи после примененных фильтров | Значение группировки не видно. |

Если сумма значения группировки равна одному миллиону рублей или больше, сумма получает формат в миллионах рублей. Если сумма меньше одного миллиона рублей, сумма получает числовой формат без дробной части.

## 6. Пользовательские группы и закладки

| Состояние | Правило |
|---|---|
| Пользовательских групп нет | Блок пользовательских групп не виден. |
| Есть минимум одна пользовательская группа | Блок пользовательских групп виден. |
| Пользовательская группа содержит задачи, не проходящие примененные фильтры | Эти задачи не участвуют в счетчике группы и не видны в списке задач. |
| Пользовательская группа удалена | Пункт группы исчезает из левой навигации. |
| Активная пользовательская группа удалена | Активная область получает значение `Все задачи`. |
| Закладок нет | Список закладок пуст. |
| Закладка удалена | Закладка и группы внутри нее исчезают из левой навигации. |
| Активная группа находилась внутри удаленной закладки | Активная область получает значение `Все задачи`. |

## 7. Правила изменения ширины

| Условие | Видимый результат |
|---|---|
| Курсор находится вне области изменения ширины | Курсор имеет обычное состояние. |
| Курсор находится на области изменения ширины | Курсор имеет состояние изменения горизонтального размера. |
| Пользователь перетаскивает область изменения ширины | Левая навигация получает новую ширину; правая рабочая область начинается от новой правой границы панели. |
| Левая навигация достигла минимальной ширины | Дальнейшее перетаскивание влево не уменьшает левую навигацию; правая рабочая область не заходит под нее. |
| Левая навигация достигла максимальной ширины | Дальнейшее перетаскивание вправо не увеличивает левую навигацию. |

## 8. Влияние выбранной области на соседние компоненты

| Соседний компонент | Условие выбранной области | Видимый результат |
|---|---|---|
| Заголовок правой рабочей области | Выбран `Все задачи`, бизнес-домен или пользовательская группа | Заголовок получает название выбранной области. |
| Заголовок правой рабочей области | Выбрано значение группировки | Заголовок получает формат `Область • Значение`. |
| Панель инструментов | Выбраны `Все задачи` или бизнес-домен | Панель показывает обычный набор инструментов и этапы над списком задач. |
| Панель инструментов | Выбрана пользовательская группа | Этапы над списком скрыты; часть действий переносится в область заголовка правой рабочей области. |
| Список задач | Выбрана любая область левой навигации | Список задач показывает задачи выбранной области, которые проходят активный этап, поиск, примененные фильтры и текущую страницу. |
| Панель фильтрации | Выбран бизнес-домен | Панель фильтрации показывает параметры, применимые к выбранному бизнес-домену. |
| Панель фильтрации | Выбраны `Все задачи` или область без одного бизнес-домена | Панель фильтрации показывает параметры для доступного набора бизнес-доменов. |
| Настройки колонок | Выбран бизнес-домен | Настройки колонок показывают колонки выбранного бизнес-домена. |
| Настройки колонок | Выбраны `Все задачи` | Настройки колонок показывают объединенный набор колонок доступных бизнес-доменов. |
