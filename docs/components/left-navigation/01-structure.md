# Левая навигация: структура интерфейса

## 1. Расположение

Левая навигация находится слева от рабочей области списка задач. Компонент занимает всю высоту рабочей зоны под верхним глобальным меню.

| Область | Положение | Состав |
|---|---|---|
| Левая навигация | Слева от списка задач | Основной список областей задач; блок пользовательских групп; блок закладок; область изменения ширины. |
| Правая рабочая область | Справа от левой навигации | Заголовок текущей области; панель инструментов; этапы; список задач. |

Правая рабочая область не входит в левую навигацию. Левая навигация задает область задач, которую правая рабочая область показывает в заголовке, панели инструментов и списке задач.

## 2. Каркас компонента

| Часть компонента | Позиция | Состав |
|---|---|---|
| Основной список областей задач | Верхняя часть панели | Пункт `Все задачи`; пункты бизнес-доменов; счетчики задач; точечные индикаторы у отдельных пунктов. |
| Вложенные группировки | Под пунктом `Все задачи` или под пунктом бизнес-домена | Заголовки группировки по атрибуту; значения группировки; счетчики и индикаторы по значениям. |
| Пользовательские группы | Под основным списком областей задач | Аккордеон с пользовательскими группами, созданными из выбранных задач. |
| Закладки | Под пользовательскими группами | Аккордеоны закладок; внутри каждой закладки находятся сохраненные группы задач. |
| Область изменения ширины | Между левой навигацией и правой рабочей областью | Узкая вертикальная область для перетаскивания ширины панели. |

## 3. Основные пункты навигации

| Пункт | Позиция | Видимый текст в текущем прототипе | Результат выбора |
|---|---|---|---|
| Все задачи | Первый пункт | `Все задачи` | Правая рабочая область показывает задачи всех бизнес-доменов, доступных текущей роли. |
| Бизнес-домен 1 | Под `Все задачи` | `Строительный договор (ECM)` | Правая рабочая область показывает задачи строительного домена. |
| Бизнес-домен 2 | Под строительным доменом | `Закупки ТМЦ (Трекер)` | Правая рабочая область показывает задачи домена закупок ТМЦ. |
| Бизнес-домен 3 | Под доменом закупок | `Поручения СД` | Правая рабочая область показывает задачи домена поручений СД. |

Список бизнес-доменов в текущем прототипе соответствует одной роли. Для другой роли состав пунктов бизнес-доменов может быть меньше, больше или другим.

## 4. Состав пункта навигации

| Элемент пункта | Условие показа | Видимое состояние |
|---|---|---|
| Иконка | Всегда у пункта `Все задачи`, бизнес-домена, пользовательской группы и группы внутри закладки | Слева от названия пункта. |
| Название | Всегда | Текст выбранной области задач. |
| Счетчик задач | Всегда у основных пунктов и пользовательских групп | Бейдж справа от названия; внутри бейджа видны иконка задачи и число задач. |
| Точечный индикатор | Только у пунктов, где он предусмотрен текущим прототипом | Маленькая точка справа от пункта. |
| Активное состояние | Пункт выбран пользователем | Пункт получает активное визуальное состояние; остальные пункты и значения группировки теряют активное состояние. |
| Кнопка раскрытия группировок | У пункта есть созданные группировки | Справа от названия виден бейдж с количеством значений группировки. |
| Кнопка сброса группировок | У пункта есть созданные группировки | Справа у пункта видна иконка сброса всех группировок этого пункта. |

## 5. Вложенные группировки

После действия группировки в табличном списке задач под пунктом `Все задачи` или под бизнес-доменом виден блок вложенных группировок.

| Часть группировки | Позиция | Состав |
|---|---|---|
| Обертка пункта | Вокруг пункта `Все задачи` или бизнес-домена | Основной пункт; область вложенных группировок. |
| Заголовок группировки | Под основным пунктом | Текст `По ...`; иконка раскрытия или сворачивания. |
| Список значений | Под заголовком группировки | Значения выбранного атрибута. |
| Значение группировки | Внутри списка значений | Название значения; набор бейджей; кнопка удаления значения. |
| Пустое значение | В конце списка значений | Значение `Без значения`, если такие задачи есть. |

| Бейдж внутри значения группировки | Условие показа | Видимое значение |
|---|---|---|
| Просрочено | В значении есть минимум одна просроченная задача | Красный бейдж с числом просроченных задач. |
| Новые задачи | В значении есть минимум одна новая задача | Зеленый бейдж с числом новых задач. |
| Сумма | Сумма задач в значении больше нуля | Синий бейдж с суммой. |
| Количество задач | Значение содержит минимум одну задачу | Бейдж с иконкой задачи и числом задач. |

## 6. Пользовательские группы

После создания группы из выбранных задач в левой навигации виден блок пользовательских групп.

| Состояние | Видимое состояние блока |
|---|---|
| Пользовательских групп нет | Блок пользовательских групп не виден. |
| Есть минимум одна пользовательская группа | Видим аккордеон с заголовком `Новая группа`; внутри видны созданные группы. |

| Элемент пользовательской группы | Состав |
|---|---|
| Пункт группы | Иконка папки; название группы; бейдж с количеством задач; кнопка удаления группы. |
| Активная группа | Пункт группы получает активное состояние; правая рабочая область показывает задачи этой группы. |

## 7. Закладки

Закладки содержат сохраненные наборы пользовательских групп.

| Часть закладки | Состав |
|---|---|
| Заголовок закладки | Иконка раскрытия; название закладки; кнопка удаления закладки. |
| Тело закладки | Список групп, сохраненных внутри закладки. |
| Группа внутри закладки | Иконка папки; название группы; бейдж с количеством задач. |

При отсутствии закладок список закладок пуст.

## 8. Изменение ширины

| Элемент | Позиция | Видимое состояние |
|---|---|---|
| Область изменения ширины | Между левой навигацией и правой рабочей областью | При наведении курсор получает состояние изменения горизонтального размера. |
| Левая навигация при перетаскивании | Слева от рабочей области | Ширина панели получает значение в пределах минимальной и максимальной ширины. |

Область изменения ширины не должна выглядеть как постоянный правый бордер левой навигации.

## 9. Внешние области

| Внешняя область | Связь с левой навигацией |
|---|---|
| Заголовок правой рабочей области | После выбора пункта левой навигации заголовок получает название выбранной области. |
| Панель инструментов | Для обычных областей задач панель показывает поиск, действия и этапы; для пользовательской группы часть действий переносится в заголовок правой рабочей области. |
| Этапы над списком задач | После выбора пункта левой навигации активный этап ограничивает задачи внутри выбранной области. |
| Список задач | После выбора пункта левой навигации список показывает задачи выбранной области. |
| Панель фильтрации | Выбранная область задач задает состав доменных блоков и значения параметров внутри фильтрации. |
| Настройки колонок | Выбранная область задач задает набор колонок для табличного вида. |
