# Настройки колонок: поведение

## 1. Основной пользовательский путь

1. Пользователь находится в табличном виде списка задач.
2. Пользователь нажимает кнопку `Настройка колонок` в панели инструментов.
3. Поверх интерфейса появляется затемнение фона.
4. Справа открывается панель `Настройки`.
5. Панель показывает черновик колонок текущего контекста.
6. Пользователь меняет видимость или порядок колонок.
7. Основная таблица задач сохраняет прежний состав и порядок колонок до нажатия `Применить`.
8. Пользователь нажимает `Применить`.
9. Панель `Настройки` закрывается.
10. Затемнение фона скрывается.
11. Таблица задач показывает примененный состав и порядок колонок.

## 2. Открытие панели

| Действие пользователя | Состояние панели | Состояние затемнения | Состояние таблицы задач |
|---|---|---|---|
| Нажимает `Настройка колонок` в табличном виде | Справа открыта панель `Настройки` | Затемнение фона видимо | Таблица сохраняет текущие колонки |
| Панель открыта | Создан черновик из примененных настроек текущего контекста | Затемнение фона блокирует работу с основной областью | Таблица сохраняет текущие колонки |
| Панель открыта впервые для контекста без пользовательских настроек | Черновик собран из базового набора колонок текущего контекста | Затемнение фона видимо | Таблица сохраняет базовый набор колонок |
| Панель открыта | Библиотека атрибутов скрыта | Футер показан в обычной ширине | Таблица сохраняет текущие колонки |

## 3. Закрытие без применения

| Действие пользователя | Состояние панели | Состояние таблицы задач |
|---|---|---|
| Нажимает крестик в хедере панели | Панель закрыта, затемнение фона скрыто | Таблица сохраняет прежние примененные колонки |
| Нажимает по затемнению фона | Панель закрыта, затемнение фона скрыто | Таблица сохраняет прежние примененные колонки |
| Закрывает панель после изменения черновика | Черновик сбрасывается | Таблица сохраняет прежние примененные колонки |
| Закрывает панель при раскрытой библиотеке | Библиотека скрыта вместе с панелью | Таблица сохраняет прежние примененные колонки |

Закрытие панели без `Применить` не записывает черновик в примененные настройки.

## 4. Скрытие и показ динамической колонки

| Действие пользователя | Состояние строки в панели | Состояние таблицы до `Применить` | Состояние таблицы после `Применить` |
|---|---|---|---|
| Снимает чекбокс у колонки | Колонка получает состояние `скрыта` в черновике | Колонка остается видимой, если была видима до открытия панели | Колонка скрыта |
| Ставит чекбокс у колонки | Колонка получает состояние `видима` в черновике | Колонка остается скрытой, если была скрыта до открытия панели | Колонка видима |

Закрепленные колонки `чекбокс выбора`, `номер` и `наименование` не показаны в списке настроек и не отключаются через панель.

## 5. Изменение порядка колонок

| Действие пользователя | Состояние списка в панели | Состояние таблицы до `Применить` | Состояние таблицы после `Применить` |
|---|---|---|---|
| Начинает перетаскивать строку колонки | Строка получает состояние перетаскивания | Таблица сохраняет прежний порядок | Таблица сохраняет прежний порядок |
| Перетаскивает строку выше или ниже | Позиция строки в черновике меняется | Таблица сохраняет прежний порядок | Таблица сохраняет прежний порядок |
| Завершает перетаскивание | Новый порядок записан в черновик | Таблица сохраняет прежний порядок | Таблица сохраняет прежний порядок |
| Нажимает `Применить` | Панель закрывается | Таблица перестраивает динамические колонки | Таблица показывает новый порядок |

Порядок закрепленных колонок не меняется: `чекбокс выбора`, `номер`, `наименование` всегда остаются первыми слева.

## 6. Кнопка `По умолчанию`

| Действие пользователя | Состояние черновика | Состояние таблицы до `Применить` | Состояние таблицы после `Применить` |
|---|---|---|---|
| Нажимает `По умолчанию` | Черновик возвращается к базовому набору колонок текущего контекста | Таблица сохраняет прежние колонки | Таблица показывает базовый набор колонок |
| Нажимает `По умолчанию` при выбранном пресете | В черновике выбран `Базовое отображение` | Таблица сохраняет прежние колонки | Таблица показывает базовый набор колонок |

Кнопка `По умолчанию` не удаляет сохраненные пресеты.

## 7. Кнопка `Применить`

| Шаг применения | Конкретное изменение |
|---|---|
| Пользователь нажимает `Применить` | Видимый список строк панели считывается как черновик |
| Черновик проверяется | Повторы, недоступные колонки и закрепленные колонки исключаются из динамической части |
| Настройки записываются для текущего контекста | Черновик становится примененным состоянием |
| Панель закрывается | Затемнение фона скрывается, библиотека атрибутов закрывается |
| Таблица задач перестраивается | Таблица показывает примененные динамические колонки |
| Левая навигация и счетчики очередей получают актуальные значения | Числа соответствуют текущему контексту после применения |

`Применить` — единственная точка, после которой изменения из панели попадают в основную таблицу.

## 8. Библиотека атрибутов внутри панели

| Действие пользователя | Состояние панели | Состояние таблицы задач |
|---|---|---|
| Нажимает кнопку библиотеки в футере | Слева от `Общие атрибуты` открывается область `Библиотека атрибутов` | Таблица не меняется |
| Нажимает кнопку библиотеки повторно | Область `Библиотека атрибутов` закрывается | Таблица не меняется |
| Добавляет атрибут из библиотеки | Атрибут добавляется в конец черновика колонок | Таблица не меняется |
| Возвращает добавленный атрибут в библиотеку | Атрибут удаляется из черновика колонок | Таблица не меняется |
| Нажимает `Применить` после добавления или возврата | Черновик становится примененным состоянием | Таблица показывает примененный набор колонок |

## 9. Сброс библиотечных добавлений

| Условие | Состояние кнопки `Сбросить` |
|---|---|
| Библиотека закрыта | Кнопка не показана |
| Библиотека открыта, библиотечных добавлений нет | Кнопка показана в неактивном состоянии |
| Библиотека открыта, есть минимум одно библиотечное добавление | Кнопка активна |

| Действие пользователя | Состояние черновика | Состояние таблицы задач |
|---|---|---|
| Нажимает активную кнопку `Сбросить` | Все добавленные из библиотеки атрибуты удаляются из черновика | Таблица не меняется |
| Нажимает `Применить` после сброса | Черновик без библиотечных добавлений становится примененным состоянием | Таблица не показывает сброшенные колонки |

## 10. Пресеты

Пресеты колонок являются вложенным компонентом внутри панели `Настройки`.

| Действие пользователя | Состояние черновика | Состояние таблицы задач |
|---|---|---|
| Выбирает `Базовое отображение` или сохраненный пресет | Черновик получает состав, порядок и видимость выбранного варианта | Таблица не меняется |
| Сохраняет новый пресет | Список пресетов текущего контекста получает новый пункт | Таблица не меняется |
| Удаляет сохраненный пресет | Список пресетов текущего контекста теряет этот пункт | Таблица не меняется |
| Нажимает `Применить` после выбора, сохранения или удаления пресета | Черновик становится примененным состоянием | Таблица показывает колонки выбранного черновика |

Полный сценарий выбора, сохранения, удаления и модального окна относится к компоненту `Пресеты колонок`.

## 11. Пустые и граничные состояния

| Ситуация | Состояние панели |
|---|---|
| Нет пользовательского состояния колонок | Панель показывает базовый набор колонок |
| В библиотеке нет атрибутов по поиску | В списке библиотеки показано `Атрибуты не найдены` |
| В черновике остались только закрепленные колонки | Закрепленные колонки не показаны в списке, но остаются в таблице |
| Сохраненный пресет содержит недоступные колонки | Недоступные колонки исключаются при чтении пресета |
