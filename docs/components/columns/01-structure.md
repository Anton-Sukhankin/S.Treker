# Настройки колонок: структура интерфейса

Файл описывает, где находится интерфейс настройки колонок и из каких видимых частей он состоит. Поведение действий описано в `02-behavior.md`; данные, состояния и правила применения - в `03-data-states-rules.md`.

## 1. Границы компонента

Настройки колонок управляют тем, какие атрибуты задачи видны в табличном списке и в каком порядке они стоят.

| Входит в компонент | Не входит в компонент, но влияет на него |
|---|---|
| Кнопка открытия настроек колонок | Карточный вид списка задач |
| Overlay панели настроек | Список задач и активная таблица |
| Правая панель `Настройки` | Левая навигация и текущий бизнес-домен |
| Список настраиваемых колонок | Очереди задач |
| Чекбоксы видимости колонок | Данные задач и атрибутов |
| Drag-and-drop порядка колонок | Фильтрация и поиск |
| Кнопки `По умолчанию` и `Применить` | |
| Вход в библиотеку атрибутов | |
| Краткое управление пресетами внутри панели | |

Компонент меняет только отображение таблицы. Данные задач и значения атрибутов не удаляются и не редактируются.

## 2. Точки входа

| Точка входа | Где находится | Когда доступна |
|---|---|---|
| Основная кнопка настройки колонок | Верхняя панель правой рабочей области со списком задач | Видна в табличном режиме списка задач |
| Кнопка настройки колонок в системном overlay | Локальная панель инструментов системного overlay | Видна внутри overlay |

Обе кнопки открывают одну и ту же правую панель `Настройки`.

## 3. Overlay

При открытии панели появляется overlay `columns-overlay`.

| Свойство | Описание |
|---|---|
| Расположение | За правой панелью настроек |
| Содержимое | Не содержит собственных настроек |
| Назначение | Создает модальный режим панели и закрывает ее при клике |

## 4. Правая панель `Настройки`

Панель открывается справа поверх интерфейса.

| Область панели | Что содержит |
|---|---|
| Хедер | Заголовок `Настройки`, кнопка закрытия с крестиком |
| Тело | Основную область `Общие атрибуты`; при раскрытии - левую область `Библиотека атрибутов` |
| Футер | Кнопку открытия библиотеки, кнопку сброса библиотечных добавлений, `По умолчанию`, `Применить` |

## 5. Основная область `Общие атрибуты`

Основная область видна сразу после открытия панели.

| Элемент | Назначение |
|---|---|
| Заголовок `Общие атрибуты` | Обозначает список колонок, которыми управляет пользователь |
| Управление пресетом | Показывает выбранный вариант отображения и позволяет выбрать другой |
| Кнопка `Сохранить` | Сохраняет текущий черновик колонок как пресет; по умолчанию disabled |
| Список колонок | Показывает настраиваемые колонки текущего контекста |

Фиксированные колонки `cb`, `id`, `title` не показываются в списке настроек. Они всегда остаются в таблице и не выключаются через эту панель.

## 6. Строка колонки

Каждая строка в списке колонок описывает одну настраиваемую колонку.

| Элемент строки | Когда виден | Назначение |
|---|---|---|
| Чекбокс | Всегда | Управляет видимостью колонки в черновике |
| Название колонки | Всегда | Показывает человекочитаемое имя атрибута |
| Действие возврата в библиотеку | Только у колонок, добавленных из библиотеки атрибутов | Убирает атрибут из черновика колонок и возвращает его в библиотеку |
| Drag handle | Всегда в строке | Позволяет изменить порядок колонок через перетаскивание |

Состояние строки внутри панели меняется сразу, но основная таблица не меняется до `Применить`.

## 7. Управление пресетами внутри панели

В верхней части основной области есть блок пресетов.

| Элемент | Что показывает |
|---|---|
| Текущий пресет | `Базовое отображение` или название выбранного пресета |
| Стрелка раскрытия | Открывает список пресетов |
| Выпадающий список | Содержит `Базовое отображение` и сохраненные пресеты текущего контекста |
| Действие удаления | Показывается у сохраненного пресета внутри списка |
| Кнопка `Сохранить` | Активируется, когда черновик отличается от выбранного пресета или базы |

Пресет относится к отображению таблицы, а не к данным задачи.

## 8. Библиотека атрибутов внутри панели

Библиотека атрибутов не видна сразу после открытия панели.

| Состояние панели | Что видно |
|---|---|
| Библиотека закрыта | Видна только основная область `Общие атрибуты` |
| Библиотека открыта | Слева появляется область `Библиотека атрибутов`, основная область остается справа |

В области библиотеки есть:

| Элемент | Назначение |
|---|---|
| Заголовок `Библиотека атрибутов` | Обозначает список скрытых доступных атрибутов |
| Поле `Поиск атрибута...` | Фильтрует список атрибутов внутри библиотеки |
| Список атрибутов | Показывает атрибуты, которых сейчас нет в черновике колонок |
| Кнопка добавления у атрибута | Добавляет атрибут в черновик колонок |

В этой спецификации библиотека описана только как часть интерфейса настройки колонок. Главная роль библиотеки здесь - добавить скрытый атрибут в черновик колонок или вернуть его обратно.

## 9. Футер панели

Футер закреплен внизу панели.

| Элемент | Когда виден | Назначение |
|---|---|---|
| Кнопка библиотеки | Всегда | Открывает или закрывает библиотеку атрибутов |
| Кнопка `Сбросить` с иконкой сброса | Видна только когда библиотека раскрыта | Убирает из черновика все атрибуты, добавленные из библиотеки |
| `По умолчанию` | Всегда | Возвращает черновик к базовому отображению текущего контекста |
| `Применить` | Всегда | Применяет черновик к основной таблице и закрывает панель |

Кнопка `Сбросить` находится в disabled-состоянии, если в черновике нет атрибутов, добавленных из библиотеки.

## 10. Связь с таблицей задач

Настройки колонок влияют только на табличный вид списка задач.

| Область таблицы | Как влияет настройка |
|---|---|
| Заголовок таблицы | После применения показывает новый состав и порядок колонок |
| Строки таблицы | После применения показывают значения в новом порядке |
| Закрепленные колонки | `cb`, `id`, `title` остаются первыми и не скрываются |
| Карточный вид | Не меняется от настроек колонок |
