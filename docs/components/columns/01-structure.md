# Настройки колонок: структура интерфейса

## 1. Граница компонента

Настройки колонок управляют составом и порядком динамических колонок табличного списка задач.

| Входит в компонент | Не входит в компонент |
|---|---|
| Кнопка открытия настроек колонок в панели инструментов | Карточный вид списка задач |
| Затемнение фона при открытой панели | Левая навигация |
| Правая панель `Настройки` | Этапы согласования |
| Область `Общие атрибуты` | Поиск по задачам |
| Список настраиваемых колонок | Панель фильтрации |
| Чекбоксы видимости колонок | Данные задач и значения атрибутов |
| Перетаскивание строк колонок | |
| Пресеты колонок | |
| Библиотека атрибутов внутри панели | |
| Футер с действиями `По умолчанию` и `Применить` | |

Компонент меняет только отображение таблицы. Значения задач и атрибутов не редактируются.

## 2. Точка входа

| Точка входа | Где находится | Когда доступна |
|---|---|---|
| Кнопка `Настройка колонок` | Панель инструментов правой рабочей области | Только в табличном виде списка задач |

Нажатие на кнопку открывает правую панель `Настройки` и затемнение фона.

## 3. Затемнение фона

| Свойство | Значение |
|---|---|
| Положение | За правой панелью настроек поверх основного интерфейса |
| Содержимое | Не содержит собственных элементов управления |
| Назначение в интерфейсе | Отделяет открытую панель от основной рабочей области |
| Действие по клику | Закрывает панель без применения черновика |

## 4. Правая панель `Настройки`

Панель открывается справа поверх интерфейса.

| Область панели | Состав |
|---|---|
| Хедер | Заголовок `Настройки`, кнопка закрытия |
| Тело | Область `Общие атрибуты`; при раскрытии библиотеки — область `Библиотека атрибутов` слева |
| Футер | Кнопка библиотеки, кнопка `Сбросить` для библиотечных добавлений, `По умолчанию`, `Применить` |

## 5. Область `Общие атрибуты`

| Элемент | Что показывает |
|---|---|
| Заголовок `Общие атрибуты` | Основной список колонок, которыми управляет пользователь |
| Пресеты колонок | Вложенный компонент выбора, сохранения и удаления пресетов текущего контекста |
| Кнопка `Сохранить` | Часть компонента пресетов; по умолчанию неактивна |
| Список колонок | Настраиваемые динамические колонки текущего контекста |

Закрепленные колонки `чекбокс выбора`, `№` и `Наименование` не показаны в списке настроек. Они всегда остаются в таблице.

## 6. Строка колонки

Каждая строка описывает одну настраиваемую динамическую колонку.

| Элемент строки | Когда виден | Что делает |
|---|---|---|
| Чекбокс | Всегда | Меняет видимость колонки в черновике |
| Название колонки | Всегда | Показывает пользовательское название атрибута |
| Действие возврата в библиотеку | Только у колонок, добавленных из библиотеки | Удаляет атрибут из черновика и возвращает его в библиотеку |
| Ручка перетаскивания | Всегда | Меняет порядок колонок в черновике |

Изменение строки сразу меняет только черновик внутри панели. Основная таблица меняется только после `Применить`.

## 7. Пресеты

| Элемент | Что показывает |
|---|---|
| Текущий пресет | `Базовое отображение` или название выбранного сохраненного пресета |
| Стрелка раскрытия | Открывает список пресетов |
| Список пресетов | `Базовое отображение` и сохраненные пресеты текущего контекста |
| Действие удаления | Видно у сохраненного пресета внутри списка |
| Кнопка `Сохранить` | Активна, если черновик отличается от выбранного пресета или базового набора |

Пресеты колонок являются вложенным компонентом внутри панели `Настройки`. В рамках настроек колонок пресеты меняют черновик колонок; таблица задач получает выбранный пресет только после `Применить`.

## 8. Библиотека атрибутов внутри панели

Библиотека атрибутов скрыта при открытии панели.

| Состояние панели | Что видно |
|---|---|
| Библиотека закрыта | Видна только область `Общие атрибуты` |
| Библиотека открыта | Слева видна область `Библиотека атрибутов`, справа остается область `Общие атрибуты` |

| Элемент библиотеки | Что делает |
|---|---|
| Заголовок `Библиотека атрибутов` | Обозначает список скрытых доступных атрибутов |
| Поле `Поиск атрибута...` | Оставляет в списке атрибуты с совпадением по названию |
| Список атрибутов | Показывает атрибуты, которых сейчас нет в черновике колонок |
| Кнопка добавления у атрибута | Добавляет атрибут в черновик `Общие атрибуты` |

## 9. Футер панели

| Элемент | Когда виден | Что делает |
|---|---|---|
| Кнопка библиотеки | Всегда | Открывает или закрывает библиотеку атрибутов |
| `Сбросить` | Только при открытой библиотеке | Удаляет из черновика все атрибуты, добавленные из библиотеки |
| `По умолчанию` | Всегда | Возвращает черновик к базовому набору колонок текущего контекста |
| `Применить` | Всегда | Применяет черновик к таблице и закрывает панель |

Кнопка `Сбросить` неактивна, если в черновике нет атрибутов, добавленных из библиотеки.

## 10. Связь с таблицей задач

| Область таблицы | Как влияет настройка колонок |
|---|---|
| Заголовок таблицы | После `Применить` показывает новый состав и порядок динамических колонок |
| Строки таблицы | После `Применить` показывают значения в новом порядке |
| Закрепленные колонки | Чекбокс выбора, `№` и `Наименование` остаются первыми и не скрываются |
| Карточный вид | Не меняется |
