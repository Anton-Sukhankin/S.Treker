# Библиотека атрибутов: данные, состояния и правила

Этот файл описывает состояния дочерней зоны `Библиотека атрибутов`. Наличие атрибута в библиотеке и состояние кнопки действия зависят от черновика колонок родительского компонента `Настройки колонок`.

## 1. Владельцы состояния

| Область | Состояние или значение | Правило видимого результата |
|---|---|---|
| `Настройки колонок` | Черновик колонок и примененное состояние таблицы | Библиотека меняет только черновик; таблица меняется только после `Применить` в родительской панели. |
| `Библиотека атрибутов` | Раскрытие библиотеки, поисковая строка, список доступных атрибутов, состояние кнопок действия | Открытая библиотека показывает доступные атрибуты текущего контекста и состояние действия для каждого атрибута. |
| `Общие атрибуты` | Строки колонок, включая добавленные из библиотеки | Добавленный атрибут виден в `Общие атрибуты` как колонка черновика. |
| Табличный список задач | Видимые динамические колонки | Таблица получает библиотечные колонки только после `Применить`. |

## 2. Интерфейсные состояния библиотеки

| Состояние | Пустое или базовое значение | Активное значение | Видимый результат |
|---|---|---|---|
| Раскрытие библиотеки | Библиотека закрыта | Библиотека открыта | При открытом состоянии видна область библиотеки рядом с `Общие атрибуты`. |
| Поиск по библиотеке | Поле поиска пустое | Введен текст | Список атрибутов ограничен совпадениями по названию. |
| Доступность атрибута | Атрибут недоступен | Атрибут доступен | Доступный атрибут виден в списке библиотеки. |
| Наличие атрибута в черновике | Атрибута нет в черновике | Атрибут добавлен в черновик | Атрибут виден в библиотеке; кнопка действия показывает иконку плюса или иконку стрелки назад. |
| Библиотечные добавления | Добавлений нет | Добавлен минимум один атрибут из библиотеки | Кнопка `Сбросить` в футере библиотеки активна. |
| Примененное состояние колонок | Таблица использует прежние колонки | Нажато `Применить` в настройках колонок | Таблица показывает колонки из примененного состояния. |

## 3. Правило доступности атрибута

Атрибут виден в библиотеке, если одновременно выполнены все условия.

| Условие | Требование |
|---|---|
| Скрыт по умолчанию | Атрибут не входит в дефолтный набор колонок таблицы. |
| Относится к текущему контексту | Атрибут принадлежит выбранному бизнес-домену или одному из доменов режима `Все задачи`. |
| Проходит поиск | Название атрибута содержит введенный текст или поле поиска пустое. |

Если хотя бы одно условие не выполнено, атрибут не виден в списке библиотеки.

Добавление атрибута в черновик не скрывает атрибут из библиотеки. Добавление меняет только состояние кнопки действия в строке атрибута.

## 4. Правила выбранной области задач

| Выбранная область | Доступные атрибуты библиотеки |
|---|---|
| `Все задачи` | Скрытые атрибуты всех бизнес-доменов, доступных текущей роли. |
| Конкретный бизнес-домен | Скрытые атрибуты выбранного бизнес-домена. |
| Пользовательская группа | Скрытые атрибуты всех доступных бизнес-доменов. |
| Сгруппированное представление | Скрытые атрибуты всех доступных бизнес-доменов. |

Если один и тот же атрибут доступен в нескольких доменах, в библиотеке он показан один раз.

## 5. Правила черновика

| Состояние черновика | Видимый результат |
|---|---|
| Атрибут добавлен из библиотеки | В `Общие атрибуты` видна новая включенная колонка; в библиотеке этот атрибут виден с кнопкой действия с иконкой стрелки назад. |
| Атрибут возвращен через кнопку действия с иконкой стрелки назад в библиотеке или в `Общие атрибуты` | В `Общие атрибуты` строка атрибута не видна; в библиотеке этот атрибут виден с кнопкой действия с иконкой плюса. |
| Выполнен сброс библиотечных добавлений | Все атрибуты, добавленные из библиотеки, удалены из черновика; обычные колонки остаются в черновике; в библиотеке у сброшенных атрибутов видна кнопка действия с иконкой плюса. |
| Выбран сохраненный пресет с библиотечными атрибутами | В `Общие атрибуты` видны библиотечные атрибуты из пресета; в библиотеке эти атрибуты видны с кнопкой действия с иконкой стрелки назад. |
| Выбрано `Базовое отображение` | В `Общие атрибуты` нет библиотечных добавлений; в библиотеке доступные атрибуты видны с кнопкой действия с иконкой плюса. |
| Библиотечных добавлений нет | Кнопка `Сбросить` неактивна. |
| Есть минимум одно библиотечное добавление | Кнопка `Сбросить` активна. |

Сброс библиотечных добавлений не равен действию `По умолчанию`. `Сбросить` удаляет только добавления из библиотеки; `По умолчанию` возвращает весь черновик колонок к базовому отображению.

## 6. Контракты взаимодействия

| Связанный компонент | Роль связанного компонента | Условие | Результат в библиотеке атрибутов | Результат в связанном компоненте |
|---|---|---|---|---|
| `Настройки колонок` | Родительский владелец черновика | Пользователь добавил, вернул или сбросил атрибут библиотеки | Строка атрибута в библиотеке получает иконку плюса или стрелки назад | Черновик родительской панели получает новый состав колонок. |
| `Общие атрибуты` | Получатель добавленного атрибута | Атрибут добавлен из библиотеки | Строка атрибута в библиотеке остается видимой с кнопкой действия со стрелкой назад | В `Общие атрибуты` видна строка добавленного атрибута. |
| `Пресеты колонок` | Источник сохраненного состояния с библиотечными атрибутами | Пользователь выбрал сохраненный пресет с библиотечными атрибутами | Строки этих атрибутов показывают кнопку действия со стрелкой назад | `Общие атрибуты` показывает библиотечные атрибуты из пресета. |
| Табличный список задач | Получатель примененного состояния родительской панели | Пользователь нажал `Применить` в настройках колонок | Библиотека сохраняет состояние кнопок по примененному черновику | Таблица показывает добавленные библиотечные колонки. |

## 7. Правило применения к таблице

| Состояние до `Применить` | Состояние после `Применить` |
|---|---|
| Атрибут добавлен из библиотеки и виден только в `Общие атрибуты` | В таблице видна новая колонка. |
| Атрибут возвращен в библиотеку и не виден в `Общие атрибуты` | Колонка не видна в таблице. |
| Библиотечные добавления сброшены в черновике | Таблица показывает колонки без библиотечных добавлений. |
| Панель закрыта без `Применить` | Таблица сохраняет прежний набор колонок. |

## 8. Пустые значения в таблице

| Условие | Видимый результат |
|---|---|
| У задачи есть значение добавленного атрибута | В ячейке показано значение атрибута. |
| У задачи нет значения добавленного атрибута | В ячейке показано пустое значение. |
| В режиме `Все задачи` атрибут относится не ко всем доменам | У задач без такого атрибута показано пустое значение. |

Пустое значение после добавления атрибута является допустимым состоянием объединенной таблицы.

## 9. Ограничения ответственности

| Область | Правило |
|---|---|
| Значения задач | Библиотека атрибутов не изменяет значения задач. |
| Примененное состояние таблицы | Библиотека атрибутов не изменяет таблицу до `Применить` в родительской панели. |
| Обычные колонки | Сброс библиотечных добавлений не удаляет обычные колонки из черновика. |
| Пресеты | Библиотека атрибутов не создает, не сохраняет и не удаляет пресеты. |
| Карточный вид | Библиотека атрибутов не изменяет карточный вид списка задач. |
