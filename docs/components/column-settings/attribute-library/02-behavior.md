# Библиотека атрибутов: поведение

Этот файл описывает поведение дочерней зоны `Библиотека атрибутов` внутри родительского компонента `Настройки колонок`. Действия библиотеки меняют черновик родителя; таблица задач получает результат только после `Применить`.

## 1. Основной пользовательский путь

1. Пользователь открывает панель `Настройки колонок`.
2. Пользователь нажимает кнопку раскрытия библиотеки атрибутов в футере панели.
3. Слева от области `Общие атрибуты` становится видима область `Библиотека атрибутов`.
4. Область `Библиотека атрибутов` показывает доступные скрытые атрибуты текущего контекста.
5. Пользователь находит атрибут в списке или через поле `Поиск атрибута...`.
6. Пользователь нажимает действие добавления у нужного атрибута.
7. Выбранный атрибут остается в списке библиотеки; кнопка действия у этого атрибута показывает иконку стрелки назад.
8. Выбранный атрибут добавляется в конец списка `Общие атрибуты`; у добавленного атрибута видна кнопка действия с иконкой стрелки назад.
9. Основная таблица задач сохраняет прежний состав колонок до нажатия `Применить`.
10. Пользователь нажимает `Применить` в панели `Настройки колонок`.
11. Панель закрывается.
12. Табличный список задач показывает новую колонку.

## 2. Открытие и закрытие библиотеки

| Действие пользователя | Состояние библиотеки | Состояние футера | Состояние таблицы задач |
|---|---|---|---|
| Нажимает кнопку библиотеки в футере закрытой панели настроек | Библиотека видима слева от `Общие атрибуты` | Футер расширен на ширину раскрытой панели | Таблица не меняется |
| Нажимает кнопку библиотеки повторно | Библиотека скрыта | Футер возвращается к обычной ширине | Таблица не меняется |
| Закрывает всю панель `Настройки колонок` | Библиотека скрыта вместе с панелью | Футер скрыт вместе с панелью | Таблица не меняется, если `Применить` не нажата |

## 3. Поиск атрибута

| Действие пользователя | Состояние списка библиотеки |
|---|---|
| Вводит текст в поле `Поиск атрибута...` | В списке остаются атрибуты, в названии которых есть введенный текст |
| Очищает поле поиска | Список показывает все доступные атрибуты текущего контекста |
| Введенный текст не совпадает ни с одним атрибутом | В области списка показано сообщение `Атрибуты не найдены` |
| Добавляет атрибут при активном поиске | Добавленный атрибут остается в результатах текущего поиска; кнопка действия у этого атрибута показывает иконку стрелки назад |

Поиск работает только внутри библиотеки атрибутов. Поиск не меняет состав колонок таблицы, фильтры, список задач и выбранный контекст.

## 4. Добавление атрибута

| Действие пользователя | Состояние библиотеки | Состояние `Общие атрибуты` | Состояние таблицы задач |
|---|---|---|---|
| Нажимает кнопку действия с иконкой плюса у атрибута | Атрибут остается в списке; кнопка действия у этого атрибута показывает иконку стрелки назад | Атрибут добавляется в конец списка колонок | Таблица не меняется |
| Добавляет первый атрибут из библиотеки | Кнопка `Сбросить` в футере становится активной | У добавленного атрибута доступно действие возврата | Таблица не меняется |
| Добавляет несколько атрибутов | У каждого добавленного атрибута кнопка действия показывает иконку стрелки назад | Добавленные атрибуты остаются в списке до сброса, возврата или закрытия без применения | Таблица не меняется |
| Нажимает `Применить` после добавления | Библиотека скрывается вместе с панелью | Черновик колонок становится примененным состоянием | Таблица показывает добавленные колонки |

## 5. Возврат атрибута в библиотеку

| Действие пользователя | Состояние `Общие атрибуты` | Состояние библиотеки | Состояние таблицы задач |
|---|---|---|---|
| Нажимает кнопку действия с иконкой стрелки назад у добавленного атрибута в `Общие атрибуты` | Атрибут удаляется из списка колонок | В библиотеке у этого атрибута кнопка действия снова показывает иконку плюса, если атрибут подходит под текущий поиск | Таблица не меняется |
| Нажимает кнопку действия с иконкой стрелки назад у добавленного атрибута в библиотеке | Атрибут удаляется из списка колонок | В библиотеке у этого атрибута кнопка действия снова показывает иконку плюса | Таблица не меняется |
| Возвращает последний добавленный атрибут | В списке не остается библиотечных добавлений | Кнопка `Сбросить` становится неактивной | Таблица не меняется |
| Нажимает `Применить` после возврата | Черновик становится примененным состоянием | Панель закрывается | Таблица не показывает возвращенную колонку |

## 6. Сброс библиотечных добавлений

Кнопка `Сбросить` относится только к атрибутам, добавленным из библиотеки в текущем черновике.

| Условие | Состояние кнопки `Сбросить` |
|---|---|
| Библиотека скрыта | Кнопка не показана |
| Библиотека открыта, библиотечных добавлений нет | Кнопка показана в неактивном состоянии |
| Библиотека открыта, есть минимум один добавленный библиотечный атрибут | Кнопка активна |

| Действие пользователя | Состояние `Общие атрибуты` | Состояние библиотеки | Состояние таблицы задач |
|---|---|---|---|
| Нажимает активную кнопку `Сбросить` | Все библиотечные добавления удаляются из списка колонок | У сброшенных атрибутов кнопки действия снова показывают иконку плюса | Таблица не меняется |
| Нажимает `Применить` после сброса | Черновик без библиотечных добавлений становится примененным состоянием | Панель закрывается | Таблица не показывает сброшенные колонки |

Сброс библиотечных добавлений не сбрасывает обычные колонки и не возвращает весь список колонок к базовому состоянию. Для полного возврата к базовому набору используется отдельная кнопка `По умолчанию`.

## 7. Закрытие без применения

| Условие закрытия | Итог для таблицы задач |
|---|---|
| Пользователь добавил атрибуты и закрыл панель без `Применить` | Таблица сохраняет прежние колонки |
| Пользователь вернул атрибуты в библиотеку и закрыл панель без `Применить` | Таблица сохраняет прежние колонки |
| Пользователь нажал `Сбросить` и закрыл панель без `Применить` | Таблица сохраняет прежние колонки |
