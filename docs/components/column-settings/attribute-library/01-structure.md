# Библиотека атрибутов: структура интерфейса

## 1. Граница компонента

Библиотека атрибутов встроена в панель `Настройки колонок`. Она добавляет в черновик колонок скрытые атрибуты, которые не показаны в таблице по умолчанию.

`Библиотека атрибутов` является дочерней зоной родительского компонента `Настройки колонок`. Библиотека не владеет применением к таблице: она добавляет или возвращает атрибуты в черновике родительской панели.

| Зона | Относится к библиотеке | Что это означает |
|---|---|---|
| Область библиотеки | Да | Левая область, которая появляется внутри панели настроек колонок |
| Заголовок библиотеки | Да | Текст `Библиотека атрибутов` |
| Поиск атрибута | Да | Поле поиска внутри библиотеки |
| Список скрытых атрибутов | Да | Атрибуты, доступные для добавления |
| Действие добавления | Да | Добавляет атрибут в черновик `Общие атрибуты` |
| Пустое состояние списка | Да | Текст `Атрибуты не найдены` |
| Область `Общие атрибуты` | Связанная область | Принимает добавленные атрибуты |
| Футер панели настроек | Связанная область | Содержит кнопку открытия библиотеки и кнопку `Сбросить` |
| Таблица задач | Связанная область | Меняется только после `Применить` |

Библиотека не создает новые атрибуты, не редактирует значения задач и не влияет на карточный вид списка задач.

## 2. Состояния раскрытия

| Состояние | Что видит пользователь |
|---|---|
| Библиотека закрыта | Видна только область `Общие атрибуты`; список скрытых атрибутов не показан |
| Библиотека открыта | Слева показана область `Библиотека атрибутов`; справа остается область `Общие атрибуты` |

Библиотека не видна сразу после открытия панели настроек колонок. Пользователь открывает ее кнопкой в футере панели.

## 3. Область библиотеки

| Элемент | Положение | Что показывает |
|---|---|---|
| Заголовок `Библиотека атрибутов` | Верх области библиотеки | Название области со скрытыми атрибутами |
| Поле `Поиск атрибута...` | Под заголовком | Текстовый ввод для поиска атрибута по названию |
| Список атрибутов | Под поиском | Скрытые атрибуты, доступные для добавления в текущем контексте |
| Действие добавления | Справа в строке атрибута | Добавляет выбранный атрибут в `Общие атрибуты` |
| Пустое состояние | На месте списка | Текст `Атрибуты не найдены` |

## 4. Строка атрибута

| Элемент строки | Что показывает |
|---|---|
| Название атрибута | Пользовательское название, например `Юридическое лицо` или `Дата поставки` |
| Кнопка действия | Показывает иконку плюса, если атрибут не добавлен в `Общие атрибуты`; показывает иконку стрелки назад, если атрибут уже добавлен в `Общие атрибуты` |

После добавления атрибут остается в списке библиотеки. В строке этого атрибута кнопка действия меняет иконку плюса на иконку стрелки назад. Такая же кнопка действия с иконкой стрелки назад видна у добавленного атрибута в списке `Общие атрибуты`.

## 5. Футер при открытой библиотеке

Футер панели настроек масштабируется вместе с раскрытой библиотекой.

| Элемент футера | Когда виден | Что делает |
|---|---|---|
| Кнопка библиотеки | Всегда | Открывает или закрывает библиотеку |
| `Сбросить` | Только при открытой библиотеке | Удаляет из черновика все атрибуты, добавленные из библиотеки |
| `По умолчанию` | Всегда | Возвращает черновик колонок к базовому набору |
| `Применить` | Всегда | Применяет текущий черновик колонок к таблице |

Кнопка `Сбросить` неактивна, пока в черновике нет атрибутов, добавленных из библиотеки. После первого добавления атрибута кнопка становится активной.

## 6. Наборы атрибутов текущего прототипа

Состав библиотеки зависит от выбранного бизнес-домена или режима `Все задачи`.

| Выбранная область | Атрибуты, доступные в библиотеке |
|---|---|
| `Строительные договоры` | `Внутренний номер договора`, `Юридическое лицо`, `Ставка НДС`, `Валюта договора`, `Дата начала действия`, `Дата окончания действия`, `Дата фактической регистрации`, `Ссылка на скан-копию`, `Условия оплаты`, `Тип доп. соглашения`, `Ответственный от контрагента`, `Гарантийный срок`, `Этап реализации проекта`, `Ссылка на связанные сметы` |
| `Закупки ТМЦ` | `Инициатор заявки`, `Поставщик`, `Дата поставки`, `Приоритет`, `Этап закупки` |
| `Поручения СД` | `Ответственный контролер`, `План устранения`, `Зона проверки`, `Приоритет` |
| `Все задачи` | Объединенный список скрытых атрибутов всех доступных бизнес-доменов; одинаковые атрибуты показываются один раз |

Текущий набор бизнес-доменов относится к одной роли в прототипе. Для другой роли количество доменов и состав библиотеки могут отличаться.

## 7. Связь с таблицей задач

| Действие в библиотеке | Что меняется сразу | Что меняется после `Применить` |
|---|---|---|
| Добавить атрибут | Атрибут появляется в черновике `Общие атрибуты`; в библиотеке у этого атрибута кнопка действия показывает иконку стрелки назад | В таблице появляется новая колонка |
| Нажать кнопку действия с иконкой стрелки назад у атрибута в библиотеке или в `Общие атрибуты` | Атрибут исчезает из черновика `Общие атрибуты`; в библиотеке у этого атрибута кнопка действия снова показывает иконку плюса | Колонка исчезает из таблицы |
| Нажать `Сбросить` | Все библиотечные добавления удаляются из черновика; в библиотеке у этих атрибутов кнопки действия снова показывают иконку плюса | Таблица возвращается к колонкам без библиотечных добавлений |
| Закрыть панель без `Применить` | Черновик не применяется | Таблица не меняется |
