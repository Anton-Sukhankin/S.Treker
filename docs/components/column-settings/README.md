# Настройки колонок

Эта папка описывает компонентный комплекс `Настройки колонок`.

`Настройки колонок` являются родительским компонентом для зон, которые меняют состав, порядок и видимость динамических колонок таблицы задач.

`Черновик колонок` — временный набор динамических колонок внутри открытой панели. Он включает состав, порядок, видимость и признак добавления из библиотеки. Таблица задач не читает черновик напрямую.

`Базовое отображение` — минимальный набор отображаемых динамических колонок текущего контекста без библиотечных добавлений. Для одного и того же контекста этот набор одинаковый до пользовательских изменений и используется как начальное состояние черновика.

## Состав документации

| Зона | Путь | Контракт с родительским компонентом |
|---|---|---|
| Родительский компонент `Настройки колонок` | `01-structure.md`, `02-behavior.md`, `03-data-states-rules.md` | Владеет черновиком колонок, футером действий и применением черновика к таблице. |
| Список колонок | `01-structure.md`, `02-behavior.md`, `03-data-states-rules.md` | Показывает черновик колонок и меняет в нем видимость, порядок и возврат библиотечных атрибутов. |
| Пресеты колонок | `presets/01-structure.md`, `presets/02-behavior.md`, `presets/03-data-states-rules.md` | Подставляют базовый или сохраненный набор колонок в черновик родительского компонента. |
| Библиотека атрибутов | `attribute-library/01-structure.md`, `attribute-library/02-behavior.md`, `attribute-library/03-data-states-rules.md` | Добавляет скрытые атрибуты в черновик или удаляет из него атрибуты, добавленные из библиотеки. |

## Правило связи

Дочерние зоны не применяют изменения к таблице напрямую. Они меняют черновик колонок внутри `Настроек колонок`.

Таблица задач получает новый состав, порядок и видимость динамических колонок только после действия `Применить` в родительском компоненте.
