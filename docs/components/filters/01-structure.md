# Фильтрация: структура интерфейса

## 1. Раскрытие компонента

Интерфейс фильтрации имеет внешнюю точку входа: кнопку `Фильтры` в панели инструментов.

Кнопка `Фильтры` не входит в компонент фильтрации. Состояния, состав и поведение кнопки относятся к спецификации панели инструментов.
После открытия правая панель фильтрации перекрывает область внешней кнопки `Фильтры`. Закрытие выполняется только через кнопку закрытия в хедере самой панели фильтрации.

| Точка входа вне компонента | Раскрытие компонента фильтрации |
|---|---|
| Кнопка `Фильтры` в панели инструментов | Затемнение фона видимо; правая панель фильтрации видима; область внешней кнопки перекрыта панелью. |

## 2. Каркас компонента

| Часть компонента | Позиция | Состав |
|---|---|---|
| Затемнение фона | Поверх рабочей области, за правой панелью | Полупрозрачный слой без собственных параметров и действий. |
| Правая панель фильтрации | Правый край экрана, поверх рабочей области | Боковой дровер с хедером, областью параметров и футером действий. |

## 3. Внешние связи компонента

| Внешняя область | Связь с фильтрацией |
|---|---|
| Левая навигация бизнес-доменов | Определяет видимость доменных блоков внутри панели. |
| Очереди | Ограничивают набор задач, из которого формируются значения мультивыбора. |
| Список задач | После `Применить` показывает задачи, прошедшие параметры панели фильтрации. |
| Пагинация списка задач | После `Применить` строится по отфильтрованному набору задач. |

## 4. Состояния компонента

| Состояние | Видимые части | Недоступные части |
|---|---|---|
| Закрыто | Нет видимых частей компонента фильтрации. | Правая панель; затемнение фона; хедер панели; параметры; футер. |
| Открыто | Затемнение фона; правая панель; хедер; область параметров; футер. | Нет. |

## 5. Структура правой панели

| Зона панели | Состав | Условие показа |
|---|---|---|
| Хедер | Иконка фильтра; заголовок `Фильтры`; строка `Выполнено настроек: N`; кнопка закрытия с иконкой крестика | Только при открытой панели. |
| Область параметров | Общий блок; блок `Строительные договоры`; блок `Закупки ТМЦ`; блок `Поручения СД` | Только при открытой панели. |
| Футер | Кнопка `Сбросить всё`; кнопка `Применить` | Только при открытой панели. |

`N` — количество активных параметров внутри панели фильтрации.

## 6. Блоки параметров

Таблица ниже фиксирует текущий видимый набор блоков и параметров для роли, представленной в прототипе. Для другой роли состав бизнес-доменов и связанных доменных блоков может отличаться.

| Блок | Заголовок в панели | Условие показа | Параметры |
|---|---|---|---|
| Общие параметры | Не отображается | Всегда | `Номер`; `Наименование`. |
| Строительные договоры | `Строительные договоры` | Выбран `Все задачи` или бизнес-домен `Строительные договоры` | `Тип документа`; `Контрагент`; `Сумма, ₽`. |
| Закупки ТМЦ | `Закупки ТМЦ` | Выбран `Все задачи` или бизнес-домен `Закупки ТМЦ` | `Бизнес-Юнит`; `SLA`. |
| Поручения СД | `Поручения СД` | Выбран `Все задачи` или бизнес-домен `Поручения СД` | `Объект`; `Критичность`. |

## 7. Параметры фильтрации

| Параметр | Блок | Тип управления | Видимое значение до выбора |
|---|---|---|---|
| `Номер` | Общие параметры | Мультивыбор | `Выберите номер` |
| `Наименование` | Общие параметры | Мультивыбор | `Выберите наименование` |
| `Тип документа` | Строительные договоры | Мультивыбор | `Выберите тип документа` |
| `Контрагент` | Строительные договоры | Мультивыбор | `Выберите контрагента` |
| `Сумма, ₽` | Строительные договоры | Денежный диапазон | Поля `От` и `До` пустые |
| `Бизнес-Юнит` | Закупки ТМЦ | Мультивыбор | `Выберите БЮ` |
| `SLA` | Закупки ТМЦ | Мультивыбор | `Выберите SLA` |
| `Объект` | Поручения СД | Мультивыбор | `Выберите объект` |
| `Критичность` | Поручения СД | Мультивыбор | `Выберите критичность` |

## 8. Мультивыбор

| Элемент | Условие показа | Видимое состояние |
|---|---|---|
| Метка параметра | Всегда | Текст названия параметра над полем. |
| Поле выбора | Всегда | Кликабельная область с плейсхолдером или выбранными тегами. |
| Плейсхолдер | Нет выбранных значений | Текст подсказки внутри поля. |
| Тег выбранного значения | Выбрано минимум одно значение | Первый выбранный вариант внутри поля. |
| Крестик в теге | Показан тег выбранного значения | Кнопка удаления значения внутри тега. |
| Тег `+N` | Выбрано больше одного значения | Количество выбранных значений сверх первого. |
| Кнопка очистки поля | Выбрано минимум одно значение | Кнопка очистки всех выбранных значений только этого параметра. |
| Стрелка раскрытия | Всегда | Индикатор выпадающего списка. |
| Закрытое состояние поля | После открытия панели или после закрытия поля | Выпадающий список не виден. |
| Открытое состояние поля | Поле раскрыто | Выпадающий список виден под полем. |
| Выпадающий список | Поле раскрыто | Список доступных значений. Если значений нет, строка `Нет доступных значений`. |

## 9. Денежный диапазон `Сумма, ₽`

| Элемент | Видимое состояние |
|---|---|
| Поле `От` | Текстовое поле нижней границы; принимает числовое значение. |
| Поле `До` | Текстовое поле верхней границы; принимает числовое значение. |
| Разделитель `-` | Находится между полями `От` и `До`. |
| Шкала | Горизонтальная линия под полями ввода. |
| Левый бегунок | Позиция нижней границы. |
| Правый бегунок | Позиция верхней границы. |
| Закрашенный участок | Интервал между левым и правым бегунком. |
| Пресет `до 500 тыс.` | Кнопка быстрого выбора диапазона до 500 000 ₽. |
| Пресет `1 млн - 5 млн` | Кнопка быстрого выбора диапазона 1 000 000-5 000 000 ₽. |
| Пресет `свыше 10 млн` | Кнопка быстрого выбора диапазона от 10 000 000 ₽. |

Правила ввода, границ шкалы, синхронизации полей и бегунков описаны в `02-behavior.md` и `03-data-states-rules.md`.

## 10. Футер панели

| Кнопка | Позиция | Назначение в футере |
|---|---|---|
| `Сбросить всё` | Левая часть футера | Кнопка полного сброса параметров панели. |
| `Применить` | Правая часть футера | Кнопка применения параметров панели к списку задач. |
