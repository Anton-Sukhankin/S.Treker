# Фильтрация: поведение

## 1. Основной пользовательский путь

1. Пользователь нажимает внешнюю кнопку `Фильтры` в панели инструментов.
2. Затемнение фона становится видимым; справа видна панель фильтрации.
3. Открытая панель перекрывает область внешней кнопки `Фильтры`.
4. Пользователь выбирает один или несколько параметров в панели.
5. Число в строке `Выполнено настроек: N` получает значение количества активных параметров панели.
6. Пользователь нажимает `Применить`.
7. Правая панель фильтрации и затемнение фона скрыты.
8. Таблица или карточки показывают только задачи, которые проходят все активные параметры панели фильтрации.

## 2. Открытие панели

| Действие пользователя | Условие | Итог интерфейса |
|---|---|---|
| Клик по внешней кнопке `Фильтры` в панели инструментов | В левой панели выбран `Все задачи` | Видны общий блок, `Строительные договоры`, `Закупки ТМЦ`, `Поручения СД`. |
| Клик по внешней кнопке `Фильтры` в панели инструментов | В левой панели выбран конкретный бизнес-домен | Видны общий блок и блок выбранного бизнес-домена; блоки остальных бизнес-доменов не видны. |
| Клик по внешней кнопке `Фильтры` в панели инструментов | Есть активные параметры скрытых бизнес-доменов | Параметры скрытых бизнес-доменов получают пустое значение до показа панели. |
| Клик по внешней кнопке `Фильтры` в панели инструментов | Есть значение `Сумма, ₽` | Поля `От` и `До`, бегунки, закрашенный участок шкалы и активный пресет получают состояние текущего диапазона. |
| Клик по внешней кнопке `Фильтры` в панели инструментов | Любое состояние параметров панели | Строка `Выполнено настроек: N` показывает количество активных параметров панели. |
| Клик по внешней кнопке `Фильтры` в панели инструментов | Всегда | Затемнение фона и правая панель фильтрации становятся видимыми. |
| Клик по внешней кнопке `Фильтры` в панели инструментов | Панель уже открыта | Действие недоступно: правая панель перекрывает область внешней кнопки. |

## 3. Закрытие панели без применения

| Действие пользователя | Итог интерфейса | Что сохраняется |
|---|---|---|
| Клик по кнопке закрытия с иконкой крестика в хедере панели | Затемнение фона и правая панель скрыты. | Текущие значения параметров панели остаются выбранными. |

Если пользователь изменил параметры внутри панели и не нажал `Применить`, таблица или карточки сохраняют прежний набор задач.

## 4. Мультивыбор: раскрытие поля

После открытия панели фильтрации все мультивыборы закрыты. Выпадающие списки не видны.

| Состояние параметра | Видимое состояние поля |
|---|---|
| Значений нет | В поле виден плейсхолдер параметра. |
| Выбрано одно значение | В поле виден тег выбранного значения; кнопка очистки поля видима. |
| Выбрано больше одного значения | В поле виден тег первого выбранного значения; рядом виден тег `+N`; кнопка очистки поля видима. |

| Действие пользователя | Условие | Итог интерфейса |
|---|---|---|
| Клик по закрытому полю мультивыбора | Все мультивыборы закрыты | Текущий мультивыбор раскрыт; выпадающий список видим; список вариантов построен заново. |
| Клик по закрытому полю мультивыбора | Другой мультивыбор раскрыт | Другой мультивыбор закрыт; текущий мультивыбор раскрыт; выпадающий список текущего поля видим; список вариантов построен заново. |
| Клик по открытому полю мультивыбора | Текущий мультивыбор раскрыт | Текущий мультивыбор закрыт; выбранные значения параметра сохраняются. |
| Клик вне мультивыборов | Минимум один мультивыбор раскрыт | Все мультивыборы закрыты; выбранные значения параметров сохраняются. |
| Клик по кнопке очистки поля | В параметре выбрано минимум одно значение | Поле получает плейсхолдер; теги удалены; кнопка очистки скрыта; варианты в списке не имеют выбранного состояния. |
| Клик по кнопке удаления с иконкой крестика внутри тега | В параметре выбрано несколько значений | Значение этого тега удалено; остальные значения остаются выбранными; поле показывает актуальный первый тег и тег `+N`, если осталось больше одного значения. |
| Клик по кнопке удаления с иконкой крестика внутри тега | В параметре выбрано одно значение | Поле получает плейсхолдер; тег удален; кнопка очистки скрыта. |

## 5. Выбор значения в мультивыборе

| Действие пользователя | Состояние значения до клика | Итог интерфейса |
|---|---|---|
| Клик по значению в раскрытом списке | Значение не выбрано | Значение добавлено в список выбранных значений; строка значения получает выбранное состояние; поле показывает первый выбранный тег; кнопка очистки поля видима; выпадающий список остается раскрытым. |
| Клик по значению в раскрытом списке | Значение выбрано, и в параметре выбрано больше одного значения | Значение удалено из списка выбранных значений; строка значения теряет выбранное состояние; поле показывает оставшиеся выбранные значения; выпадающий список остается раскрытым. |
| Клик по значению в раскрытом списке | Значение выбрано и является единственным выбранным значением | Значение удалено из списка выбранных значений; поле показывает плейсхолдер; кнопка очистки поля скрыта; выпадающий список остается раскрытым. |

После выбора или снятия значения число `N` в строке `Выполнено настроек: N` получает количество активных параметров панели. Таблица или карточки получают новый набор задач только после `Применить`.

## 6. Список вариантов мультивыбора

| Событие | Условие | Итог в выпадающем списке |
|---|---|---|
| Поле мультивыбора раскрыто | Есть доступные значения | Список показывает значения из задач текущего контекста; выбранные значения имеют выбранное состояние. |
| Поле мультивыбора раскрыто | Нет доступных значений | Список показывает строку `Нет доступных значений`. |
| Значение выбрано или снято | Список раскрыт | Список вариантов построен заново; выбранные значения получают выбранное состояние; снятые значения не имеют выбранного состояния. |

При построении вариантов текущий параметр не учитывает собственное выбранное значение. Остальные активные параметры панели сужают набор задач, из которого собираются варианты.

## 7. Ручной ввод суммы

| Действие пользователя | Условие | Итог интерфейса |
|---|---|---|
| Ввод в поле `От` или `До` | Введены цифры и другие символы | В значение границы попадают только цифры. |
| Ввод в поле `От` или `До` | Значение не кратно `10 000` | Значение границы округлено к шагу `10 000`. |
| Ввод в поле `От` или `До` | Значение меньше `0` | Значение границы установлено в `0`. |
| Ввод в поле `От` или `До` | Значение больше `20 000 000` | Значение границы установлено в `20 000 000`. |
| Ввод в поле `От` | `От` больше `До` | Верхняя граница получает значение нижней границы. |
| Ввод в поле `До` | `До` меньше `От` | Нижняя граница получает значение верхней границы. |
| Ввод в поле `От` или `До` | Заполнена минимум одна граница | Параметр `Сумма, ₽` считается активным; число `N` получает количество активных параметров панели. |

Пока поле находится в фокусе, значение показано без знака рубля. После потери фокуса значение показано с разделением разрядов и знаком `₽`.

## 8. Изменение суммы ползунком

| Действие пользователя | Условие | Итог интерфейса |
|---|---|---|
| Перемещение левого бегунка | Бегунок находится выше минимума `0` | Поле `От` получает значение левого бегунка. |
| Перемещение левого бегунка | Бегунок находится на минимуме `0` | Нижняя граница считается пустой; поле `От` пустое. |
| Перемещение правого бегунка | Бегунок находится ниже максимума `20 000 000` | Поле `До` получает значение правого бегунка. |
| Перемещение правого бегунка | Бегунок находится на максимуме `20 000 000` | Верхняя граница считается пустой; поле `До` пустое. |
| Перемещение любого бегунка | Бегунки задают интервал | Закрашенный участок шкалы занимает расстояние между левым и правым бегунком. |
| Перемещение любого бегунка | Заполнена минимум одна граница | Число `N` получает количество активных параметров панели. |

Левый бегунок не может оказаться правее правого. Правый бегунок не может оказаться левее левого.

## 9. Быстрые пресеты суммы

| Действие пользователя | Итог значения `Сумма, ₽` | Итог интерфейса |
|---|---|---|
| Клик по `до 500 тыс.` | `От` пустое; `До` = `500 000` | Поле `До`, правый бегунок, закрашенный участок и активность пресета соответствуют диапазону до 500 000 ₽. |
| Клик по `1 млн - 5 млн` | `От` = `1 000 000`; `До` = `5 000 000` | Поля, бегунки, закрашенный участок и активность пресета соответствуют диапазону 1 000 000-5 000 000 ₽. |
| Клик по `свыше 10 млн` | `От` = `10 000 000`; `До` пустое | Поле `От`, левый бегунок, закрашенный участок и активность пресета соответствуют диапазону от 10 000 000 ₽. |

После клика по пресету число `N` получает количество активных параметров панели. Таблица или карточки получают новый набор задач только после `Применить`.

## 10. Применение фильтров

| Действие пользователя | Условие | Итог интерфейса |
|---|---|---|
| Клик по `Применить` | В полях суммы есть текст | Значения `От` и `До` очищены от лишних символов, ограничены диапазоном `0`-`20 000 000` и записаны как текущий диапазон `Сумма, ₽`. |
| Клик по `Применить` | До этого пользователь нажал `Сбросить всё` в футере панели | Список задач получает набор без параметров панели фильтрации. |
| Клик по `Применить` | Выбран конкретный бизнес-домен | Параметры других бизнес-доменов получают пустое значение. |
| Клик по `Применить` | Любое состояние параметров панели | Строка `Выполнено настроек: N` показывает количество активных параметров панели. |
| Клик по `Применить` | Всегда | Затемнение фона и правая панель скрыты. |
| Клик по `Применить` | Есть задачи, проходящие параметры панели | Таблица или карточки показывают отфильтрованный набор задач. |
| Клик по `Применить` | Нет задач, проходящих параметры панели | Список задач показывает пустое состояние. |

Табличное и карточное отображение показывают один и тот же отфильтрованный набор задач.

## 11. Сброс значений через футер панели

| Действие пользователя | Область панели | Итог интерфейса |
|---|---|---|
| Клик по `Сбросить всё` | Мультивыборы | Теги удалены; плейсхолдеры видимы; кнопки очистки скрыты; значения в списках не имеют выбранного состояния. |
| Клик по `Сбросить всё` | `Сумма, ₽` | Поля `От` и `До` пустые; левый бегунок на `0`; правый бегунок на `20 000 000`; пресеты неактивны. |
| Клик по `Сбросить всё` | Раскрытые мультивыборы | Выпадающие списки скрыты; выбранные значения параметров удалены. |
| Клик по `Сбросить всё` | Строка `Выполнено настроек: N` в хедере панели | Значение `N` установлено в `0`. |
| Клик по `Сбросить всё` | Правая панель фильтрации | Затемнение фона и правая панель остаются видимыми. |
| Клик по `Сбросить всё` | Список задач за пределами панели | Таблица или карточки сохраняют текущий набор задач до клика по `Применить`. |
