# Фильтрация: данные, состояния и правила

## 1. Владельцы состояния

| Область | Состояние или значение | Правило видимого результата |
|---|---|---|
| Панель фильтрации | Значения параметров открытой панели; счетчик `Выполнено настроек: N` | Хедер панели показывает количество активных параметров открытой панели. |
| Примененное состояние фильтрации | Параметры, подтвержденные через `Применить` или очищенные быстрым сбросом | Список задач, счетчики левой навигации, счетчики очередей и пагинация используют примененные параметры. |
| Панель инструментов | Счетчик на кнопке `Фильтры` | Кнопка `Фильтры` показывает круглый счетчик, если применен минимум один параметр панели фильтрации. |
| Список задач | Итоговый набор задач после применения параметров | Табличный и карточный вид показывают задачи, которые проходят примененные параметры. |

## 2. Интерфейсные состояния параметров

| Параметр | Тип состояния | Область действия | Пустое значение | Активное значение |
|---|---|---|---|---|
| `Номер` | Список выбранных значений | Все области задач | Список пуст | Выбрано минимум одно значение |
| `Наименование` | Список выбранных значений | Все области задач | Список пуст | Выбрано минимум одно значение |
| `Тип документа` | Список выбранных значений | `Строительные договоры` | Список пуст | Выбрано минимум одно значение |
| `Контрагент` | Список выбранных значений | `Строительные договоры` | Список пуст | Выбрано минимум одно значение |
| `Сумма, ₽` | Диапазон `От`-`До` | `Строительные договоры` | `От` пусто; `До` пусто | Заполнена минимум одна граница |
| `Бизнес-Юнит` | Список выбранных значений | `Закупки ТМЦ` | Список пуст | Выбрано минимум одно значение |
| `SLA` | Список выбранных значений | `Закупки ТМЦ` | Список пуст | Выбрано минимум одно значение |
| `Объект` | Список выбранных значений | `Поручения СД` | Список пуст | Выбрано минимум одно значение |
| `Критичность` | Список выбранных значений | `Поручения СД` | Список пуст | Выбрано минимум одно значение |

Пустой параметр не исключает задачи из списка и не увеличивает число `N` в строке `Выполнено настроек: N`.

## 3. Правила доменного контекста

| Выбранная область задач | Активные области параметров | Правило для параметров вне контекста |
|---|---|---|
| `Все задачи` | Общие параметры; `Строительные договоры`; `Закупки ТМЦ`; `Поручения СД` | Параметры всех доменных областей сохраняют текущие значения и участвуют в отборе. |
| `Строительные договоры` | Общие параметры; `Строительные договоры` | Параметры `Закупки ТМЦ` и `Поручения СД` получают пустые значения при открытии панели и при `Применить`. |
| `Закупки ТМЦ` | Общие параметры; `Закупки ТМЦ` | Параметры `Строительные договоры` и `Поручения СД` получают пустые значения при открытии панели и при `Применить`. |
| `Поручения СД` | Общие параметры; `Поручения СД` | Параметры `Строительные договоры` и `Закупки ТМЦ` получают пустые значения при открытии панели и при `Применить`. |

Параметр вне текущего доменного контекста не проверяет задачу.

## 4. Счетчик активных параметров

Счетчик считает активные параметры панели, а не количество выбранных значений внутри параметра.

| Состояние параметра | Вклад в счетчик |
|---|---:|
| В `Контрагент` выбраны три значения | `+1` |
| В `Контрагент` выбрано одно значение | `+1` |
| Выбраны `Контрагент` и `Тип документа` | `+2` |
| В `Сумма, ₽` заполнены `От` и `До` | `+1` |
| В `Сумма, ₽` заполнено только `От` | `+1` |
| Параметр пустой | `+0` |
| Параметр находится вне текущего доменного контекста | `+0` |

| Место счетчика | Условие | Видимый результат |
|---|---|---|
| Хедер панели | Активных параметров нет | `Выполнено настроек: 0` |
| Хедер панели | Активен минимум один параметр | `Выполнено настроек: N`, где `N` равно количеству активных параметров панели |
| Кнопка `Фильтры` в панели инструментов | Активных параметров нет | Видна иконка фильтра; круглый счетчик скрыт |
| Кнопка `Фильтры` в панели инструментов | Активен минимум один параметр | Иконка фильтра скрыта; круглый счетчик показывает `N` |

Поисковая строка `Поиск по задачам` не входит в счетчик параметров панели фильтрации.

## 5. Правило прохождения задачи

Задача видна после применения фильтров, если она проходит все активные параметры текущего контекста.

| Условие параметра | Итог для задачи |
|---|---|
| Параметр пустой | Параметр не проверяет задачу. |
| Параметр находится вне текущего доменного контекста | Параметр не проверяет задачу. |
| Параметр активен и относится к текущему контексту | Задача проверяется по правилу этого параметра. |
| Задача не проходит минимум один активный параметр | Задача не видна в списке. |
| Задача проходит все активные параметры | Задача видна в списке. |

## 6. Правила по типам параметров

| Тип состояния | Параметры | Задача проходит | Задача не проходит |
|---|---|---|---|
| Список выбранных значений | `Номер`; `Наименование`; `Тип документа`; `Контрагент`; `Бизнес-Юнит`; `SLA`; `Объект`; `Критичность` | Значение задачи точно совпадает минимум с одним выбранным значением. | Значение задачи не совпадает ни с одним выбранным значением. |
| Диапазон `От`-`До` | `Сумма, ₽` | У задачи есть сумма; сумма не меньше `От`, если `От` заполнено; сумма не больше `До`, если `До` заполнено. | У задачи нет суммы; сумма меньше `От`; сумма больше `До`. |

## 7. Правила вариантов мультивыбора

Варианты мультивыбора строятся из задач текущего контекста.

| Условие | Набор вариантов |
|---|---|
| Выбраны `Все задачи` | Значения берутся из задач всех доступных бизнес-доменов. |
| Выбран конкретный бизнес-домен | Значения берутся из задач выбранного бизнес-домена. |
| Активна очередь | Значения берутся из задач активной очереди. |
| Есть активные параметры панели, кроме текущего | Значения берутся из задач, которые проходят эти параметры. |
| Текущий параметр имеет выбранное значение | Собственное выбранное значение текущего параметра не сужает список его вариантов. |
| Доступных значений нет | Выпадающий список показывает строку `Нет доступных значений`. |

Настройка одного параметра сужает список доступных значений в других параметрах. Последовательность выбора параметров не задана.

## 8. Контракты взаимодействия

| Связанный компонент | Роль связанного компонента | Условие | Результат в фильтрации | Результат в связанном компоненте |
|---|---|---|---|---|
| Панель инструментов | Источник открытия панели и быстрого сброса примененных параметров | Пользователь открывает панель или нажимает круглый счетчик на кнопке `Фильтры` | Панель видна после открытия; после быстрого сброса параметры получают пустые значения | Кнопка `Фильтры` показывает иконку фильтра и скрывает круглый счетчик. |
| Список задач | Получатель примененных параметров | Пользователь нажал `Применить` | Примененное состояние фильтрации получает активные параметры панели | Табличный и карточный вид показывают задачи, которые проходят примененные параметры. |
| Левая навигация | Получатель примененных параметров для счетчиков | Пользователь нажал `Применить` или быстро сбросил фильтры | Примененное состояние фильтрации получает новый набор параметров | Счетчики левой навигации соответствуют задачам, которые проходят примененные параметры. |
| Очереди | Получатель примененных параметров для счетчиков | Пользователь нажал `Применить` или быстро сбросил фильтры | Примененное состояние фильтрации получает новый набор параметров | Счетчики очередей соответствуют задачам, которые проходят примененные параметры. |
| Пагинация | Получатель итогового набора задач | Пользователь нажал `Применить` или быстро сбросил фильтры | Примененное состояние фильтрации получает новый набор параметров | Текущая страница получает значение `1`; страницы строятся по задачам, которые проходят примененные параметры. |

## 9. Ограничения ответственности

| Область | Правило |
|---|---|
| Исходные данные задач | Панель фильтрации не изменяет значения задач. |
| Поисковая строка | Панель фильтрации не изменяет значение поля `Поиск по задачам`. |
| Режим отображения | Панель фильтрации не переключает карточный и табличный вид. |
| Настройки колонок | Панель фильтрации не изменяет состав, видимость и порядок колонок. |
| Бизнес-домены | Панель фильтрации не изменяет состав бизнес-доменов, доступных текущей роли. |
