# Список задач: структура интерфейса

## 1. Граница компонента

Список задач расположен в правой рабочей области под заголовком, панелью инструментов и очередями.

| Зона | Входит в список задач | Что видит пользователь |
|---|---|---|
| Область карточного вида | Да | Вертикальный список карточек задач |
| Область табличного вида | Да | Таблица задач с закрепленными и динамическими колонками |
| Карточка задачи | Да | Одно представление задачи в карточном виде |
| Строка таблицы | Да | Одно представление задачи в табличном виде |
| Чекбоксы выбора задач | Да | Выбор одной задачи или всех видимых строк таблицы |
| Пагинация | Нет | Расположена под списком задач; задает текущую страницу итогового набора |
| Пустое состояние | Да | Сообщение вместо карточек или строк, если задач нет |
| Панель инструментов над списком | Нет | Содержит поиск, переключатель вида, фильтры и действия |
| Очереди | Нет | Задают активный этап согласования над списком задач |
| Левая навигация | Нет | Задает область задач: все задачи, бизнес-домен, группу или сгруппированное значение |
| Панель фильтрации | Нет | Передает списку только примененные параметры фильтрации |
| Настройки колонок | Нет | Передают таблице примененный состав динамических колонок |

Список задач не открывает фильтры, не выбирает бизнес-домен и не меняет настройки колонок. Он показывает итоговый набор задач после применения внешних условий.

## 2. Внешние условия, которые меняют список

| Внешнее условие | Как влияет на список задач |
|---|---|
| Раздел `Все задачи` | В список попадают задачи всех бизнес-доменов, доступных текущей роли |
| Выбранный бизнес-домен | В список попадают только задачи выбранного бизнес-домена |
| Выбранная пользовательская группа | В список попадают только задачи выбранной группы |
| Выбранное сгруппированное значение | В список попадают задачи выбранного значения внутри бизнес-домена |
| Активная очередь | В список попадают задачи только активного этапа согласования |
| Поисковая строка | В список попадают задачи, совпадающие с введенным текстом |
| Примененные фильтры | В список попадают задачи, прошедшие все выбранные параметры |
| Текущая страница | Виден только фрагмент итогового набора задач |
| Примененные настройки колонок | В табличном виде меняется состав и порядок динамических колонок |

## 3. Режимы отображения

| Режим | Когда виден | Состав |
|---|---|---|
| Карточный вид | Активен сегмент карточного вида в переключателе вида панели инструментов | Вертикальный список карточек задач |
| Табличный вид | Активен сегмент табличного вида в переключателе вида панели инструментов | Таблица с заголовком, строками задач, горизонтальным скроллом и закрепленными колонками |

Переключатель вида не входит в список задач. В каждый момент виден только один режим. Переключение режима не меняет итоговый набор задач.

## 4. Общие элементы задачи

Карточка и строка таблицы показывают одну и ту же задачу в разных компоновках.

| Элемент задачи | Карточный вид | Табличный вид |
|---|---|---|
| Чекбокс выбора | Слева от содержимого карточки | Первая закрепленная колонка |
| Номер задачи | В основной зоне карточки перед наименованием | Закрепленная колонка `№` |
| Наименование | В основной зоне карточки после номера | Закрепленная колонка `Наименование` |
| Иконка задачи | Слева от текста наименования | Слева от текста наименования |
| Статус задачи | Цветной тег в правой зоне карточки | Цветной тег в колонке `Статус`, если колонка активна |
| Автор | Аватар, имя и email в правой зоне карточки | Аватар, имя и email в колонке автора, если колонка активна |
| Дата | Одна дата в правой зоне карточки | Значения в активных колонках дат |

Общие правила:

| Элемент | Правило отображения |
|---|---|
| Наименование | Текст от двух до десяти слов; при нехватке ширины обрезается многоточием |
| Иконка задачи | Стоит слева от наименования |
| Дата | Слева от значения даты стоит иконка календаря |
| Автор | Аватар 24 x 24 px, имя автора, email под именем |
| Email автора | Кликабельный текст для копирования адреса |
| Статус | Цветной контрастный тег с текстом статуса |

## 5. Карточка задачи

Карточка задачи — одна горизонтальная строка внутри карточного вида.

| Зона карточки | Состав |
|---|---|
| Левая зона | Чекбокс выбора задачи |
| Основная зона | Номер задачи, иконка задачи, наименование задачи |
| Правая зона | Тег статуса, автор, дата |
| Левая индикация | Цветная вертикальная индикация для новой или просроченной задачи |

| Состояние карточки | Визуальный признак |
|---|---|
| Обычная задача | Без зеленой или красной левой индикации |
| Наведение на обычную задачу | Фон карточки становится светло-серым |
| Выбранная задача | Чекбокс активен; карточка сохраняет выбранный фон до снятия чекбокса |
| Новая задача | Зеленоватая подложка и зеленая левая индикация |
| Просроченная задача | Розоватая подложка и красная левая индикация; значение срока красное |

## 6. Табличный вид

Табличный вид состоит из заголовка таблицы, строк задач и области горизонтального скролла.

| Часть таблицы | Состав |
|---|---|
| Заголовок | Названия активных колонок и чекбокс выбора всех видимых строк |
| Тело таблицы | Строки задач текущей страницы |
| Закрепленная область | Колонки `чекбокс выбора`, `№`, `Наименование` |
| Динамическая область | Колонки, примененные через настройки колонок |
| Горизонтальный скролл | Доступен, если ширина колонок больше ширины области таблицы |

## 7. Закрепленные колонки таблицы

| Колонка | Положение | Настройки колонок |
|---|---|---|
| Чекбокс выбора | Первая колонка слева | Не скрывается и не перемещается |
| `№` | После чекбокса | Не скрывается и не перемещается |
| `Наименование` | После номера | Не скрывается и не перемещается |

При горизонтальном скролле закрепленные колонки остаются слева. Динамические колонки скроллятся под ними.

## 8. Заголовок таблицы

| Колонка заголовка | Что видно |
|---|---|
| Чекбокс выбора | Чекбокс выбора всех видимых строк текущей страницы |
| `№` | Текст `№` |
| `Наименование` | Текст `Наименование` без подчеркивания |
| Динамическая колонка | Название примененного атрибута |
| Группируемая колонка | Название колонки и контрол группировки справа |

Контрол группировки находится в заголовке группируемой колонки таблицы. Список задач является местом запуска группировки; видимый блок вложенной группировки принадлежит левой навигации.

| Состояние списка задач | Видимость действия группировки |
|---|---|
| Табличный вид; активен `Все задачи` или бизнес-домен | Действие группировки видно в заголовках группируемых колонок. |
| Карточный вид | Действие группировки не видно. |
| Сгруппированное представление | Действие группировки не видно. |
| Пользовательская группа | Действие группировки не видно. |

Табличный список задач не показывает вложенные группировки. Вложенные группировки показывает левая навигация.

## 9. Строка таблицы

Каждая строка таблицы соответствует одной задаче текущей страницы.

| Ячейка | Как отображается значение |
|---|---|
| Чекбокс выбора | Контрол выбора конкретной строки |
| Номер | Номер задачи, например `ECM-1001` |
| Наименование | Иконка задачи, затем подчеркнутый текст задачи; длинный текст обрезается многоточием |
| Статус | Цветной тег статуса |
| Автор | Аватар 24 x 24 px, имя автора, email под именем |
| Дата | Иконка календаря, затем значение даты |
| Пустое значение | Длинное тире |

| Состояние строки | Визуальный признак |
|---|---|
| Обычная строка | Без зеленой или красной левой индикации |
| Наведение на обычную строку | Фон меняется у всех ячеек строки, включая закрепленные |
| Выбранная строка | Чекбокс активен; строка получает выбранное состояние |
| Новая задача | Зеленоватая подложка у всех ячеек и зеленая левая индикация |
| Просроченная задача | Розоватая подложка у всех ячеек и красная левая индикация; значение срока красное |

## 10. Отличия карточки и строки таблицы

| Отличие | Карточка задачи | Строка таблицы |
|---|---|---|
| Количество атрибутов | Компактный набор: номер, наименование, статус, автор, дата | Активные колонки таблицы |
| Настройки колонок | Не влияют | Меняют только динамические колонки |
| Горизонтальный скролл | Нет | Есть при нехватке ширины |
| Закрепленные зоны | Нет | Закреплены чекбокс, номер и наименование |
| Выбор всех задач | Нет | Есть чекбокс выбора всех видимых строк в заголовке |

## 11. Связь с пагинацией

Пагинация расположена под списком задач. Она не входит в карточку, строку таблицы или тело таблицы.

| Условие итогового набора | Видимое состояние списка задач |
|---|---|
| Пагинация скрыта | Список показывает все задачи итогового набора или пустое состояние. |
| Пагинация видна | Список показывает только задачи текущей страницы. |

Полный состав, состояния и правила пагинации описывает компонент `Пагинация`.

## 12. Пустое состояние

| Режим | Где показано пустое состояние |
|---|---|
| Карточный вид | В области карточного списка |
| Табличный вид | В одной строке таблицы; ячейка растягивается на все активные колонки |

Пустое состояние видно, если после применения области задач, активной очереди, поиска, фильтров и текущей страницы нет задач для отображения.
