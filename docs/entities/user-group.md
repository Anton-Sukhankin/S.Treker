# Сущность: Пользовательская группа

Пользовательская группа — сохраненный пользователем набор задач. Группа отображается в левой навигации и может быть выбрана как область списка задач.

## 1. Данные группы

| Данные | Тип значения | Где используется |
|---|---|---|
| Идентификатор группы | Системное значение | Выбор группы в левой навигации и связь с набором задач. |
| Название группы | Текст | Пункт пользовательской группы в левой навигации и заголовок выбранной области задач. |
| Набор задач | Список идентификаторов задач | Отбор задач при выборе группы. |
| Тип контейнера | Группа или закладка с вложенными группами | Размещение группы в левой навигации. |
| Название закладки | Текст | Заголовок закладки, если группа находится внутри закладки. |

## 2. Правила видимого результата

| Условие | Видимый результат |
|---|---|
| Пользовательская группа создана | Левая навигация содержит пункт группы. |
| Пользователь выбирает группу | Список задач показывает задачи из набора группы после поиска, фильтров и пагинации. |
| Группа находится внутри закладки | В левой навигации группа показана внутри блока закладки. |
| В группе нет задач после примененных фильтров | Список задач пуст для выбранной группы; счетчик группы показывает результат примененных условий. |
| Из группы удалены все задачи | Группа исчезает из левой навигации. |
| Активная группа удалена | Активная область задач получает значение `Все задачи`. |

## 3. Связанные компоненты

| Компонент | Как использует группу |
|---|---|
| Левая навигация | Показывает пользовательские группы, закладки и активную группу. |
| Список задач | Отбирает задачи выбранной группы. |
| Массовые действия | Создают группу из выбранных задач или удаляют выбранные задачи из активной группы. |
| Пагинация | Строит страницы по задачам выбранной группы. |
| Фильтрация | Ограничивает видимые задачи внутри выбранной группы. |

## 4. Ограничения

| Область | Правило |
|---|---|
| Полная система закладок | Entity-файл не описывает все возможные типы папок, прав доступа и совместного доступа. |
| Массовые действия | Порядок создания, перемещения и удаления задач из группы описывается в компоненте `Массовые действия`. |
| Левая навигация | Расположение групп и закладок описывается в компоненте `Левая навигация`. |
