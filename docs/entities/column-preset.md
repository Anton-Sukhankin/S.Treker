# Сущность: Пресет колонок

Пресет колонок — сохраненный набор настроек табличного отображения задач. Пресет используется только для табличного вида.

## 1. Данные пресета

| Данные | Тип значения | Где используется |
|---|---|---|
| Название пресета | Текст | Список пресетов в настройках колонок. |
| Контекст пресета | `Все задачи` или конкретный бизнес-домен | Определяет, где пресет доступен. |
| Набор колонок | Список атрибутов | Черновик настроек колонок и таблица после применения. |
| Порядок колонок | Последовательность атрибутов | Порядок динамических колонок таблицы после применения. |
| Видимость колонок | Видимая или скрытая колонка | Состав таблицы после применения. |
| Библиотечное происхождение атрибута | Признак атрибута из библиотеки | Индикация добавленного атрибута в общих атрибутах и библиотеке. |

## 2. Правила видимого результата

| Условие | Видимый результат |
|---|---|
| Пользователь выбирает сохраненный пресет | Черновик настроек колонок получает состав, порядок, видимость и библиотечные атрибуты пресета. |
| Пользователь нажимает `Применить` | Табличный вид показывает колонки из выбранного черновика. |
| Пресет содержит атрибут из библиотеки | В общих атрибутах атрибут показан как добавленный; в библиотеке этот же атрибут получает индикацию возврата. |
| Пользователь выбирает базовое отображение | Черновик настроек колонок получает базовый набор; библиотечные атрибуты возвращаются в состояние по умолчанию. |
| Пользователь удаляет пресет | Пресет исчезает из списка пресетов. |

## 3. Связанные компоненты

| Компонент | Как использует пресет |
|---|---|
| Настройки колонок | Хранят черновик и примененное состояние колонок. |
| Пресеты колонок | Показывают базовое отображение, сохраненные пресеты, сохранение и удаление. |
| Библиотека атрибутов | Отражает библиотечные атрибуты, которые входят в выбранный пресет. |
| Список задач | Получает примененный набор колонок в табличном виде. |

## 4. Ограничения

| Область | Правило |
|---|---|
| Карточный вид | Пресет колонок не меняет карточное отображение задач. |
| Данные задач | Пресет колонок не изменяет значения задач. |
| Атрибуты прототипа | Пресет не создает новые атрибуты; он сохраняет настройки существующих атрибутов. |
| Серверное хранение | Entity-файл не описывает серверное хранение пресетов. |
