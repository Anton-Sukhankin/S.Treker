# Сущность: Бизнес-домен

Бизнес-домен — область задач, доступная текущей роли пользователя. Бизнес-домен ограничивает набор задач и определяет набор атрибутов для таблицы, фильтров, группировок и настроек колонок.

## 1. Текущие бизнес-домены прототипа

| Системное значение | Название в интерфейсе | Назначение в прототипе |
|---|---|---|
| `ecm` | Строительные договоры | Задачи по согласованию и сопровождению строительных договоров. |
| `tracker` | Закупки ТМЦ | Задачи по заявкам, поставкам и закупкам. |
| `control` | Поручения СД | Задачи контроля поручений и замечаний. |

Текущий набор бизнес-доменов является набором для одной роли прототипа. Другая роль может получить другой состав и другой порядок бизнес-доменов.

## 2. Правила видимого результата

| Условие | Видимый результат |
|---|---|
| Пользователю доступен бизнес-домен | Пункт бизнес-домена виден в левой навигации. |
| Пользователь выбирает бизнес-домен | Список задач показывает задачи этого бизнес-домена после очереди, поиска, фильтров и пагинации. |
| Пользователь выбирает `Все задачи` | Список задач показывает задачи всех бизнес-доменов, доступных текущей роли. |
| Компонент работает в контексте конкретного бизнес-домена | Фильтры, настройки колонок и атрибуты используют набор этого бизнес-домена. |
| Компонент работает в контексте `Все задачи` | Фильтры, настройки колонок и таблица используют объединенный набор доступных бизнес-доменов. |

## 3. Связанные компоненты

| Компонент | Как использует бизнес-домен |
|---|---|
| Левая навигация | Показывает доступные бизнес-домены и область `Все задачи`. |
| Список задач | Ограничивает итоговый набор задач выбранным бизнес-доменом или всеми доступными бизнес-доменами. |
| Фильтрация | Показывает параметры, применимые к выбранному доменному контексту. |
| Настройки колонок | Показывают атрибуты выбранного домена или объединенный набор атрибутов. |
| Библиотека атрибутов | Показывает скрытые атрибуты выбранного доменного контекста. |
| Группировка | Создает значения группировки внутри выбранного домена или области `Все задачи`. |

## 4. Ограничения

| Область | Правило |
|---|---|
| Роли | Entity-файл не фиксирует полный набор ролей системы. |
| Порядок бизнес-доменов | Entity-файл не задает постоянный порядок для всех ролей. |
| Состав реальной системы | Entity-файл не перечисляет бизнес-домены, которых нет в текущем прототипе. |
