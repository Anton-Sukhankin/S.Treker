# Сущность: Автор

Автор — пользователь, связанный с задачей как создатель задачи. Интерфейс использует автора для отображения человека в карточном и табличном списке задач.

## 1. Данные автора

| Данные | Тип значения | Где используется |
|---|---|---|
| Идентификатор автора | Системное значение | Связь задачи с автором. |
| Фамилия | Текст | Отображение имени автора в списке задач. |
| Имя | Текст | Формирование полного имени автора и справочных значений. |
| Отчество | Текст | Отображение имени автора в списке задач. |
| Пол | Значение `male` или `female` | Подбор набора аватаров в прототипе. |
| Email | Email-адрес | Подпись под именем автора и копирование адреса. |
| Аватар | Путь к изображению | Фото автора в списке задач. |
| Должность | Текст | Справочное значение пользователя; не является обязательным видимым полем списка задач. |
| Подразделение | Текст | Справочное значение пользователя; не является обязательным видимым полем списка задач. |
| Активность | Логический признак | Определяет, может ли пользователь попадать в справочные списки прототипа. |

## 2. Правила видимого результата

| Условие | Видимый результат |
|---|---|
| У задачи есть автор | В строке или карточке задачи показаны аватар автора, имя автора и email. |
| Пользователь нажимает email автора | Email копируется; интерфейс показывает уведомление о копировании. |
| Email автора длиннее доступной области | Текст email обрезается многоточием. |
| Имя автора длиннее доступной области | Имя автора обрезается многоточием. |
| Аватар автора доступен | В списке задач показано фото автора. |

## 3. Отображаемое имя

| Контекст | Состав имени |
|---|---|
| Основной список задач | Фамилия и отчество автора; email находится под именем. |
| Справочные значения и фильтры | Полное имя автора может использовать фамилию, имя и отчество. |

## 4. Связанные компоненты

| Компонент | Как использует автора |
|---|---|
| Список задач | Показывает автора в карточном и табличном отображении. |
| Фильтрация | Может использовать автора как значение параметра, если параметр присутствует в текущем наборе. |
| Настройки колонок | Используют атрибут `Автор` как колонку, если он доступен в текущей области задач. |
| Группировка | Может группировать задачи по автору, если действие группировки применено к колонке автора. |

## 5. Ограничения

| Область | Правило |
|---|---|
| Управление пользователями | Entity-файл не описывает создание, редактирование и удаление пользователей. |
| Права доступа | Entity-файл не описывает роли, полномочия и авторизацию автора. |
| Компонент аватара | Entity-файл не описывает внутреннюю механику изображения или fallback-компонента дизайн-системы. |
