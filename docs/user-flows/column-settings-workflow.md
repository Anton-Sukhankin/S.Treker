# Flow: настройка колонок

Основной пользовательский путь настройки колонок теперь описан в спецификации компонента:

`docs/components/column-settings/02-behavior.md`

Этот файл остается навигационной ссылкой, чтобы старые ссылки на flow не ломались.

## Где смотреть детали

| Что нужно понять | Где описано |
|---|---|
| Из каких частей состоит интерфейс настройки колонок | `docs/components/column-settings/01-structure.md` |
| Основной пользовательский путь | `docs/components/column-settings/02-behavior.md` |
| Данные, состояния, черновик и правила применения | `docs/components/column-settings/03-data-states-rules.md` |

## Правило обновления

Если меняется интерфейс настройки колонок, обновлять нужно спецификацию в `docs/components/column-settings/`. Если меняется дочерняя зона пресетов или библиотеки атрибутов, обновлять нужно соответствующую вложенную папку внутри `docs/components/column-settings/`.

Этот файл не должен содержать отдельную версию сценария, чтобы не создавать вторую точку истины.
