# Flow: группировка задач из таблицы в левую навигацию

Этот flow описывает связь табличного списка задач и левой навигации при создании вложенной группировки.

Детальные правила заголовков таблицы, раскрытия группировок, счетчиков и удаления значений остаются в компонентных спецификациях.

## Старт сценария

1. Активен `Все задачи` или один бизнес-домен.
2. Список задач открыт в табличном виде.
3. В таблице видна колонка, которая поддерживает действие группировки.

## Участники и роли

| Компонент | Роль в flow |
|---|---|
| Табличный список задач | Источник действия группировки в заголовке колонки. |
| Левая навигация | Владелец созданного блока `По ...` и его значений. |
| Список задач | Получатель выбранного значения группировки как условия отбора. |

## Основной путь

1. Пользователь нажимает действие группировки в заголовке группируемой колонки.
2. Таблица остается в табличном виде и сохраняет текущий набор задач.
3. Если под активным пунктом нет группировки по выбранному атрибуту, левая навигация показывает новый блок `По ...`.
4. Если под активным пунктом уже есть группировка по выбранному атрибуту, левая навигация сохраняет один блок `По ...` и показывает в нем актуальный список значений.
5. Блок `По ...` показывает значения выбранного атрибута, которые имеют задачи после текущего поиска и примененных фильтров.
6. Пользователь выбирает значение внутри группировки.
7. Значение получает активное состояние.
8. Заголовок правой рабочей области получает формат `Область • Значение`.
9. Список задач показывает задачи выбранной области, у которых атрибут группировки равен выбранному значению.

## Передача состояния

| Источник | Условие | Получатель | Видимый итог |
|---|---|---|---|
| Табличный список задач | Пользователь нажал действие группировки в заголовке колонки | Левая навигация | Под активным пунктом виден блок `По ...` со значениями атрибута. |
| Табличный список задач | Пользователь повторно нажал группировку по тому же атрибуту | Левая навигация | В левой навигации остается одна группировка по этому атрибуту с актуальным набором значений. |
| Левая навигация | Пользователь выбрал значение группировки | Список задач | Список показывает только задачи выбранного значения. |
| Левая навигация | Выбрано значение `Без значения` | Список задач | Список показывает задачи выбранной области без значения в атрибуте группировки. |
| Поиск или фильтры | После применения скрывают все задачи значения группировки | Левая навигация | Значение группировки не видно в списке значений. |

## Когда обновлять этот документ

Обновлять, если меняется передача результата группировки между таблицей, левой навигацией и списком задач.

Если меняется только внутренний вид заголовка таблицы или внутренний вид блока группировки, обновлять нужно соответствующую компонентную спецификацию.
