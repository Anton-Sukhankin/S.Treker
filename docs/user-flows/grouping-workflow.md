# Flow: группировка задач из таблицы в левую навигацию

## Зачем нужен сценарий

Пользователь создает группировку по колонке таблицы и затем выбирает значение этой группировки в левой навигации. Финальный результат: список задач показывает только задачи выбранного значения группировки.

## Граница flow

| Граница | Значение |
|---|---|
| Начальное состояние | Активен `Все задачи` или один бизнес-домен; список задач открыт в табличном виде; в таблице видна колонка с действием группировки. |
| Конечный результат | В левой навигации виден блок группировки `По ...`; выбранное значение группировки ограничивает список задач. |
| Не входит в flow | Внутренний вид заголовка таблицы, правила доступности группировки для каждой колонки, раскрытие и удаление группировок в левой навигации. |

## Участники интерфейса

| Участник | Роль в сценарии | Источник деталей |
|---|---|---|
| Табличный список задач | Источник действия группировки в заголовке колонки. | `docs/components/tasks/02-behavior.md`, `docs/components/tasks/03-data-states-rules.md` |
| Левая навигация | Получает созданную группировку, показывает значения и хранит активное значение. | `docs/components/left-navigation/02-behavior.md`, `docs/components/left-navigation/03-data-states-rules.md` |
| Список задач | Получает выбранное значение группировки как условие отбора задач. | `docs/components/tasks/03-data-states-rules.md` |
| Панель инструментов и фильтры | Могут менять текущий итоговый набор, из которого строятся значения группировки. | `docs/components/toolbar/03-data-states-rules.md`, `docs/components/filters/03-data-states-rules.md` |

## Основной сценарий

| Шаг | Действие пользователя или событие | Компонент-источник | Передаваемый результат | Компонент-получатель | Видимый итог |
|---|---|---|---|---|---|
| 1 | Пользователь нажимает действие группировки в заголовке группируемой колонки. | Табличный список задач | Атрибут выбранной колонки. | Левая навигация | Под активной областью виден блок группировки `По ...`. |
| 2 | Группировка создана. | Табличный список задач | Значения выбранного атрибута в текущем итоговом наборе задач. | Левая навигация | В блоке `По ...` виден список значений атрибута. |
| 3 | Пользователь выбирает значение внутри группировки. | Левая навигация | Выбранное значение группировки. | Список задач | Список показывает задачи выбранной области, у которых атрибут группировки равен выбранному значению. |
| 4 | Значение группировки выбрано. | Левая навигация | Название активной области и выбранного значения. | Заголовок рабочей области | Заголовок получает формат `Область • Значение`. |

## Изменения состояния

| Состояние | Владелец состояния | Когда меняется | Видимый результат |
|---|---|---|---|
| Созданная группировка | Левая навигация | Пользователь нажимает действие группировки в заголовке таблицы. | В левой навигации виден блок `По ...` под активной областью. |
| Список значений группировки | Левая навигация | Создана группировка или изменился итоговый набор задач после поиска и фильтров. | В блоке группировки видны только значения, существующие в текущем итоговом наборе. |
| Активное значение группировки | Левая навигация | Пользователь выбирает значение в блоке группировки. | Значение получает активное состояние. |
| Итоговый набор задач | Список задач | Активное значение группировки изменилось. | Список показывает задачи выбранного значения. |

## Ветки

| Ситуация | Условие | Видимый результат |
|---|---|---|
| Повторная группировка по тому же атрибуту | Под активной областью уже есть блок `По ...` для этого атрибута. | В левой навигации остается один блок группировки по этому атрибуту; список значений актуализирован. |
| Значение `Без значения` выбрано | У части задач отсутствует значение атрибута группировки. | Список задач показывает задачи выбранной области без значения в атрибуте группировки. |
| Поиск или фильтры скрывают все задачи значения | После пересборки итогового набора у значения нет задач. | Значение не видно в списке значений группировки. |
| Пользователь находится в карточном виде, сгруппированном значении или пользовательской группе | Действие группировки недоступно в текущем состоянии списка. | Новая группировка не создается. |

## Когда обновлять документ

- Меняется действие создания группировки из заголовка таблицы.
- Меняется компонент, который показывает созданную группировку.
- Меняется правило выбора значения группировки и результат для списка задач.
- Меняется связь группировки с поиском, фильтрами или текущим итоговым набором задач.
