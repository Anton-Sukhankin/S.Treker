# Flow: выбор задач и создание пользовательской группы

Этот flow описывает связь списка задач, функционального блока `Массовые действия`, модального окна перемещения и левой навигации.

Детальное поведение чекбоксов и выбранного состояния задач остается в компоненте `Список задач`. Детальное поведение плавающей панели, кнопок панели и модального окна остается в функциональном блоке `Массовые действия`.

## Старт сценария

1. Пользователь видит карточный или табличный список задач.
2. В списке есть минимум одна видимая задача.
3. Ни одна задача не выбрана или уже есть выбранные задачи.

## Участники и роли

| Участник | Роль в flow |
|---|---|
| Список задач | Содержит чекбоксы карточек и строк; показывает выбранное состояние задач. |
| Массовые действия | Показывает плавающую панель, счетчик выбранных задач и модальное окно перемещения. |
| Левая навигация | Показывает созданную пользовательскую группу. |
| Список задач | После выбора пользовательской группы показывает задачи этой группы. |

## Основной путь

1. Пользователь выбирает чекбокс одной карточки или строки.
2. Выбранная задача получает выбранное состояние.
3. Плавающая панель массовых действий видна в нижней центральной части экрана.
4. Счетчик выбранных задач на плавающей панели показывает количество выбранных задач.
5. Пользователь нажимает `Переместить в группу`.
6. Поверх интерфейса видно модальное окно `Перемещение в группу`.
7. Пользователь вводит название группы или оставляет поле пустым.
8. Пользователь нажимает `Переместить`.
9. Модальное окно закрывается.
10. Левая навигация показывает новую пользовательскую группу.
11. Все выбранные задачи теряют выбранное состояние.
12. Плавающая панель массовых действий скрывается.

## Передача состояния

| Источник | Условие | Получатель | Видимый итог |
|---|---|---|---|
| Список задач | Выбрана минимум одна задача | Массовые действия | Плавающая панель видима; счетчик показывает количество выбранных задач. |
| Массовые действия | Пользователь нажал `Переместить в группу` | Модальное окно перемещения | Окно `Перемещение в группу` видимо; поле названия группы доступно для ввода. |
| Модальное окно перемещения | Пользователь нажал `Переместить`, поле названия заполнено | Левая навигация | Создана пользовательская группа с введенным названием. |
| Модальное окно перемещения | Пользователь нажал `Переместить`, поле названия пустое | Левая навигация | Создана пользовательская группа с названием `Группа`. |
| Левая навигация | Пользователь выбирает созданную группу | Список задач | Список показывает задачи, сохраненные в выбранной пользовательской группе. |
| Массовые действия | Группа создана | Список задач и массовые действия | Выбранные задачи теряют выбранное состояние; плавающая панель скрыта. |

## Ветки, влияющие на flow

| Ситуация | Видимый итог |
|---|---|
| Пользователь закрывает модальное окно или нажимает `Отмена` | Модальное окно закрыто; выбранные задачи сохраняют выбранное состояние; плавающая панель остается видимой. |
| Пользователь выбирает все видимые строки через чекбокс заголовка таблицы | Выбраны только строки текущей страницы; счетчик выбранных задач на плавающей панели показывает общий набор выбранных задач. |
| Выбранные задачи не видны после смены страницы, поиска, фильтров или области | Счетчик выбранных задач на плавающей панели сохраняет эти задачи в общем количестве выбранных задач. |
| Пользователь удаляет все задачи из активной пользовательской группы | Пункт группы исчезает из левой навигации; активная область получает значение `Все задачи`. |

## Когда обновлять этот документ

Обновлять, если меняется передача выбранных задач в массовые действия, создание пользовательской группы или результат выбора созданной группы в левой навигации.

Если меняется только состав кнопок плавающей панели или внутреннее состояние модального окна, обновлять нужно спецификацию функционального блока `Массовые действия`.
