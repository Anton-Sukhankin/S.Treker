# Flow: выбор задач и создание пользовательской группы

## Зачем нужен сценарий

Пользователь выбирает задачи в списке, создает из них пользовательскую группу и видит эту группу в левой навигации. Финальный результат: левая навигация показывает новую группу, а список задач может быть ограничен задачами этой группы.

## Граница flow

| Граница | Значение |
|---|---|
| Начальное состояние | Пользователь видит карточный или табличный список задач; в списке есть минимум одна видимая задача. |
| Конечный результат | Левая навигация показывает созданную пользовательскую группу; выбранные задачи теряют выбранное состояние; плавающая панель массовых действий скрыта. |
| Не входит в flow | Вид чекбоксов, состав кнопок плавающей панели, внутреннее состояние модального окна и локальные правила удаления из группы. |

## Участники интерфейса

| Участник | Роль в сценарии | Источник деталей |
|---|---|---|
| Список задач | Содержит чекбоксы выбора задач и показывает выбранное состояние карточек или строк. | `docs/components/tasks/02-behavior.md`, `docs/components/tasks/03-data-states-rules.md` |
| Массовые действия | Показывает плавающую панель, счетчик выбранных задач и действие `Переместить в группу`. | `docs/components/bulk-actions/02-behavior.md`, `docs/components/bulk-actions/03-data-states-rules.md` |
| Модальное окно перемещения | Показывает поле названия группы и действие подтверждения перемещения. | `docs/components/bulk-actions/01-structure.md`, `docs/components/bulk-actions/02-behavior.md` |
| Левая навигация | Показывает созданную пользовательскую группу как пункт навигации. | `docs/components/left-navigation/02-behavior.md`, `docs/components/left-navigation/03-data-states-rules.md` |
| Список задач как получатель | Показывает задачи выбранной пользовательской группы. | `docs/components/tasks/03-data-states-rules.md` |

## Основной сценарий

| Шаг | Действие пользователя или событие | Компонент-источник | Передаваемый результат | Компонент-получатель | Видимый итог |
|---|---|---|---|---|---|
| 1 | Пользователь выбирает чекбокс одной карточки или строки. | Список задач | Набор выбранных задач. | Массовые действия | Плавающая панель массовых действий видна; счетчик показывает количество выбранных задач. |
| 2 | Пользователь нажимает `Переместить в группу`. | Массовые действия | Набор выбранных задач и выбранное действие перемещения. | Модальное окно перемещения | Поверх интерфейса видно окно `Перемещение в группу`; поле названия группы доступно для ввода. |
| 3 | Пользователь вводит название группы или оставляет поле пустым. | Модальное окно перемещения | Название группы или пустое значение. | Модальное окно перемещения | Значение поля видно внутри модального окна. |
| 4 | Пользователь нажимает `Переместить`. | Модальное окно перемещения | Название группы и набор выбранных задач. | Левая навигация | В левой навигации видна новая пользовательская группа. |
| 5 | Группа создана. | Массовые действия | Количество выбранных задач равно `0`. | Список задач | Выбранные задачи теряют выбранное состояние. |
| 6 | Выбор очищен. | Список задач | Количество выбранных задач равно `0`. | Массовые действия | Плавающая панель массовых действий скрыта. |
| 7 | Пользователь выбирает созданную группу. | Левая навигация | Новая пользовательская группа как активная область. | Список задач | Список показывает задачи, сохраненные в выбранной пользовательской группе. |

## Изменения состояния

| Состояние | Владелец состояния | Когда меняется | Видимый результат |
|---|---|---|---|
| Выбранные задачи | Список задач | Пользователь выбирает или снимает чекбокс карточки или строки. | Выбранные карточки или строки имеют выбранное состояние. |
| Видимость плавающей панели | Массовые действия | Количество выбранных задач становится больше `0` или равно `0`. | Панель видна при выбранных задачах и скрыта без выбранных задач. |
| Счетчик выбранных задач | Массовые действия | Пользователь выбирает или снимает чекбокс карточки или строки. | Счетчик на плавающей панели показывает количество выбранных задач. |
| Видимость модального окна | Массовые действия | Пользователь нажимает `Переместить в группу`, `Отмена`, закрытие или `Переместить`. | Модальное окно открыто или закрыто. |
| Пользовательская группа | Левая навигация | Пользователь подтверждает перемещение выбранных задач. | В левой навигации виден пункт новой группы. |
| Активная область задач | Левая навигация | Пользователь выбирает созданную группу. | Группа показана как активная; список задач показывает задачи группы. |

## Ветки

| Ситуация | Условие | Видимый результат |
|---|---|---|
| Название группы не заполнено | Пользователь нажимает `Переместить` при пустом поле названия. | Создана пользовательская группа с названием `Группа`. |
| Модальное окно закрыто без перемещения | Пользователь нажимает `Отмена` или закрывает окно. | Модальное окно закрыто; выбранные задачи сохраняют выбранное состояние; плавающая панель остается видимой. |
| Выбраны все видимые строки таблицы | Пользователь нажимает чекбокс в заголовке таблицы. | Выбраны строки текущей страницы; счетчик плавающей панели показывает количество выбранных задач. |
| Выбранные задачи перестали быть видимыми | Пользователь меняет страницу, поиск, фильтры или область задач. | Счетчик выбранных задач сохраняет общий набор выбранных задач. |
| Все задачи удалены из активной пользовательской группы | Пользователь удаляет из группы все задачи этой группы. | Пункт группы исчезает из левой навигации; активная область получает значение `Все задачи`. |

## Когда обновлять документ

- Меняется передача выбранных задач из списка задач в массовые действия.
- Меняется создание пользовательской группы из выбранных задач.
- Меняется результат появления группы в левой навигации.
- Меняется результат выбора созданной группы как активной области задач.
