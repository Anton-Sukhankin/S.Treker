# Flow: поиск, фильтры и итоговый набор задач

## Зачем нужен сценарий

Пользователь ограничивает набор задач через выбранную область, этап согласования, поиск и панель фильтрации. Финальный результат: список задач и пагинация показывают только задачи, которые проходят все активные условия.

## Граница flow

| Граница | Значение |
|---|---|
| Начальное состояние | В левой навигации выбрана область задач; в очередях активен один этап; пользователь видит карточный или табличный список задач. |
| Конечный результат | Список задач показывает итоговый набор после области, этапа, поиска и примененных фильтров; пагинация построена по этому набору. |
| Не входит в flow | Внутренний состав панели фильтрации, локальная работа мультивыборов, диапазона суммы, кнопок панели инструментов и карточек или строк списка задач. |

## Участники интерфейса

| Участник | Роль в сценарии | Источник деталей |
|---|---|---|
| Левая навигация | Задает выбранную область задач для списка. | `docs/components/left-navigation/03-data-states-rules.md` |
| Очереди | Задают активный этап согласования для списка. | `docs/components/queues/03-data-states-rules.md` |
| Панель инструментов | Содержит поле поиска, кнопку открытия панели фильтрации и счетчик активных фильтров. | `docs/components/toolbar/02-behavior.md`, `docs/components/toolbar/03-data-states-rules.md` |
| Панель фильтрации | Хранит черновые значения до применения и примененные параметры после нажатия `Применить`. | `docs/components/filters/02-behavior.md`, `docs/components/filters/03-data-states-rules.md` |
| Список задач | Показывает задачи, которые проходят выбранную область, этап, поиск и примененные фильтры. | `docs/components/tasks/03-data-states-rules.md` |
| Пагинация | Показывает страницы итогового набора задач. | `docs/components/pagination/03-data-states-rules.md` |

## Основной сценарий

| Шаг | Действие пользователя или событие | Компонент-источник | Передаваемый результат | Компонент-получатель | Видимый итог |
|---|---|---|---|---|---|
| 1 | Пользователь выбирает область задач. | Левая навигация | Выбранная область: `Все задачи`, бизнес-домен, значение группировки или пользовательская группа. | Список задач | В списке остаются задачи выбранной области. |
| 2 | Пользователь выбирает этап согласования. | Очереди | Активный этап согласования. | Список задач | В списке остаются задачи выбранной области и активного этапа. |
| 3 | Пользователь вводит текст в поле `Поиск по задачам`. | Панель инструментов | Строка поиска. | Список задач | В списке остаются задачи, где введенный текст найден в значениях задачи. |
| 4 | Пользователь нажимает `Фильтры`. | Панель инструментов | Команда раскрытия панели фильтрации. | Панель фильтрации | Правая панель фильтрации открыта поверх интерфейса. |
| 5 | Пользователь выбирает параметры фильтрации. | Панель фильтрации | Черновые значения параметров. | Панель фильтрации | Выбранные значения видны внутри панели; список задач еще не меняется. |
| 6 | Пользователь нажимает `Применить`. | Панель фильтрации | Примененные параметры фильтрации. | Список задач | Список показывает задачи, которые проходят область, этап, поиск и примененные фильтры. |
| 7 | Фильтры применены. | Панель фильтрации | Количество активных параметров. | Панель инструментов | Кнопка `Фильтры` показывает счетчик активных параметров вместо иконки фильтра. |
| 8 | Список показывает новый итоговый набор задач. | Список задач | Количество задач в итоговом наборе. | Пагинация | Активная страница имеет значение `1`; страницы соответствуют новому набору. |

## Изменения состояния

| Состояние | Владелец состояния | Когда меняется | Видимый результат |
|---|---|---|---|
| Выбранная область задач | Левая навигация | Пользователь выбирает область задач. | Активный пункт левой навигации и заголовок рабочей области соответствуют выбранной области. |
| Активный этап согласования | Очереди | Пользователь выбирает сегмент этапа. | Выбранный сегмент имеет активное состояние; список задач показывает задачи этого этапа. |
| Строка поиска | Панель инструментов | Пользователь вводит или очищает текст поиска. | Список задач показывает задачи, значения которых соответствуют текущей строке поиска. |
| Черновые параметры фильтрации | Панель фильтрации | Пользователь меняет значения внутри открытой панели. | Значения видны внутри панели; список задач сохраняет прежний примененный набор. |
| Примененные параметры фильтрации | Панель фильтрации | Пользователь нажимает `Применить`. | Список задач показывает задачи по новым параметрам; панель фильтрации закрыта. |
| Итоговый набор задач | Список задач | Пользователь выбирает область, выбирает этап, вводит или очищает поиск, применяет или сбрасывает фильтры. | Видимые карточки или строки соответствуют всем активным условиям. |
| Текущая страница | Пагинация | Список задач показывает новый итоговый набор. | Активная страница имеет значение `1`. |

## Ветки

| Ситуация | Условие | Видимый результат |
|---|---|---|
| Итоговый набор пуст | Ни одна задача не проходит область, этап, поиск и фильтры одновременно. | Список задач показывает пустое состояние; пагинация скрыта. |
| Итоговый набор содержит 20 задач или меньше | Количество задач в итоговом наборе не превышает размер страницы. | Пагинация скрыта. |
| Итоговый набор содержит 21 задачу или больше | Количество задач в итоговом наборе больше размера страницы. | Пагинация видима и показывает доступные страницы. |
| Активные фильтры сброшены быстрым сбросом | Пользователь нажимает счетчик активных фильтров в кнопке `Фильтры`. | Примененные параметры фильтрации очищены; список задач показывает набор без параметров панели фильтрации. |

## Когда обновлять документ

- Меняется передача состояния между левой навигацией, очередями, панелью инструментов, фильтрами, списком задач или пагинацией.
- Меняется условие, по которому список задач формирует итоговый набор.
- Меняется видимый результат применения или сброса фильтров.
- Меняется компонент, который показывает результат поиска, фильтрации или пагинации.
